Dictation: Make macro methods more concise

The Macro class contains methods called runMacro, getMacroName, and
getMacroNameString. The 'Macro' part in these names is redundant,
since they're already defined on the Macro class. This change makes
the code more concise by removing the 'Macro' part of the methods.

macro.runMacro() is now macro.run()
macro.getMacroName() is now macro.getName()
macro.getMacroNameString() is now macro.getNameAsString()
