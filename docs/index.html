Prerender UKM: Ensure LinkToText doesn't calls GetPageUkmSourceId

GetPageUkmSourceId doesn't expect it is called while prerendering, and
returned ID will not be bound with any URL if it is called.

As LinkToText seems not being used while prerendering, we add CHECK
to ensure the method is called only when the page is not in the
prerendering state.

Also convert existing DCHECK to follow the up-to-date guildelines.
