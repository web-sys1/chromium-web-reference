Make ComputeNGCaretPosition() to handle upstream position after soft line wrap

This patch changes |ComputeNGCaretPosition()| to handle upstream position
after soft line wrap to return position before a space causing soft line
wrap instead of after the space for painting caret of upstream position
after soft line wrap correctly.

Upstream position "abc |xyz", where "|" specify position,
|ComputeNGCaretPosition()| returns:
 - Before: "abc |xyz"
 - After: "abc| xyz"
Note: " " is not rendered.

This patch also introduces |ComputedStyle::NeedsTrailingSpace()| to
share source code.
