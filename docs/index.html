GlicSplitButtonController: Combine delegate classes

This is the first step of a refactor to centralize the logic that
controls GlicButton and GlicActorTaskIcon, currently duplicated across
TabStripActionContainer and ToolbarView which are desktop only.

Combine the GlicButtonControllerDelegate, GlicNudgeDelegate, and
GlicActorNudgeDelegate interfaces into one new interface
GlicSplitButtonDelegate.

For more context see
https://docs.google.com/document/d/1xGsDDmHERiSTH1JXUYrRKNMTnobkDShGJ8EBQSQIHFA/edit?usp=sharing
