Change TrackedElementHandler to use element name as key

This CL changes the map that TrackedElementHandler uses to keep track of
its TrackedElementWebUI so that it is keyed by the string name of the
ui::ElementIdentifier instead of the ui::ElementIdentifier. This makes
lookups easier since we always are given the name, not the
ui::ElementIdentifier. Also, since std::string is hashable we can use a
hashmap instead of a tree-map, making lookups O(1) instead of O(log(n)).
Combined this changes lookup performance from O(N) to O(1).
