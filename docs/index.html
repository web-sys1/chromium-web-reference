[iOS][Assistant][4/N] Add sheet resizing

This CL implements the resizing capability for the Assistant Sheet using
a pan gesture on the header.

Key features:
- Pan gesture handling to resize the sheet dynamically.
- Constraints clamping to min (content) and max (safe area) heights.
- Logarithmic rubber-banding resistance when dragging beyond limits.
- Spring animation for snapping and flinging.
