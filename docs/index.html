[Deprecations] (1) Add deprecation.json5 file

This CL lays the foundation for the migration from our (currently) split
chromium and devtools_frontend manual code flow (see
third_party/blink/renderer/core/frame/deprecation/README.md) to a modern
code-generated flow that requires only modification of a json5 file.

This CL only establishes the initial json5 file and a presubmit script
to verify it.

Design Doc:
https://docs.google.com/document/d/1oFiM_GFXX4VKI5L6k78Lvh1fItJgmN2LheEGoh9QuSY/edit#

This CL is part of a series:
(1) Add deprecation.json5 file
(2) Add existing deprecations to deprecation.json5
(3) Copy deprecation.json5 into devtools_frontend via update flow
(4) Codegen c++ deprecations code
