Add metric to record printed page size

This CL adds a metric to record the size of pages to be printed to be scanned by content analysis. This allows us to monitor the size of printed pages that are being scanned to verify if we need to increase the supported size beyond 50 MB.

The maximum metric value is set to 500 MB and it's sampled with 50 buckets.
