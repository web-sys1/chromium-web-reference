[editing] Use relocatable position in PerformTrivialReplace

InsertTextCommand::PerformTrivialReplace used to store a position,
change the DOM, and then use that position. However, it could have
become invalid due to the DOM changes, and fail an assert.

Therefore, this patch uses a RelocatablePosition.
