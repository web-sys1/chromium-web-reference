Improve request scheduling with help of cache

In network service, requests that already exist in cache may still be
queued for quite a long time. Even for those that don't perform any
validation and don't contend for the network. Schedule these requests
with consideration of cache data may improve the loading.

This change introduces a timer that checks if there are any pending
requests that are cached and have been queued for a long time. Start
them if any.
