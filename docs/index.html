[fuchsia] Prepare for harmonization of pkgctl output format.

Currently, the pkgctl command  outputs a trailing newline for almost all
subcommands. pkgctl get-hash is one exception which is in the process of
getting cleaned up in https://fxrev.dev/662665. This CL prepares the
fuchsia test target for this change.
Also it fixes the misspelling of "merkel" -> "merkle" in the process.
