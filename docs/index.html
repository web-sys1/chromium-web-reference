[NTP Customization] Add metrics

This CL adds metrics to track the number of times the bottom sheet is
opened and toggles are used to disable/enable magic stack modules.

In this CL, the following three histograms are added:
1.NewTabPage.Customization.{ModuleToggleState}
2.NewTabPage.Customization.BottomSheet.Shown
3.NewTabPage.Customization.OpenBottomSheetEntry

Besides, a user action "MobileMenuNtpCustomization" is added to the
ChromeTabbedActivity and records the user clicking the main menu to open
the bottom sheet.
