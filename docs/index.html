[Voice Search] Grey out NTP background when realbox voice search is active

Currently, when clicking Voice Search in the NTP Realbox, the NTP
background and modules remain interactive and are not greyed out.

This CL updates the Realbox Voice Search coherence overlay to apply the
NTP background scrim when active, matching the behavior of the NTP
Composebox. This behavior is gated behind the realbox voice search
coherence flags.
