Introduce separate, explicit HandleEndOfSniffableResponseBody.

Before this CL, getting Decision::kSniffMore would *implicitly*
translate into Decision::kAllow when there is nothing more to sniff
(e.g. because sniffing reached the end of the response body, or we have
sniffed more than the maximum number of bytes).

After this CL, this decision is handled *explicitly* by a new,
separate method: HandleEndOfSniffableResponseBody.  The explicit
behavior is more readable.  Additionally, a separate virtual method
supports switching to a different fallback behavior in ORB.
