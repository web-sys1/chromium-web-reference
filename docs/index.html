printing: Add `reset_to_default` attribute to CDD

The current implementation of CDD default values allows the user to
overwrite the default value for a setting with the last value selected
by the user.

This CL adds a new attribute to CDD called `reset_to_default` that, when
true, will always reset a setting to its default value, regardless of
the last user selected value.

This attribute works with color, duplex, mediaSize and dpi.
