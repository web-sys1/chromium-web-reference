Keep weak reference to ViewAndroidDelegate inside SensitiveContentClient

`AwContentsGarbageCollectionTest` failed because the
`SensitiveContentClient` was the last object holding an unnecessary
reference to the `ViewAndroidDelegate`, eventually not allowing
`AwContents` to be garbage collected.

Sample failure:
https://ci.chromium.org/ui/p/chromium/builders/ci/android-15-x64-rel/2615/overview

This CL also re-enables the tests, which were disabled by Test Eng here:
crrev.com/c/5867992, crbug.com/366524564
