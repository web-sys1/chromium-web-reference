browser/printing: stop using SHA-1 for hashing -> SHA-256

SHA-1 is currently being used in both browser tests
(`pdf_to_emf_converter_browsertest.cc` and
`pwg_raster_converter_browsertest.cc`) to hash data and compare
payloads. Since the specific type of hashing should not matter, this CL
migrates from using SHA-1 to SHA-256, as SHA-1 is deprecated and advised
against using.
