[Autofill] Open Save Prompt before saving profile on Desktop

This CL kicks off the implementation of Address Profile Save prompt
on Desktop.

Namely it does the following:
1- Invokes SaveAddressProfileBubbleController from ChromeAutofillClient
2- Introduces AutofillBubbleHandler::ShowSaveAddressProfileBubble()
3- Make SaveAddressProfileBubbleController subclass of
AutofillBubbleControllerBase.
4- Implement basic functionality in SaveAddressProfileView to open the
bubble.
