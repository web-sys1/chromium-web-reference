Missing reattach of CSSStyleDeclaration for some @-rules

@position-try, @property, and @counter-style did not reattach their
CSSStyleDeclaration CSSOM wrapper when reattaching the rule.

Also, @position-try did not clone its property set as necessary which
resulted in modifying a shared property set between stylesheet when the
StyleSheetContents is shared.

@property does not have this issue since it is fully immutable.

@counter-style only creates a wrapper for internal devtools usage, but
still needs its cloning fixed.

Removed Copy() methods which were not in use.
