Retrieves computed node attributes where available at the platform layer

If there are any computed node data attributes, such as the kValue attribute for
rich text fields, they will now be exposed to all platforms using code in
BrowserAccessibility and AXPlatformNodeBase. A followup patch will implement this throughout the whole codebase.
This should make it possible to speed up the computation of the kValue attribute because we could stop
serializing and sending it from Blink to the browser process.
