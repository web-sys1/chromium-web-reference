lacros: Remove inactive WebUIs

Some WebUIs (e.g. CrashesUI) are part of Lacros but are inactive because
when asked to navigate to the corresponding URL (e.g. chrome://crashes),
Lacros will actually forward the request to Ash and thus invoke the Ash
instance of that WebUI. Remove the following such WebUIs from Lacros entirely:

chrome://app-disabled
chrome://bluetooth-internals
chrome://crashes

Also remove an unused constant (chrome::kChromeUIFirmwareUpdatesAppURL)
and fix a typo (chrome::kChromeUIInternetConfigDialogURL).
