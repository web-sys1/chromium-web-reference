Use NOTREACHED() after //sandbox switch statements

Change switch statements that exhaustively handle all cases to have an
explicit NOTREACHED() in the unreachable part where the switch
statements end. Per https://abseil.io/tips/147, this helps prevent the
compiler from generating code with undefine behavior in some situations.
