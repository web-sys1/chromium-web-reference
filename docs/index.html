Downsample WebAuthn IsUVPAA() startup metric on macOS

The macOS implementation of
IsUserVerifyingPlatformAuthenticatorAvailable() is prone to crashing or
hanging somewhere in the macOS Security Framework. IsUVPAA() calls are
usually rare, except that Chrome has a startup metric for it. So before
doing any further mitigation, downsample the metric by factor 10000.

Changes of platform authenticator availability over time are not that
interesting to use, so can probably get rid of the entire metric soon.
But CrOS just launched a platform authenticator, so for now only disable
it on macOS.
