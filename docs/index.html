[Settings/Autofill] Add basic data type counters to Your Saved Info

This change introduces new counters to be displayed on Your Saved
Info settings page:
- number of addresses,
- number of credit cards,
- number of IBANs,
- number of Pay over time issuers (Buy Now Pay Later).

This is part of the Home of Transactions project, which aims to create a
central "Your saved info" page in settings.

The new counters are fetched from the existing AutofillManager
and PaymentsManager. The counts are updated in real-time to reflect
the changes from other tabs or sync.

This change only makes these counters available on 'Your saved info'
page as properties. They will be used and displayed once the data chips
components are ready.
