sql: Clarify SQLite error vs result codes in sql::Statement.

This CL starts aligning our variable naming and comments to SQLite's
terminology in https://www.sqlite.org/rescode.html

TL;DR: SQLite result codes include success values, like SQLITE_OK.
SQLite error codes are a subset of result codes that signal failure. Our
source code will be easier to read if it's clear around where we're
dealing with result codes (all possible return values from a SQLite
call), or only error codes.
