Make LocalRecoveryFactor's account scoped

https://crrev.com/c/6521707 allows PhysicalDeviceRecoveryFactor and
ICloudKeychainRecoveryFactor to be account scoped - i.e. they're only
created if there is a primary account.

Similarly, both local recovery factors now require a non-null
TrustedVaultThrottlingConnection during creation.

Those changes allow various simplifications in
StandaloneTrustedVaultBackend, PhysicalDeviceRecoveryFactor and
ICloudKeychainRecoveryFactor.
