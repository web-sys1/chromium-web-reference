[Files]: Add optional rootDir parameter to searchFiles.

fileManagerPrivate.searchFiles always searches from the root of the
local file system. However, the function it invokes takes the root from
which to start the file search. This CL exposes that parameter as an
option, so that we can use searchFiles to efficiently search parts of
the local file system.
