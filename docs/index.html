[Autofill] Ignore first hidden save address prompt when 2nd one comes

Before this CL:
The bubble controller will silently ignore any in progress requests with
hidden prompts when second prompt arrives.

After this CL:
The bubble controller invokes the callback of the first prompt to inform
the backend that the first prompt has been ignored.
This is particularly useful for metrics which are recorded in the
backend.
