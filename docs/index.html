Change LeastRecentlyShown() to return a list.

The PromosManager aims to show the least recently shown promo, if it's
eligible for display.

However, it's possible the least recently shown promo isn't eligible for
display, but the next least recently shown promo is.

This change ensures the manager loops over the least recently shown
promos, and returns the first least recently shown promo for display, if
eligible.

It's possible no least recently shown promos are available for display.
