Remove DoubleRect

DoubleRect was only used in code converting a DOMRect to an enclosing
gfx::Rect. Now add DOMRect::ToEnclosingRect() for the usage without
needing DoubleRect.

There is a TODO(crbug.com/940846) in canvas_path.cc saying that we
would use DoubleRect/DoublePoint in the code. The TODO was actually
out-dated because crbug.com/940846 was merged into crbug.com/938800
which has already been fixed by changing APIs to accept double and
keeping implementations using float.
