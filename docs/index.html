Modernize BackupRefPtrSupport synthetic Finch

Since we moved to a new experiment model, the current syntehtic Finch
setting no longer made sense, as it was designed for the old, 3-way
binary A/B experiments. It is now going to be based on two Finch
experiments, and combines them in such a way that the experiments don't
collide with each other (they aren't independent).
