[mac] Use `VideoCaptureTransportType` instead of `int`

The current implementation stores the connection type as an integer and
deduces the VideoCaptureTransportType later on. In this CL, the
VideoCaptureTransportType is directly used.

Only macOS supports `transportType`[1] property for AVCaptureDevice
and on iOS its not available. This CL moves the platform specific code
to macOS utility file.

[1] https://developer.apple.com/documentation/avfoundation/avcapturedevice/1387804-transporttype
