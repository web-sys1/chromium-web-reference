[iOS] Add Docking Promo to the First Run Experience (FRE).

The Docking Promo is a new, animated, instructional promo that already
exists in the codebase at ios/chrome/browser/docking_promo.

This feature aims to ramp in two experimental arms: [1] after the FRE is
dismissed, [2] during the FRE. This CL introduces functionality for the
2nd arm: displaying the promo during the FRE.

This change:
- Adds the DockingPromo screen type.
- Adds new initializer, `initWithBaseNavigationController:` for the new
  FRE-based use case.
