ContentSettings Histogram not to record metrics for irregular Profiles

ContentSettings.* histograms used to record metrics for all profiles,
including irregular ones (Incognito, Guest, System Profile and other
ChromeOS specific irregular profiles).

Now those histograms only record metrics for Regular user profiles.

Renamed the histograms to follow the following logic:

ContentSettings.{RegularProfileFiltered}Default*
ContentSettings.{RegularProfileFiltered}Exceptions.*

Where RegularProfileFiltered={"","RegularProfile."}
""                 -> old obsolete histograms (all profile types).
"RegularProfile."  -> new histograms (regular profile only).
