[Extensions] Only enable the systemIndicator API on supported platforms.

This API broke on ChromeOS in 2019 and test coverage was disabled.
Since this API is abandoned, and we'd like to get rid of it, this
CL limits its use to the platforms where it still works. It also
disables the API in the Dev channel, limiting it to only the extensions
that were previously allowed in stable.

The tests were tested for flakiness and passed without issue, so they
are now all enabled only on those platforms.
