[views-ax] Migrates role to view constructor in chrome/browser/ui/views

This CL migrates setting of the accessible role from
View::GetAccessibleNodeData to the view constructor. This ensures
that the role is updated in the accessibility cache as soon as the
view is constructed.

The migration is applied to following files:
1. chrome/browser/ui/views/critical_notification_bubble_view.cc
2. chrome/browser/ui/views/desktop_capture/*
3. chrome/browser/ui/views/download/*
4. chrome/browser/ui/views/editor_menu/*
5. chrome/browser/ui/views/extensions/*

This CL is part of the ViewsAX project:
https://docs.google.com/document/d/1Ku7HOyDsiZem1yaV6ccZ-tz3lO2XR2NEcm8HjR6d-VY/edit#heading=h.ke1u3utej413
