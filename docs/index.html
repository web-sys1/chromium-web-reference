[a11y] Expose Media "more options" button as menu button

The MediaControlOverflowMenuButtonElement was missing the
aria-haspopup=menu attribute. This CL fixes it.

Without this attribute, ATs users won't know that a popup is supposed to
show up when the button is pressed. To fix it, we simply need to add
the "haspopup=menu" attribute on the element.

menu button, helping screen readers announcing that the button is hiding
a collapsed menu of options.
