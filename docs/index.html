tests: Rework DPD test arguments

This CL adjusts the config for

* dangling_pointer_detection_blink_web_tests and
* dangling_pointer_detection_blink_wpt_tests

which are failing on the BRP FYI bots on all non-Linux platforms.

This is because the `FlagSpecificConfig` setup is not used on non-Linux
platforms (c.f. crbug/1416763#c1).

This is a moderately experimental change - if these suites prove too
flaky or noisy to be useful, we will probably remove them anyway.
