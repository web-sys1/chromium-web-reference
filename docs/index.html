Add DanglingPointerDetector synthetic field trial.

The dangling pointer depends on its own binary flag and a runtime flag.
It also depends on BackupRefPointer, which has its own binary and
runtime flag.

So, it can't really be enabled using only a field trial. Adding this
synthetic one allows comparing two different binaries.
