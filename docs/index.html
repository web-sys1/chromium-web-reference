[crd host][linux] Enable remote webauthn capability

This CL enables remote WebAuthn capability on Linux and packages the
webauthn NMH into the deb, so that Google SKE engineers can test out the
NMH. The client leg of the capability is not enabled on the prod
website, so there should be no impact on general users. The only
downside is that it adds a binary that would do nothing but add ~28mb to
the installation size, but that impact is mostly limited to Googlers
given external Linux host user base is negligibly small, and the size
increase will eventually be needed once the feature becomes ready.
