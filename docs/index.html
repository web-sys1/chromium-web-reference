[iOS] Disable UNO flag for AdvancedSettingsSigninTestCase.

The AdvancedSettingsSigninCoordinator only creates the
ManageSyncSettingsCoordinator in the
SyncSettingsAccountState::kAdvancedInitialSyncSetup state.

All of its EG tests are written for the view in this state, which does
not exist in the UNO sign-in flows.
This CL disables the UNO flag kReplaceSyncPromosWithSignInPromos for
these EG tests.
