[FilesBanner] Implement diskThreshold functionality

Banners can be shown or hidden based on the amount of remaining disk
space for a watched volume. The implementations only subscribes to
volumes that are currently visible for a given Files app window. When an
underlying change occurs on the disk a call to getSizeStats is made to
get the total and remaining size of the volume.
