Revert "Schedule a Local State write to hopefully resolve a discrepancy."

This reverts commit 868004009c1ccd49c2483fa0d03134b783ed80e4.

Reason for revert: Scheduling the Local State write had the expected
impact on the Local-State-based extension renderer crash counts.
However, CommitPendingWrite() shouldn't be called all over the place.
From pref_service.h, "This should only be used if we need to save
immediately (basically, during shutdown)."

> Schedule a Local State write to hopefully resolve a discrepancy.
