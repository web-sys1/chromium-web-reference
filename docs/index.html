[iOS] Fix cold start scroll position on refactored NTP

Fixes issue where cold starting the refactored NTP would have the
initial scroll position be slightly scrolled down. This was caused by
the content suggestions' height changing.

This fix sets the content offset after updating the content suggestions
layout, if the NTP did not yet appear and if the scroll position wasn't
forcefully set from the web state.
