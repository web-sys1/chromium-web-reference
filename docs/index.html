Add histogram for debugging scenarios.

The goal of this CL is to add a single histogram that can be referenced
when debugging complex multiprocess scenarios. It will serve can serve
two purposes:
* Give us visibility into the magnitute of the problem
* Allow clients with tracing enable to capture snapshots for debugging

As an example how to use this new facility, a recent example of a
complex to track issue has been updated to reference this.
