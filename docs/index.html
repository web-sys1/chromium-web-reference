Last set of changes needed to build w/ Python3 by default.

This CL contains a few minor changes for Python3 compatibility
in the build:

1) It disables multiprocessing in mojom_parser, as the current
implementation is linux-only:

https://ci.chromium.org/p/chromium/builders/try/mac-rel/627173?

2) It marks :chromevox_test_messages_js as a python2_action
to keep several ChromeVoxBluetoothBrailleDisplayUIWebUITest tests
from failing and marks :ulp_serialized_to_static_c as a python2_action
to keep several component_unittests from failing:

https://ci.chromium.org/p/chromium/builders/try/linux-chromeos-rel/821190?

Note that there should be no functional changes due to this CL,
everything works as before under Python2 and even when we switch to
Python3 by default.
