Add chrome.enterprise.remoteApps extension API

This CL exposes the Remote Apps private Mojo API as an enterprise
extension API. The API allows an extension to add ephemeral apps to the
ChromeOS launcher. The API is supported on both Ash and Lacros.

This CL only implements the AddFolder, AddApp and DeleteApp methods. The
SortLauncher method will be implemented in a follow-up CL.

API proposal: go/remote-apps-extension-api
