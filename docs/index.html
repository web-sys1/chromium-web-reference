response is as expected. Nothing is written to the prefs until server
eligibility checking is enabled via Finch to avoid tainting the prefs.

- Sends authentication cookies (`credentials_mode = kInclude`)
- Prevents saving new tracking cookies (`net::LOAD_DO_NOT_SAVE_COOKIES`)
- Avoids SameSite restrictions (sets `site_for_cookies`)
