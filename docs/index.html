Guard ad detector hit-testing on kPrePaintClean lifecycle state

OverlayInterstitialAdDetector and StickyAdDetector call
HitTestNoLifecycleUpdate, which requires kPrePaintClean (enforced by
PaintLayer::HitTestLayer). However, MaybeFireDetection can be called
from RunPostLifecycleSteps when the lifecycle has only reached
kLayoutClean, causing a DCHECK crash.

Add a lifecycle state guard before hit-testing in both detectors.
