[Bookmarks Bar] Update scene layer background on changes

This CL updates how we handle background colors on the scene layer for
the bookmarks bar.

We update the handling of the background color on the bookmark bar to be
part of the view properties, and send it with the update method in the
Java SceneLayer. In the Coordinator, we add new observers to watch for
changes in the tab, such as moving to incognito mode, and for changes in
the theme, each of which will need to update both the Android widget
backgrounds and the SceneLayer backgrounds. We update the layer
container in the C++ code to be a SolidColorLayer using this newly set
background. We also make some updates to how we pass the resource ID
around to be more consistent with other layers and able to handle any
changes to the same. This CL fixes some of the UI blemishes with the
SceneLayer inherently.
