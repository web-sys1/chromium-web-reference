LacrosTablet: Adjust bubble if offscreen for Lacros

This CL sets PlatformStyle::kAdjustBubbleIfOffscreen true on Lacros.
If this is false, bubble position is not adjusted properly.

For example, crbug.com/1362816 is caused by this issue.
BubbleFrameView::GetUpdatedWindowBounds() didn't update the bounds to
proper position so that Lacros textfield bubble window is positioned
outside of the screen.

Bug: 1362816
