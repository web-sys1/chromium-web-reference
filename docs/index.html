[Mac Keyboard shortcuts] Flash menu titles for all command shortcuts

Chrome bypasses the normal Cocoa keyboard shortcut processing
machinery. As a result, some commands such as Cmd-N for a new window,
Cmd-T for a new tab, and others, do not flash the corresponding
menu title in the menu bar. This cl adds code that creates a menu bar
flash for these items. It also causes VoiceOver to speak these commands,
which wasn't the case before.
