[Web Payment] Use test installer in "change address" tests

Before this patch, the tests for delegating shipping address handling to
the payment app were installing the app through the JS
PaymentInstruments API, which is planned to be removed.

This patch changes the test to use the test-only app installer. The
paymentManager JS API is still being used for enabling the shipping
address delegation in the app.
