[floss] update bond and connect callbacks to provide status

Update OnCreateBond and OnConnectAllEnabledProfiles callbacks to
communicate the status or error results provided by the Floss daemon.
This would enable more refined pairing and connection metrics.

The BtifStatusToConnectErrorCode function moved to a shared header
file, as it is now required in multiple locations following this
change.

The Floss D-Bus manager is modified to provide an API version, in order
to support the API change.

The API version has been updated to 0.4.0
