[bfcache] Add a flag to evict on ax events

This CL adds a flag to evict bfcache entries on accessibility events.
This is necessary because we would like to keep enabling bfcache for
screen readers, but evict whenever AX events happen.

This CL is restoring what we removed in the CL below with a flag.
https://chromium-review.googlesource.com/c/chromium/src/+/3527935

Since the crash is only increasing on Android with bfcache expriment,
we enable this flag only on Android.
