[ReplacedNG] Make gradients fill their parent boxes

If a replaced element is the sole value of a pseudo-element's content
property, the pseudo-element is supposed to become a replaced element.
Blink doesn't do that. Instead, Blink's very loose implementation is to
make content-generated images fill the pseudo-element's box.

This patch implements this behavior in ReplacedNG.
