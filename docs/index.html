Regroup chrome_public_test_apk for android.

This CL moves chrome_public_test_apk from the original
android_specific_chromium_gtests to a separate group chrome_public_tests

Also create a new group for emulator:
android_emulator_specific_chrome_public_tests.
In this group, chrome_public_test_apk is split via "--gtest_filter"
into two test suites:
* chrome_public_test_apk: Filter out the tests that need playstore and
  wil run under google_apis images (userdebug type).
* chrome_public_test_apk_with_playstore: Contains only tests that need
  playstore and will run under google_play images (user type).

Via this split, the major tests in chrome_public_test_apk will run
under userdebug images and get a much faster test runtime
(670 mins vs the original 1000 mins)
