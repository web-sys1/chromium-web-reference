LayoutObject::ContainingScrollContainer() (previous EnclosingScrollportBox)

The new name follows PaintLayer::ContainingScrollContainerLayer().
The term "scrollport box" doesn't seem to be defined and/or used in
any css spec, and the term "scrollport" refers to the scrollable
content area in a scroll container.

It now calls PaintLayer::ContainingScrollContainerLayer() which is
more efficient than walking up the layout object tree.

The return type is changed to const LayoutBox* because we assume
LayoutBox provides scroll container functionality (though in real
world only LayoutBlock can be scrollable but that's not a constraint
of our LayoutObject classes).
