iwa: Downgrading IWAs via IsolatedWebAppInstallForceList policy

This change introduces the ability to downgrade an IWA to a lower version by using the new `allow_downgrades` field in IsolatedWebAppInstallForceList policy. For downgrading to be possible, it is also required to set the `pinned_version` field. Without it, the app will be uninstalled permanently.
