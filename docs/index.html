[Autofill] Remove bounding box parameter from AskForValuesToFill()

The parameter is effectively only passed to
AutofillClient::ShowAutofillSuggestions().

On non-iOS, the parameter is redundant because we call
AskForValuesToFill(..., field, field.bounds(), ...).

On iOS, we ignore the bounds in the iOS implementations of
ShowAutofillSuggestions() (and we pass a default-constructed
gfx::RectF to AskForValuesToFill()).

This CL removes the parameter. This propagates to
AutofillExternalDelegate and BAM::RefetchCardsAndUpdatePopup().

A followup CL will remove other redundant bounding box parameters.

A followup CL will introduce a new gfx::Rect parameter for
AskForValuesToFill() that represents the caret position rather
than the bounding box.
