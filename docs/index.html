WebUI: Fix @stylestic/no-duplicate-selectors CSS errors, part 2.

In this part fixing violations in ui/webui/resources/.

Duplicate CSS selectors blocks make the code harder to follow as the
reader has to read an entire file to understand which styles applied for
a single selector, often leading to duplicated styles or even bugs and
usually occurring in large files where it is easier for developers to
accidentally introduce them.

This is in preparation of enabling this check by default.
