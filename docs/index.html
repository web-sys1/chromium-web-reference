Remove blinkGcMetric from V8's browsing benchmarks

This CL removes blinkGcMetric from V8's browsing benchmarks. The
metrics therein have been obsolete for a long time and were removed by
https://crrev.com/c/4593451. They have been replaced by metrics
following the v8:gc:*:cpp naming scheme, which are located in gcMetric.

It also cleans up a few remaining occurrences of the obsolete category
blink_gc, that was removed in https://crrev.com/c/3251076.
