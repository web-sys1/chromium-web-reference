APS: Prevent crash on startup when Lacros is enabled

Web app installation when Lacros is enabled is not supported in APS yet.
However, the current implementation crashes on startup when Lacros is
enabled due to the missing WebAppProvider. This CL adds a simple check
to prevent the crash, as an interim measure until we implement Lacros
support.
