[FilesBanner] Add dismiss onclick handler for banners

Educational and Warning banners allow for users to dismiss them and not
have them shown for a predefined time period (usually 36 hours). This
will be set at a banner type level.

This also switches the hidden and aria-hidden attribute to not use
boolean values. CSS in Files app currently checks for the existence of
the hidden attribute (false or true) so switching to removeAttribute
and setAttribute empty instead.
