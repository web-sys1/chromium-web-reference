Fix leak in GuestOsSharePathTest.

GuestOsSharePathTest reinitialized ash::disks::DiskMountManager many
times and never called DiskMountManager::Shutdown(). Thus causing memory
leaks. Even without LSAN to catch the memory leaks, the
"DiskMountManager was already initialized" warning log messages were a
sign that something was wrong.
