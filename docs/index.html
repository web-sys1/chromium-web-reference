[Nearby] Add GetPublicCredentials on NearbyPresenceCredentialStorage

GetPublicCredentials returns stored Nearby Presence shared
credentials from the specified database (local or remote). The
platform implementation of the Nearby library, hosted in the a
sandboxed utility process, will call this method to retrieve
shared credentials that were previously saved to the user's
database by a call to the existing SaveCredentials method.

A full design doc is located at
go/cros-nearby-presence-credential-storage.
