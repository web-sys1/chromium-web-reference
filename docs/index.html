LockScreen: Add WebAppPublisherHelperTest to verify conversion to App

Conversion of web apps to App Service App structs was not unit tested!
Verifies WebApp with note-taking / lock-screen capability produces App
with note-taking / lock-screen intent filters.
Also verifies that ConvertWebApp produces same output as CreateWebApp.
Missing FileHandlers intent filter code was caught by tests and copied
from the non-mojom converter.

Misc: changes the flag for setting the lock screen URL in the WebApp
struct from blink::features::kWebAppManifestLockScreen to
features::kWebLockScreenApi because that is not part of the manifest
parsing process so can use the general API enablement flag.
