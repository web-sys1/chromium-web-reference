OptimizationGuide iOS push notification client side integration

Introduce OptimizationGuidePushNotificationClient, an abstract class
which clients that use OptimizationGuide's push notification
infrastructure inherit from. It achieves two things.
1) OptimizationGuide needs to know that the data for a URL has changed
upon a new push notification and to invalidate its cache, resulting in
new data being fetched on the next navigation.
2) It provides a mechanism for clients using OptimizationGuide's
infrastructure to acquire the
optimization_guide::proto::HintNotificationPayload from which
they can extract their own serialized payload and deserialize it.
