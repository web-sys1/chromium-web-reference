VaapiImageProcessorBackend: DCHECK GMB storage type.

This CL addresses [1]: we don't need to check two conditions:
frame->storage_type() == VideoFrame::STORAGE_GPU_MEMORY_BUFFER and
frame->HasGpuMemoryBuffer(). Instead, we can just check the latter, and
if that check passes, we know we're dealing with a
STORAGE_GPU_MEMORY_BUFFER VideoFrame and we can document that with a
DCHECK.

[1] https://chromium-review.googlesource.com/c/chromium/src/+/3170669/comments/3d1e0dac_f46b37b2
