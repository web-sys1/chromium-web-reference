Report mapped sizes in media::VideoFrameFeedback.

This CL updates WebRtcVideoFrameAdapter to report to the
media::VideoFrameFeedback which resolutions were mapped by WebRTC. This
gets forwarded to the capturers. On macOS, the capturer will make use of
this information to do Optimized Scaling of subsequent frames.

When this CL lands, Optimized Scaling on macOS is implemented behind
flags.
