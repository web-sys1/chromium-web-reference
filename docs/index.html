FLEDGE: add refcounted helper to manage context group IDs.

Matt Menke has found a subtle race in how they were previously used
(WorkletLoader could end up using an id after it's freed if a worklet
is unloaded really early in its lifecycle), and the ongoing work to
abort debugger pauses when the worklet is unloaded complicates its
lifetime further.

Therefore, this uses a helper with a refcount to make sure the ID
survives as long as it needs to.
