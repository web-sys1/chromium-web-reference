EarlyHints: Record custom page load metrics for preconnect

Before this CL, we record custom page load metrics for Early Hints only
for preload Link headers are received. We now have preconnect support so
it's better to record custom page load metrics for preconnect as well.

Affected metrics are:
* PageLoad.Clients.EarlyHints.Preload.PaintTiming.
  * NavigationToFirstContentfulPaint
  * NavigationToLargestContentfulPaint2
* PageLoad.Clients.EarlyHints.Preload.InteractiveTiming.FirstInputDelay4
