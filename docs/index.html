`onslotchange` IDL attribute is supported by interfaces:
- GlobalEventHandlers [1]
- ShadowRoot [2]

Reflects following spec changes:
- https://github.com/whatwg/html/pull/4129
- https://github.com/whatwg/dom/pull/785
