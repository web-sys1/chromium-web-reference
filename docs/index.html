[VCN] Make VirtualCardManualFallbackBubble sticky

After the change the bubble will not be dismissesd when losing focus,
but will still dismiss itself upon page navigation.

It is mainly for improving the case where multiple fields are not filled
correctly and the user has to click one field, copy/paste value, click
on the icon again to reshow the bubble and click another field. With
this change the bubble will stay during the manual filling.
