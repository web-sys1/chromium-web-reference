macOS Signing Scripts: Add support for notarytool.

Both sign_chrome.py and notarize_thing.py gain support for a selectable
--notarization-tool to choose between the existing altool mechanism,
or the newer notarytool. The CodeSignConfig can override the value or
provide a default one if it is not specified. The default remains
altool, but a subsequent CL will change that.
