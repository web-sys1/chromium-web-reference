[Extensions UI] Fix count discrepancy for allSites page and dialog

This CL fixes the extensions count discrepancy between the allSites page
and the edit site permissions dialog for the "customize per extension"
option.

It achieves this by:
 - only counting enabled extensions in both places
 - Changing the GetExtensionGrantedPermissions method to look at
   active permissions for policy installed extensions instead of
   looking in extensions prefs for granted permissions
 - Using GetExtensionGrantedPermissions in the dialog, for consistency
