Prerender: Add main frame navigation test for prerender-into-new-tab

This CL adds a test to run main frame navigation on a prerendered page
triggered for prerender-into-new-tab.

In addition to that, this CL modifies
PrerenderTestHelper::NavigatePrerenderedPage to use an appropriate
WebContents. Before this CL, the function assumed that a prerendered
page was hosted on the current visible WebContents, but this didn't work
for prerender-into-new-tab that creates a new WebContents to prerender a
page.
