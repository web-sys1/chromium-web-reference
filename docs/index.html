Projector: Fix screen recording length metric

Projector versions of the ScreenRecordingLength metric stopped
displaying properly on the UMA dashboard. This CL fixes the issue by
adding the {TabletOrClamshell} token to the histograms.xml and adds
test coverage for the Projector version of the ScreenRecordingLength
metric.
