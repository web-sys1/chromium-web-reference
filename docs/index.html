[fuchsia] Support fuchsia.component.* resolver and runner capabilities

Implement the new component framework's component-resolver and runner
capabilities.

The legacy component runner is re-implemented as a shim that delegates
execution of content to components launched under the new framework.

The new Component Framework API bindings increase the cast_runner_exe
binary size, which will be mitigated by completing the migration away
from CFv1 (issue 1065707) and removing the fuchsia.sys API dependency
as well as by migration to the new, more efficient C++ API bindings
(issue 1351487).

Bug: 1332972, 1065707, 1351487
Fuchsia-Binary-Size: Size increase due to new FIDL bindings dependencies will be mitigated by removal of legacy CFv1 dependencies, and by migration to the new, more efficient, C++ bindings, in due course.
