Add placeholder header for the unified side panel

This does not yet have a combobox for multiple entries (but shows a
placeholder label).

The header is based on the Lens side-panel header and leaves several
TODOs to look at cleaning this code up.

Note that the read-later side panel still has a header within its
content here, so there's a double-header thing going on. Optionally
removing this inner header in ReadLaterSidePanelWebView is ongoing,
which will let us host multiple copies (reading list, bookmarks, ...) of
this WebView inside the unified side panel.

Landing this with TODOs in place should hopefully make it more obvious
which next steps to take (cleanup, combobox, etc). These can all be done
in parallel.
