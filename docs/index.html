Populate in ActionChipsHandler the tab ID usable by Composebox

This CL changes the value populated into TabInfo in ActionChipsHandler
so the resulting value enables Composebox to pick the corresponding tab.

In doing so, the following changes have been also made:

- the logic in ActionChipsHandler has been changed so TabInterface becomes the main interface to interact with.
- the name of GenerateTabId method has been changed to GenerateTabHandleId so as to make explicit what value is returned.
