[fuchsia] Switch from DIR_MODULE to DIR_ASSETS

There are places in the code the use DIR_MODULE to load resources.
On many platforms, resources are in the same place as the module.
On Fuchsia, they are in a subdirectory so this breaks and DIR_ASSETS
should be used (which will function the same as DIR_MODULE on other
platforms).
