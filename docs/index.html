[Card Benefits][Metrics] Add submitted form event metric

We plan to migrate the selected form event enum,
`FORM_EVENT_SUGGESTION_FOR_SERVER_CARD_WITH_BENEFIT_AVAILABLE_SUBMITTED_ONCE`,
to a new enum in a benefit-specific form event enum class.

This CL adds `kSuggestionWithBenefitSubmitted` to a new benefit-specific
form event enum class. This CL also adds the logic to log this new form
event enum to two new histograms:
1.Autofill.FormEvents.CreditCard.Benefits
2.Autofill.FormEvents.CreditCard.Benefits.{benefit_source}
