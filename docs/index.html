Link Chrome Action to Tab + Enter key.

This change
- fixes the key handling, making sure that the ACTION_UP keys are not
  passed over to onKeyDown methods.
- allows the Autocomplete system to intercept and process key events
  before these events are passed over to the TextView.

To intercept keyevents, the code is duplicating the logic in View's
dispatchKeyEvent to work around situations where the
AutocompleteEditTextModel takes action before giving the key listener
a change.
