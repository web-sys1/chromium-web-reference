Add test assertions for the number of NewDocument calls

This adds temporary assertions that NewDocument is called by Content
Analysis Print tests so that it's easier to confirm crrev.com/c/3864429
fixes the issue of calling NewDocument. A call to NewDocument implies
that a print job has started, which is undesirable for content
analysis as it has side effects such as prompting the user, so
the added assertions can later be changed to confirm the issue
is fixed.
