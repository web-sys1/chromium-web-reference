Allow the "Clear" button to work for required date inputs

Prior to this CL, if the `required` attribute was set on any date/time
inputs, the "Clear" button on the picker would not function. That was
because IsValidValue() was used to check the validity of any setting,
and empty isn't valid. But we want the clear button to function, even
if it sets the value to something that won't validate on form
submission.
