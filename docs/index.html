AV1 VA-API encoder: Add robustness defaults for edge cases

- Handle framerate=0 in Initialize() by falling back to
  kDefaultFramerate instead of computing an invalid level.
- Fall back to Level 3.1 (index 5) when ComputeLevel() returns -1
  instead of failing initialization.
- Fall back to segment size 8 when GetMinAV1SegmentSize() fails or
  returns 0, preventing division-by-zero in segmentation map calc.
- Guard UpdateRates() against zero framerate or zero bitrate to
  prevent disrupting encoder configuration.
- Use uint64_t for bitrate_sum to prevent overflow with high bitrates.
