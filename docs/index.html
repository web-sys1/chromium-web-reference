Convert Thumbnail read/decompress to OnceCallback

This CL converts the paths that read and decompress thumbnails to use
base::OnceCallback and base::BindOnce instead of the generic base::Bind
and base::Callback.

It also notably changes a base::RepeatingCallback that seems to only be
called once into a OnceCallback.
