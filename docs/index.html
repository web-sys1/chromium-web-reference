arc: Update list of extensions supported by ArcFileSystemWatcherService

ArcFileSystemWatcherService maintains a list of file extension supported
by MediaProvider, and it skips sending media scan request to ARC when a
newly created or modified file's extension is not in the list.
The current list of supported extension is outdated as it was taken from
aosp-nougat codebase. This CL updates it with the supported extensions
in P, R and T.
