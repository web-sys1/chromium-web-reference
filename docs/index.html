media/mojo: add Mojo traits for StableVideoDecoder

This CL adds more traits needed for the StableVideoDecoder
Mojo interface (the rest will be added in follow-up CLs).
These traits cover the following mojom types from
media.stable.mojom:
- ColorSpacePrimaryID
- ColorSpaceTransferID
- ColorSpaceMatrixID
- ColorSpaceRangeID
- ColorSpace
- EncryptionScheme
- VideoCodecProfile
- VideoDecoderType
- VideoPixelFormat
- WaitingReason
