Add is_lazy flag to EventListenerInfo

EventListenerInfo didn't carry is_lazy flag. Callers have to assume lazy
listeners don't have a browser context, and depend on this assumption to
determine if teh info is for lazy listener. This small refactor removes
the unnecessary assumption callers had to make.
