ime: Add API to send IME settings to IME service.

Add an API to send IME settings to IME service. We send settings as part
of the OnFocus API because, so far, changes to the IME settings are only
observable when focused. If that changes in the future, we should add
another API for updating the settings directly.

Because Mojo unions are not extensible, we will create the Pinyin
and Zhuyin settings in anticipation of future changes, but they are
empty right now.
