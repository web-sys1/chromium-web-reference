Add localized AddError/AddWarning variants which handle placeholders

Implement overloaded AddError() and AddWarning() variants which support
passing placeholders along with the message ID. Since the error message
is retrieved by ID, it will also be localized.

This allows error messages to be used in the case of merging renamed
policies, whereas the string-based AddError cannot be used since it is
triggered before policy strings are loaded.
