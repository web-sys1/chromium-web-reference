Move the definition of PendingBuffer to vmo_buffer_writer_queue.h

VmoBufferWriterQueue has a std::deque<PendingBuffer> member. In the
latest libc++, std::deque calls sizeof on the template type when
instantiating the template class, so only having a forward declaration
of PendingBuffer in this file is insufficient and we need to move the
full declaration of struct in the header file.
