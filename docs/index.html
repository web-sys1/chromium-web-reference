Add return value to PasswordsNavigationObserver::Wait .

The return value indicates whether the event being await occurred. This allows tests to distinguish between Wait returning successfully vs due to a timeout.

Add nodiscard to force all users to check the return value.

See https://docs.google.com/document/d/1UtrsQ4nI6dV5V4FY3qo8q6jERvK95O8g4kB_q5FfE1Q/edit for more discussion of this common problem.
