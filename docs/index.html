kiosk: add Kiosk.SessionRestart.InternetAccess UMA metric

When a kiosk session is restarted, ChromeOS logs a restart internet
access info based on the previous kiosk session. The internet access
info is saved to the local state every hour. The Internet access info
contains information if device is online and if kiosk session app
supports offline mode, see |KioskInternetAccessInfo| enum values.
