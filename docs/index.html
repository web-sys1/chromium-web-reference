crostini: Reduced timeout for StartContainer and Start

We have heartbeat messages from Tremplin now, so let's lower the
timeout. Users semi-often hit this timeout (it's what you see if Garcon
doesn't start) so it's good not to hang for days.
We also lower the timeout for Start. While the step is almost a no-op,
we see users taking up to 90'ish seconds in this stage (a very
overworked DUT?) so pick 2 minutes as the timeout.
