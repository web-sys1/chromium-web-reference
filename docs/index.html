Enable policies loading without enabling sync for new signins

After the user sees the enterprise confirmation dialog and accepts it,
the pref "profile.account_management" is set to True, and allows
policies to be fetched and applied to the profile.

Enterprise profiles with "profile.account_management" is set to True are
not deleted when the sync enabling flow is cancelled.
