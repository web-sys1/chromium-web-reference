ParseCrlDistributionPoints: include all fields in output struct

Previously ParseCrlDistributionPoints only included the fullName URI and
did not return distribution points that contained fields unsupported by
the CRL checking code.  This change includes all the fields to allow the
same parsing function to be used in the certificate viewer, and updates
the CRL checking code to ignore distribution points that contain
unsupported fields.
