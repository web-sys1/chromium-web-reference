Complete Migration of WebSocket Basic Tests to `EmbeddedTestServer`

Summary:
This change completes the migration of WebSocket basic connectivity
tests from `SpawnedTestServer` to `EmbeddedTestServer`. The intermediate
`RunEmbeddedBasicSmokeTest` function was removed, and the test structure
has been simplified, restoring the original function names.

Details:
- Removed references to `SpawnedTestServer` for WebSocket basic tests.
- Deleted the intermediate function `RunEmbeddedBasicSmokeTest` used
  during the migration.
- Restored the original `RunBasicSmokeTest` function name.
- Updated `BasicSmokeTest` and `BasicSmokeTestSSL` to exclusively use
  `EmbeddedTestServer`.

Test Plan:
- Verified that all basic smoke tests pass using `EmbeddedTestServer`.
