BrowserLoader: Wait stateful lacros installation when enforced by policy

This CL lets BrowserLoader wait until stateful lacros to be installed
when the selection policy enforces the loader to load stateful lacros.

In production usage, lacros falls back to rootfs lacros when stateful
lacros is not yet ready and run installation of stateful lacros in the
background. However, for a debugging and testing purpose, the user can
set selection policy to enforce which lacros selection to use. In such
case, we cannot fallback to rootfs lacros, so we should wait the
installation to complete.
