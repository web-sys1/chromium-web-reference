Nearby: Disconnect bluetooth adapter mojo pipe on shutdown.

This change helps prevent a deadlock in the nearby utility process on
shutdown.  In some cases, there could be pending connection requests
that are blocking the utility process from shutting down cleanly.
Closing the mojo pipe terminates any pending requests, and allows the
utility process to shutdown without timing out.
