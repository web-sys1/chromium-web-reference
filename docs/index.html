dpwas: Support protocol handlers on MacOS - Add uri info to bundle.

This change adds protocol handler registration to the app shim bundle
generated with PWA's are installed. This allows MacOS to see that an
installed app can handle a specified protocol. There is a needed follow
up change to the app shim itself, to allow the app to receive and handle
the protocol.

Testing:
- Enable enable-desktop-pwas-protocol-handling
- Install https://fabiorocha.github.io/pwa/
- 'Show Package Contents' on ~/Applications/Chromium Apps/Progress Web App
- Then 'Contents/Info.plist' should contain entries for: web+testing,
  web+profile, web+github, web+extrastuff, mailto
