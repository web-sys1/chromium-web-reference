[Cast Streaming] Remove animated_playout_delay

This patch removes the animated_playout_delay field
from the cast configuration, as it is only used as an initial
value and is unnecessary--especially since adaptive playout delay
is currently disabled.

When we re-enable adaptive playout delay, the max_playout_delay can be
a good starting value with the potential to lower the playout delay if
conditions allow.
