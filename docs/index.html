[WebLayer] Replace isShowingUi() with mIsShowCalled

Context:
Before the CL, PRService#show asked the UI service to tell whether show
has been called. This is unnecessary and inaccurate, because this
state can be decided locally in PRService, with an attribute
mIsShowCalled.

This attribute is renamed and repurposed from
mIsCurrentPaymentRequestShowing. mIsCurrentPaymentRequestShowing had
a closed state which was redundant with mHasClosed, so mIsShowCalled
cuts this state.

Motivations:
* The BrowserPaymentRequest API becomes simpler.
* PRService's attributes' becomes more orthogonal.

Changes:
* mIsCurrentPaymentRequestShowing is broken down to mIsShowCalled and
  mHasClosed.
* BrowserPaymentRequest#isShowingUi() is removed.
* Added unit test for show being called twice.
