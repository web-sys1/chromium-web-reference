media/gpu/v4l2: Fix V4L2 Unittest minigbm resolution request

Previously the V4L2 Unittest would allocates buffers from minigbm from
an aligned resolution to match some devices. This resulted in a bug
caused by some devices allocating resolutions that were aligned
(Trogdor) vs those that didn't (Herobrine).

Update unit test to allocate buffers using the resolution provided
by the driver, bypassing the need to align.
