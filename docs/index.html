Reland "Reland "CSS highlight painting: bump HighlightOverlayPainting to stable""

This is a reland of commit 387b61732a4ca2f25d7aa5b376de0d373e4e72c2,
which was reverted because it caused a perf regression (bug 1335762).
Since we’ve more or less eliminated the regression in CL:3726437 (was
CL:3720688) and CL:3759693, the reason for revert should be resolved.

This patch is the same as the original change except:

• text-decoration-color-selection-001.html is no longer expected to
  fail, because the test has been removed from WPT (bug 1321106)

• highlight-pseudos-currentcolor-visited-computed-001.html refers to
  more precise bugs in its failure expectations (bug 1341621 or bug
  1147859 for disable-layout-ng, not bug 1295264)

• Some screenshot tests that needed rebaselining in the original have
  yielded different sets of expectations due to bot changes, but the
  substance of the rebaselines and the tests involved are the same
