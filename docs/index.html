[ios][PhishGuard] Map ShowWarningCallback to request instead of WebState

This CL addresses a comment in crrev.com/c/2693145.

There can be more than one ShowWarningCallback per WebState, since
there can be more than one request per WebState. It is better
to map callbacks to requests, than WebStates. This also simplifies
the code a bit.
