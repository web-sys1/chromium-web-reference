DeferredShaping: Fix crashes with a vertical writing-mode and minmax computation

Many tests using vertical writing-modes crashed by the following DCHECK
in ShapeTextOrDefer():

  DCHECK(IsHorizontalWritingMode(Style().GetWritingMode()));

because we missed to disable DeferredShaping for vertical writing-modes
in NGBlockNode::ComputeMinMaxSizes().
Note: NGBlockNode::Layout() already has the same
DeferredShapingDisallowScope.

This CL fixes hundreds of failing tests with enabling DeferredShaping
flag.
