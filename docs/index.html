Reland "wallpaper: fix broken daily refresh"

Added a small change to deflake the unit test.

Original change's description:
> wallpaper: fix broken daily refresh
>
> Since the system's color mode is set to auto by default, the system
> toggles between D/L mode daily, which also triggers setting the
> wallpaper. If the daily refresh is enabled, the wallpaper doesn't
> change but the refresh timer gets reset, causing the same wallpaper to
> be displayed indefinitely.
>
> Add a fix by letting callers specify whether to start daily refresh
> timer when setting a daily wallpaper. Also add a new test to verify this behavior.
>
