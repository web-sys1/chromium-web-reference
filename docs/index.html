Extract PasswordBubbleViewBase header into a thin GN target

Move password_bubble_view_base.h out of //chrome/browser/ui:ui into a
new thin target, //chrome/browser/ui/views/passwords:password_bubble_view_base,
so that bubble views in other directories can subclass PasswordBubbleViewBase
without depending on all of //chrome/browser/ui:ui.

This is a preparation step toward modularizing //chrome/browser/ui/views/webauthn,
whose passkey bubble views subclass PasswordBubbleViewBase.
