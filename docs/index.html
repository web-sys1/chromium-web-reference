ash: destroy ash dbus explicitly in ash shell

Ash dbus is initialized last when ash shell is initialized, so
destroy ash dbus explicitly at the start of ash shell dtor. This
explicit ordering allows ash dbus services to safely remove
themselves as observers of ash shell components.
