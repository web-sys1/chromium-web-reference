DLP: Disable blocked app service tasks

The CL enables graying out of blocked apps in the Open menu, needed for
DLP files restrictions.

To check DLP restrictions, when gathering a list of tasks, file manager
passes the entries and their source URLs which are kept in the DLP
metadata, while the app service, where the tasks are gathered, calls
the DlpFilesController to check if an app is blocked from opening a
file with any of these source URLs. This information is then passed back
to the file manager, and it then used to disable necessary menu options.
