[iOS] Add avatar API for ChromeAccountManagerService

This patch adds a method to simplify the way to get avatar. The caller
doesn't have to care about the default avatar if the real avatar has not
been fetched yet.

This API also helps to hide ChromeIdentityService from the rest of
Chrome code.

Added methods:
UIImage* GetIdentityAvatarWithIdentity(ChromeIdentity* identity,
                                         CGFloat size) const
