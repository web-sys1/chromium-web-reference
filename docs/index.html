[Extensions] Use BrowserContext::IsOffTheRecord in the webRequest API

The WebRequestEventRouter had a separate method,
IsIncognitoBrowserContext(), to determine if a browser context was
incognito. I suspect this was a carryover of the days when the
WebRequestEventRouter ran on the I/O thread (pre-network service), and
so we couldn't just use BrowserContext::IsOffTheRecord(). Now that the
WebRequestEventRouter runs on the UI thread, we can.

Remove WebRequestEventRouter::IsIncognitoBrowserContext() and the
corresponding is_incognito property on the BrowserContextData
struct and just use BrowserContext::IsOffTheRecord().
