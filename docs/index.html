chromeos: Add extension ID crash key for crosapi SetWallpaper() call

crosapi::mojom::Wallpaper::SetWallpaper() in ash-chrome implements the
Wallpaper extension API in lacros-chrome. If it crashes while doing
asynchronous work, it's helpful to know which extension caused the
crash. Use extension_function_crash_keys to track the extension ID for
the current in-flight request. It will end up in a crash key named
"extension-function-caller" similar to how extension functions in
pre-lacros ash-chrome track extension IDs.
