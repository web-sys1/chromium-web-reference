Add Histogram for Active and Inactive accessibility usage

This change adds histograms to measure the active and inactive time
spent using accessibility, using a pattern similar to the histograms
related to auto-disable. With this information, we can have a better
idea of how much time is spent with accessibility on but no API calls,
a situation we would like to avoid. This will also give a better idea
of common usage patterns.
