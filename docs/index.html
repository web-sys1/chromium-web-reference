tests: Fix ForceSigninForTesting handling

Some tests were calling SetForceSigninForTesting without calling
ResetForceSigninForTesting, which made the ForceSignin state leak
between tests, causing flakiness.

Added a ScopedForceSigninSetterForTesting which calls
ResetForceSigninForTesting when destroyed, to make it easier to
avoid this mistake.
