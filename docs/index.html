ignore the policy: #lacros-selection-ignore.

Original change's description:
> [Lacros] Load Lacros according to LacrosSelection policy.
>
> go/lacros-selection-policy
>
> Use LacrosSelection policy to decide which Lacros binary to load: rootfs
> or stateful. If the policy is set to user_choice or unset, the user can
> decide which one to load with lacros-selection flag, or load the
> freshest binary version wise.
>
> If the policy is set to rootfs or stateful, the corresponding Lacros is
> loaded and lacros-selection flag is ignored and hidden.
>
> The effective policy value does not change during the user session. The
> cached value is used for decision making. If the policy changes during
> the session, the new value will be applied on the next login.
>
