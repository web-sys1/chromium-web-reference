[iOS] Add AIM Omnibox client and its delegate

This change introduces AIMOmniboxClient, a new OmniboxClient
implementation for the AIM prototype on iOS. The client and its delegate
are responsible for handling omnibox interactions within the prototype.

It also introduces the `AIMOmniboxClientDelegate` protocol, which is now
implemented by the `AIMPrototypeMediator` to handle actions after
omnibox accepted text.
