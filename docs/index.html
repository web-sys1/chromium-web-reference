multiword: add OnActivate to AssistiveSuggester

This CL adds a new OnActivate method to the AssistiveSuggester. The
method has been added in anticipation for utilizing the new predictive
writing preference setting added in crrev/c/3313233.

The new setting is scoped per input engine id, and thus requires the
currently active engine id to lookup and inspect the predictive writing
preference.

The new method OnActivate simply captures the currently active engine id
and stores it for later use.
