[LayoutNGTextCombine] Hit testing with LayoutNGFragmentTraversal.

With LayoutNGFragmentTraversal enabled, the main NodeAtPoint() in
NGBoxFragmentPainter is called directly from HitTestAllPhases(), so we
wouldn't set up HitTestContext::text_combine.

To fix this, detect text_combine where it's used instead, in
HitTestTextItem().

This makes the following two unit tests pass when
LayoutNGFragmentTraversal is enabled:

  LayoutViewHitTestTest.TextCombineOneTextNode
  LayoutViewHitTestTest.TextCombineTwoTextNodes
