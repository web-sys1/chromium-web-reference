[bfcache] Update NotRestoredReasons before committing navigation

This CL adds code to update NotRestoredReasons when sending commit
navigation so that we send the right reasons to the renderer.

Before this we only populated certain reasons at DidCommitNavigation,
but now we cover all the reasons at SendCommitNavigation.

This CL also adds a test case that would fail without calling UpdateNotRestoredReasons in SendCommitNavigation.
