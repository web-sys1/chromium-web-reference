Remove access to fonts dir for web renderers

This CL removes access from web renderers to the
windows fonts directory - this is not needed as all
font access is proxied via the DWriteFontProxy.

Other sandboxes, including PDF renderers, can still
directly access fonts as this feature now only applies
to web renderers.
