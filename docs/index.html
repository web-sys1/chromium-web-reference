DisplayChangeObserver: Respect --force-color-profile

Currently this setting seems to be ignored on ChromeOS. It should
force chrome to treat the display as a fixed color profile regardless
of its actual properties. This is needed so that tast tests can check
what color was rendered in a screenshot without being color-profile
aware.
