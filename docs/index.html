Revert "youtubekids is also a valid youtube domain."

This reverts commit 60bb841482d0330a7aa6410856db9e68e485a053.

Reason for revert:
1) youtubekids.com does not behave like a regular Google owned domain,
so it does not use the regular Google auth stack. The fact that it was
listed as a regular Youtube and Google domain breaks the Chrome / Gaia
sign-in protocol.
2) the documentation of IsYoutubeDomainUrl() does not mention
youtubekids.com and its TLDs
3) it is unclear is the code using IsYoutubeDomainUrl() or IsGoogleAssociatedDomainUrl() was actually reviewed for the youtubekids
addition - comments like the one in IsGAIACookie() in file chrome/browser/ash/login/profile_auth_data.cc mention youtube.com, but not youtubekids.com

Note about patches:
* Patch 1 is the original 1.
* Following patches add regression tests.
