[ProtegoPing] Add a new metric to log protego pings over 28 days

This change defines a new histogram called
SafeBrowsing.Enhanced.ProtegoRequestSentInLast28Days which is similar to
SafeBrowsing.Enhanced.ProtegoRequestSentInLast7Days. The difference is
that the new metric will record a type of WithToken or WithoutToken if a
ping was sent any time in the last 28 days.
