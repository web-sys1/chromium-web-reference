Remove redundant sql::Database construction in InterestGroupStorage

InterestGroupStorage creates, but does not open, a database at 
construction. Then, if the database is not open, it creates the
database again in EnsureDBInitialized, replacing the previously
constructed instance. This change removes the redundant creation
in the constructor.
