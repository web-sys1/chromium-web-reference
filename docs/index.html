arc: Receive shortcuts from ARC apps

This CL adds ArcKeyboardShortcutBridge, which receives shortcuts
forwarded from ARC through mojo so that they can be shown in Chrome OS
Shortcut Viewer. As Chrome OS Shortcut Viewer is currently in the
process of deprecation and the new one will be ready soon with its
new AcceleratorConfigurationProvider interface, this CL leaves the
actual implementation of ArcKeyboardShortcutBridge empty.
