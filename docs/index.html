[SW+Prefetch] Honor skip_service_worker when serving

SW-controlled prefetches should not serve navigation with
skip_service_worker=true, e.g. when DevTools `bypass-service-worker` is
set.

Note: When prefetching, `skip_service_worker` is never set. Namely,
DevTools `bypass-service-worker` is still not honored when prefetching.
The current serving-time `skip_service_worker` check assumes
prefetching-time `skip_service_worker` is always false, and we should
revisit the check when we support prefetch-time `skip_service_worker`.
