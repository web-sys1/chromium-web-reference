[Incognito][NTP] Fix cookie controls layout

As part of cleaning up the revamped incognito NTP, a call to adjust the
cookie controls layout was removed by mistake. This issue affected
tablet layouts as they can span wider than the max expected width.

This CL reintroduces the call back in the text formatting method so that
upon inflation is complete the layout width is set appropriately.

The call to adjust the view in `onFinishedInflate()` is not sufficient
as the card is returned to be null and the additional call to format the
text fixes this issue.

Will merge back to M126 once tested in canary.
