[mullet m3] Updates BAM to emit fill after suggestion metrics

This CL adds the changes that will log metrics related to an user
seeing and selecting an Autofill suggestion that was triggered from the
context menu on an unclassified field or a field classified as
something different from the triggering field.

The implementation is based on the current metrics for unrecognized
autocomplete fields manual fallback.
