[iOS] Rename PriceTrackingPromo fieldtrial to PriceTrackingPromoIOS

This CL renames the PriceTrackingPromo field trial entry to
PriceTrackingPromoIOS to align with the Finch config filename. This
change ensures consistency in naming conventions between the field trial
and its corresponding configuration file.

Additionally, the CL includes the SegmentationPlatformTipsEphemeralCard
feature in the enable_features list. This inclusion enables Finch
FieldTrialConfig analysis to correctly identify and process this feature
within the PriceTrackingPromoIOS field trial.
