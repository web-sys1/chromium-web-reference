[Chromecast] Fix a potential race condition in AudioSocketBrokerTest.

The socket FD is default to non-blocking, which makes recvmsg return -1
immediately if there are no pending incoming messages. Make the accepted
FD blocking to make sure the message sent with sendmsg is guaranteed to
reach the receiving end.
