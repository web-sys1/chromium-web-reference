Add FilePath::FromASCII

Add a convenience factory method for instantiating a FilePath instance
from a string that's known to be ASCII.

This is intended to be useful in cases where the argument is known to be
ASCII - for example, when it's a hardcoded constant that cannot be a
file path literal because it's used as an std::string in other callsites.
