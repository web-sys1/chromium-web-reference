content-visibility: Don't expose NearestLockedExclusiveAncestor.

This function is being used to check for layout blocking and paint
blocking and maybe other cases too. However, when we move to the model
that allows some of the phases to happen, then it will be confusing
to figure out whether this is the right function to use.

This patch changes all uses to an explicit LockedAncestorPreventing*
functions.

Also cleans up unused functions.
