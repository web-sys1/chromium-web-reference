[Vertical Tab] Dont hide hover card on mouse exit for tab views

We are seeing hover card flickering in vertical tabs, and suspect this
is because we are hiding the hover card in
VerticalTabView::OnMouseExited (a deviation from the handling of hover
cards in the horizontal tabs counterpart).

Low-Coverage-Reason:HARD_TO_TEST This CL only adds one extra line to the
untested function. We do not yet have extensive testing support for
gesture events, and for now are satisfied with manual testing this
seldom used function.
