launcher: add UMA metrics for image search.

This CL adds several image search metrics.
1. As part of PnP request, we add a UMA to track the user image indexing
distribution and the indexing success rate.
2. We add more re-tries and add another metric to check when is the ICA
DLC available. The ICA initialization fail rate is high, and it is
suspicious that it is not ready until all re-tries have been attempted.
