[dpwa] Add a function to automatically add tests to appropriate test files.

Unit tests are added to check for the following scenarios.
1. Adding a new test to an existing test file.
2. Deleting the only test in an existing test file and then adding a new test to it.
3. If the test file doesn't exist, a message will be printed to console
that involves the file name, a reminder for the user to add the new file
to BUILD file and the new tests.
