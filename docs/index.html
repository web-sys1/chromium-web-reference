Remove components/sync/js/

This code is used to dispatch all events shown in the
chrome://sync-internals "Events" tab *except* onProtocolEvent, which is
fired from another codepath that also powers the "Traffic log" tab.
Such events haven't been observed by the sync team in quite a while so
we intend to remove the "Events" tab.
This CL removes the components/sync/js firing code. The UI itself will
be removed in next CLs.
