[iOS][Bookmarks] Update BookmarkNode pointer after moving

BookmarksFolderEditorViewController needs the updated node pointer
after a folder is moved across model. To support this use case (and
possible future cases where the updated pointer is needed), following
changes are performed:
* MoveToOtherModelWithNewNodeIdsAndUuids now has a return value.
* bookmark_utils_ios::MoveBookmarks and
  bookmark_utils_ios::MoveBookmarksWithUndoToast now get bookmarks
  in a mutable vector that is updated during moves.
* Tests are updated accordingly.
