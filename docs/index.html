Modify AfterStartupTaskUtils to use first paint instead of OnLoad

This change modifies AfterStartupTaskUtils to only consider startup
complete after the first paint event is fired instead of OnLoad.  This
will push startup out further where post startup tasks will run
resulting in less contention with page rendering.
