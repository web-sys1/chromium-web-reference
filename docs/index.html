Expose certain child nodes of focusable nodes on Android

This CL updates the approach to which nodes are exposed as leaf nodes
on Android.

With the current implementation, a focusable node will drop all of its
children except in a small number of edge cases. With this CL we expand
the number of cases for which the children are not dropped.

This change will update all affected nodes to allow swipe navigation
to explore their children.

AX-Relnotes: Children of certain focusable nodes will now be exposed for swipe navigation on Android.
