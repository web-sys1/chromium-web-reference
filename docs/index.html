hid: Treat detached buffers as empty byte sequences

In crrev/c/3269352 I added checks for detached BufferSources being
passed to sendReport() and sendFeatureReport(). It was then pointed out
to me that the WebIDL specification defines a different behavior for
converting a BufferSource to a byte sequence and so rather than throwing
an InvalidStateError DOMException the BufferSource should be considered
empty.

This change updates the implementation and tests.
