Webium: Avoid unnecessary vector copy in LoadAccelerators()

GetAcceleratorList() returns a base::span. Explicitly constructing a
std::vector from it forces an unnecessary heap allocation and a deep
copy of the entire array.

This CL updates WebUIBrowserWindow::LoadAccelerators() to iterate
directly over the returned span, avoiding the allocation and
improving efficiency.
