[Reading mode] Add ReadAnythingDistillationState enum.

Define ReadAnythingDistillationState to allow the
ReadAnythingController to keep track of the content returned by the
distiller. The OnDistillationStateChanged method notifies the
ReadAnythingController of distillation content changes, and calls
TogglePresentation() to update the presentation state from
kImmersiveOverlay to kSidePanel so the user can see the original
content.

The ReadAnythingController also keeps track of the distillation
content to determine if Reading Mode should be opened in Immersive
or Side Panel mode.
