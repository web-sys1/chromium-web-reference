[NewEV] Add File SHA256 Hash Signal

Created a hashing utility function that uses crypto::SecureHash to
incrementally calculate the SHA256 hash value of a given file's
bytes and returns the hash as a byte string.

This utility function is used by the FileSystemService to calculate
SHA256 values of files as request by the new getFileSystemInfo
SecureConnect Extension API.

The byte string is then websafe-base64 encoded (without padding) right
before being returned through the Extension API.
