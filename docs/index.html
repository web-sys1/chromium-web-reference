[A11y][fuchsia] Handle generated Scroll events

Implement `BrowserAccessibilityManagerFuchsia::FireGeneratedEvent` to
handle horizontal and vertical scroll events, by updating the node in
the Fuchsia tree.

This is needed because scroll events no longer trigger
`OnLocationChanged` after https://crrev.com/c/5878051. The linked bug
shows an example where bounding boxes are drawn incorrectly after
scrolling.
