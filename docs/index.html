[Identifiability Study] Fix tracing of surfaces

Tracing of IdentifiableSurfaces was introduced by
https://crrev.com/c/3842351. However, that required the
IdentifiabilityStudy to be enabled. This CL fixes this so that tracing
can be enabled independently.

Moreover, we make the trace events instant, since they do not have a
meaningful scope.
