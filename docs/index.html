Update PolicyStatusProvider classes to use base::Value::Dict

PolicyStatusProvider implementation classes as base::DictionaryValue is
being deprecated now.

base::Value::Dict instead of getting a pointer to empty
base::DictionaryValue to make the function more safe.

as the style guide PRESUBMIT test suggests. See https://groups.google.com/a/google.com/g/bling-team/c/5I31Gwrz2TE/m/FqeqFw_hAAAJ for more detail.
