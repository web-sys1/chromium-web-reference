Add tracing to NotificationManagerProxy.

We have noticed performance problems related to a media observer. We think that it might be related to the Android Binder when the observer calls the "Notify" function in the notification service. So to prove or disprove it, we need to add tracing to see how much time are spent in the "Notify" function.
