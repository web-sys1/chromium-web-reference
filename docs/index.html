Introduce a new ChromeOS.Lacros.ResumeTime metric

It's misleading to report ChromeOS.Lacros.LaunchTime when prelaunching
Lacros at login screen. As an example, if the user spends 30 seconds
at the login screen, those 30 seconds will be counted as part of the
launching time.

This change introduces a new metric, ChromeOS.Lacros.ResumeTime,
which measures the time from when the Lacros binary is resumed,
to when the browser service is connected instead.
