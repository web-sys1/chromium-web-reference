flags: remove offline-pages-alternate-dino-page & clean up tests

This change removes offline-pages-alternate-dino-page (which expired
in M76) and the backing OfflinePagesShowAlternateDinoPage feature,
which does nothing in production. It also removes the tests for
that feature, and a handful of other tests for analogous features,
because those tests are essentially change-detectors that cover
trivial glue code and don't test any actual logic.
