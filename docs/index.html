[lacros] Add lacros impl of ScreenCaptureNotificationUI

Today lacros shares the same ScreenCaptureNotificationUI impl
(screen_capture_notification_ui_views) with non-chromeOS platforms,
i.e. it displays a notification bar which appears as a separate
window in overview mode on ChromeOS; it's the issue this cl is about to
address.

This CL makes lacros align with Ash by adding
ScreenCaptureNotificationUILacros that no longer displays a notification
bar but a notification toast in system tray.
