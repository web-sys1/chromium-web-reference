Add MultiProfileDownloadNotifier abstraction

MultiProfileDownloadNotifier wraps multiple AllDownloadItemNotifiers and
handles the responsibility of creating and destroying those notifiers as
their DownloadManagers' Profiles are created and destroyed. Notably, a
MultiProfileDownloadNotifier has the capability of recognizing when a
profile that it observes spawns a child profile, and it will observe the
downloads from that profile as well.
