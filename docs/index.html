corner-shape: render all curved superellipse values

This uses a formula that approximates a superellipse using
bezier curves, written by fserb@.

The superellipse value creates 5 control points, that are used
to draw two bezier curves per corner.

Notch/bevel are special-cased, so the formula doesn't have to
deal with infinites.

Note that the WPT uses the same formula, but those were manually tested
to demonstrate that they produce results very close to a superellipse,
in a manner that works today on all browsers (using canvas).
