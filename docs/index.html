[mparch] Use PrimaryPageChanged() for TabLoadTracker

This CL is a part of the efforts to remove DidReceiveResponse(). It
is supposed to be called when we start receiving body of the response
but we need it anyway for the navigation to commit, so we can just use
PrimaryPageChanged() as we are interested only in non-same-document
primary main frame navigations.

It replaces DidReceiveResponse() with PrimaryPageChanged()
from ResourceCoordinatorTabHelper and TabLoadTracker and moves the logic
that updates `ukm_source_id_` from DidFinishNavigation() to
PrimaryPageChanged() and cleans up DidFinishNavigation().

For TabLoaderTest, it introduces SimulatePrimaryPageChangedIfNecessary()
to simulate PrimaryPageChanged() since the test is unit tests and it
doesn't get PrimaryPageChanged() from loading a page.
