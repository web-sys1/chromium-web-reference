Clang Plugin: Enable `use-classify-type` feature

This CL turns on the `use-classify-type` clang plugin feature, which was
landed in a previosu CL:
https://chromium-review.googlesource.com/c/chromium/src/+/3329384

This refactors the code for how the clang plugin determines if a type is
complex enough to require all ctors+dtors to be defined in a TU, which
makes writing "helper structs" require less boilerplate.
