[CADisplayLink #3] Add ExternalDisplayLinkMac and VSyncProviderMac to support CADisplayLink in the Browser process

ExternalDisplayLinkMac differs from CVDisplayLinkMac and
CADisplayLinkMac because it does not generate a CADisplayLink within the
GPU process. ExternalDisplayLinkMac obtains VSync from VSyncProviderMac,
and VSyncProviderMac obtains VSync from ExternalBeginFrameSourceMojoMac.

Only one VSyncProviderMac instance is created to handle all VSync
requests in both the VizCompositorThread and the GpuMain thread.

VSync Flow:
Mac CoreAnimation CADisplayLink ->
DisplayLinkMacMojo(VSyncThread, Browser process) ->
IPC (viz::mojom::ExternalBeginFrameController) ->
ExternalBeginFrameSourceMojoMac(VizCompositorThread, GPU process) ->
VSyncProviderMac(VizCompositorThread) ->
ExternalDisplayLinkMac (VizCompositorThread or CrGpuMain) ->
ExternalBeginFrameSourceMac (VizCompositorThread) or ImageTransportSurfaceOverlayMacEGL (CrGpuMain)
