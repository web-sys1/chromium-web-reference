[CrOS Bluetooth] Conditionally disable feature pod.

This change conditionally disables the feature pod depending on whether
the Bluetooth state can be modified. The feature pod is the only entry
point to the detailed Bluetooth view, so blocking access here prevents
inappropriate modification in that view as well.
