capture_mode: Fix UAF when session starts after sign out

It's possible, but super hard (repros only via a script)
to start a new session after the sign out button was pressed.
OnChromeTerminating() was already called before the new session
started, so the new session remain alive until a UAF is hit.

This CL fixes the issue and adds a regression test.
