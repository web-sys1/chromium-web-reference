[PDF Ink Signatures] Add PdfInkModuleClient::DiscardText()

in PDFiumEngine, add a method that is conceptually similar to
DiscardStroke(), but for text annotations. One major difference is the
need to call PDFiumPage::ReloadTextPage(), which gets interleaved
between the RemovePageObjectsFromPage() and DeletePageObjects() calls.
This is made available to PdfInkModule as
PdfInkModuleClient::DiscardText(), which will be called in the text
annotation editing process to modify or delete a text annotation.
