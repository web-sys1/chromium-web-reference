Fix Dynamic Type support in Inactive Tabs preamble

Prior to this CL, the Inactive Tabs preamble (the text appearing at the
top of the Inactive Tabs screen) was not resizing when the Dynamic Type
setting changed.
This was because the attributed string was not specifying a font. Now,
we specify the font as attribute to the text, and it works.
