webauthn: Draw a passkey icon over the QR code for pairing a phone

This adds an option to QRCodeGeneratorServiceImpl to request a passkey
icon to be drawn on top of the QR code center instead of the Chrome
dino.

The WebAuthn QR sheet view is changed to draw the passkey icon, use
the rounded locator style, and remove the QR code's outer border. These
changes are gated behind the
`device::kWebAuthnNewDiscoverableCredentialsUi` feature (default off).
