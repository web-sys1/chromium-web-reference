[Identifiability Study] Actively sample available fonts

This CL adds logic to actively sample which font families are
available on the system for the Identifiability Study. The fonts to be
sampled are configured via a feature param.

Sampling is performed in the browser process, on a low priority task
which is posted after browser startup.
