Safety check settings starts safety check right after settings is opened

We are working on bringing the Omnibox Pedals to Clank.
Currently, we are able to open the safety check settings, but cannot
run the safety check immediately upon landing on the settings page.
https://docs.google.com/document/d/14k513ufFh4cGU73L_V3bRuKvSu4L7hKRtMPThbJFVA8/edit?disco=AAAARHJoN0k

To match to the desktop pedals behavior, we want to run the safety check
immediately when the settings page is opened.

To implement this, this CL added a new Bundle argument for the safety
check settings.
