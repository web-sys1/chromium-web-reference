[CrOS WebAPKs] Add WebApkManager to determine when to install WebAPKs

This adds a new class, WebApkManager, which reads from App Service
to determine which apps should have a WebAPK installed. It currently
only reads the app list at startup, but it will be extended in the
future to catch new installations as well.

Code to actually generate and install WebAPKs will be added in
follow-up CLs.
