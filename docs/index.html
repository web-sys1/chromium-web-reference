[Tab Group Sync] Hook up frozen tab creation logic

This CL enables frozen tab creation logic to tab group sync.
1. For new tabs, a new tab is created and immediately frozen with
the URL and title.
2. For existing tabs, the tab is converted to a frozen tab as
well with the URL and title.

Also added a boolean to the TabCreationDelegate to be able to
create immediate / live tabs if required. Will be helpful to
reuse the mutation helpers for creation tab groups from recent
tabs and revisit surface.
