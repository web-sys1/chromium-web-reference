[CrOS Bluetooth] Add new BluetoothFeaturePodController.

This change creates the new BluetoothFeaturePodController and
conditionally initializes it in place of
BluetoothFeaturePodControllerLegacy when the kBluetoothRevamp feature
flag is enabled. This controller observes CrosBluetoothConfig instead of
TrayBluetoothHelper, but largely preserves much of the pre-existing
logic.
