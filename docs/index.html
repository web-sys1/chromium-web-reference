CCA: Don't crash Chrome when no mapped widget for CCA instance

For some rare use cases (e.g. Launch CCA via devtool inspector), there
will be no mapped widget for CCA instance and cause the Chrome crash.

Per discussion, we don't want to support such use cases since they
should be very rare. Therefore, this CL adds some handling logic to
avoid crashing Chrome under such use case.
