Replace optparse with argparse and made `--use-local-cache` the default.

This CL upgrades optparse for `ParseCommandLine` to argparse, as
optparse was deprecated since Python 3.2 and lacks features like
`add_mutually_exclusive_group`. With this CL, `--use-local-cache` is
now enabled by default, with the `--no-local-cache` option available
to disable it.

This CL also improved `PrintChangeLog` with simpler crrev.com link.
