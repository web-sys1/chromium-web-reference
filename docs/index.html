Use '#if DCHECK_IS_ON()' in c++ for WebDisallowedTransitionScope

This makes it obvious to code readers that the class is
available only when DCHECK_IS_ON(). If the class is mistakenly
used without DCHECK enabled, a clear error at compile time will
be given, instead of an obscure error at link time.

This also makes it easier to override the default DCHECK settings
e.g. on for a try job on bots, by modifying base/dcheck_is_on.h or
build/config/BUILD.gn temporarily.
