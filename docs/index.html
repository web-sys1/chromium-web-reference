[fuchsia] Update README with developer guidelines

Add a new section to fuchsia_web/README.md to help Chromium developers
reason about Fuchsia support during feature development.

The guidelines cover:
- The role of WebEngine on Smart Displays.
- Prioritizing Web Standards compliance (with hardware API exceptions).
- Favoring code maintainability and unified codebase over strictly
  stripping unused code (avoiding unnecessary is_fuchsia blocks).
- Handling platform limitations by prioritizing simplicity over performance.
