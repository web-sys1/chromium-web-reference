Remove redundant weather icon download.

This is a minor point, but it seems that we're effectively fetching
the weather in 2 spots:
1) When we fetch new topics, the weather seems to also be returned
   in the response.
2) A dedicated weather fetch.

This means we're downloading the weather icon twice when the
screensaver starts. Since the dedicated weather fetch is sufficient
and we have a refresh timer that runs every 5 minutes, it seems that
we can ignore the weather in 1)'s response so prevent an extra
download. I noticed this because we're going to add more topic
fetches in the future (a landscape one and portrait one), so that
may add yet another unnessary weather icon download.
