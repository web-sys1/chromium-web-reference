ChromeVox Tutorial: Create separate component for lesson container.

The ChromeVox tutorial is implemented as one large polymer
component. This makes writing tutorials for other accessibility
extensions difficult; each new tutorial would have to be written from
scratch and much of the glue code would have to be duplicated.

This change is a pure refactor and shouldn't cause any change in
behavior. For more details, please see the following design doc:
https://docs.google.com/document/d/12vNcRGUkMc
CSDpYk6Vc-s3uPZNbpra5B2wrMRLVn_ew/edit?usp=sharing
