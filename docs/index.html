Add WeakPtrFactory::GetMutableWeakPtr()

Update the WeakPtrFactory interface from:
  WeakPtr<T> GetWeakPtr() const;
to:
  WeakPtr<T> GetWeakPtr();
  WeakPtr<T> GetWeakPtr() const;
  WeakPtr<T> GetMutableWeakPtr() const;

See Chromium cxx discussion: https://groups.google.com/a/chromium.org/g/cxx/c/KQNaPvB_3tg

Once all uses of `WeakPtr<T> GetWeakPtr() const` are replaced with
`WeakPtr<T> GetMutableWeakPtr() const`,
`WeakPtr<T> GetWeakPtr() const` will be replaced with
`WeakPtr<const T> GetWeakPtr() const` to help enforce semantic
const correctness.
