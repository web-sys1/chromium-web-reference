Migrate away from binding callbacks to base::DoNothing.

This is actually somewhat inefficient: base::DoNothing() generates a
callback, which is then wrapped by another callback. Calling Run() on
the result callback requires jumping through two thunks before actually
dispatching the bound functor (which, of course, does nothing). Convert
the various uses to use a lambda instead.
