focus-mode: Implement media control for selected playlist

This CL will add the following changes. After creating the media widget
for the selected playlist, once it gained the audio focus, we can get
its media session request id. Then, we bind the media controller
observer with the request id, so that we can observe the state changes
for the specific media session.

When the playlist starts playing, the equalizer animation will start
playing. When the user press pause button through the media controls
on the shelf, when entering the focus panel again, the playlist will
shown in the pause state. When the ending moment comes, the playlist
will be selected state.
