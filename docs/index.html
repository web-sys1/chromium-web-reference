[Blink] Restrict low-latency flow to WebGL/Canvas2D

HTMLCanvasElement::LowLatencyEnabled() just checks the `desynchronized`
attribute, which can be set for all context types. However, a
low-latency specific flow is implemented only for WebGL/Canvas2D. [1]
introduced a regression wherein this flow could be entered for other
context types. This CL fixes that regression.
