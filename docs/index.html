[Media Tabs] Display media alert indicators in Tab Search + tests

This CL adds the initial code to display media alert indicators on tab
search items. This CL focuses mainly on the logic needed to get the
correct indicator to display. These alerts will only be displayed if
#tab-search-media-tabs flag is enabled. This CL also adds a test for
displaying the media alert indicator. Currently UI only has specs for
supporting Media recording and audio playing and muting so only those
cases are included.

The following still needs to be done:
- Unify resources with tab_strip/ alert indicator when UI finializes
design. We may want see if we can move shared resources to
ui/webui/resources/cr_components
- Formatting and coloring for alert indicators such as keeping the alert
indicator in a fixed state.
