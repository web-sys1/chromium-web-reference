[copy_or_move] Add VLOG to filemanager delegate hook reporting

Follow-up to crrev.com/1090001. Add VLOG to the file manager delegate
hook callbacks emitted during a copy_or_move operation to aid debug.

The logging shows that regardless of whether a source URL is local or
not, the delegate hook callbacks happen for the source, including any
child items of the source if it is a directory [1].

[1] This is useful to know say if the files app conflict dialog needs
to add merge dialog option.
