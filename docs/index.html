[css-scroll-snap-2] Implement snapchanging for programmatic scrolls

snapchanging[1] should fire as soon as the user agent determines new snap
targets to which it will snap, letting web pages know ahead of the
actual snapping.
This patch enqueues "snapchanging" events when a programmatic scroll
that causes a change in a snap container's snap target occurs.

[1]https://drafts.csswg.org/css-scroll-snap-2/#snap-events
