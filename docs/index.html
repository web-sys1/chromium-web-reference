Implement Global Ctrl-Tab MRU switching.

This feature allows users to toggle between their two most recently used
tabs using the Ctrl-Tab shortcut. The logic is extended to search across
all browser windows within the same profile, ensuring that tearing off a
tab into a new window still allows for seamless MRU toggling back to the
original window.

Key Changes:
- Core Logic: Updated SelectNextTab in browser_commands.cc to find the
globally most recent tab using ProfileBrowserCollection.
- Focus Management: If the MRU tab is in a different window, that window is brought to the foreground before tab activation.
- Settings: Added a toggle in Appearance settings to enable/disable this behavior (gated by kCtrlTabMru feature flag).
- Metrics: Adds a daily UMA metric TabStrip.Tab.KeyboardTabSwitchMode to record the keyboard tab switch mode.
- Testing: Added a unit test in browser_commands_unittest.cc to verify the MRU logic and added tests in tab_stats_tracker_unittest.cc for the new metric.
