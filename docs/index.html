Introduce ui::ClipboardUrlInfo for multiple URL drag and drop

This change introduces the ui::ClipboardUrlInfo structure to support the
drag and drop of multiple URLs. It's part of a larger effort to enable
setting multiple URLs for the text/uri-list drag type, detailed in this
CL: https://chromium-review.googlesource.com/c/chromium/src/+/6608042.

For more information, please refer to the design document:
https://docs.google.com/document/d/1zNvn2hQOfMCc9-L9_Wt-Q4rBhmSAQHhUYWMDdNKLCGU/edit?usp=sharing.
