RawDraw: implement recording side of raw draw

This CL adds features::kRawDraw, and implement related logic in
RasterDecoder. So when RaWDraw is enabled, the RasterDecoder will use
SharedImageRepresentationRaster to access the backing, and store paint
ops for tile in the shared image backing. When the backing is about to
be consumed, the paint ops will be executed against the texture in the
shared image backing, and the backing texture will be used for
compositing.
