ozone/x11: reset display with ScopedEnvironmentVariableOverride

Given USE_X11 is deprecated in favor of Ozone/X11, and
we are working on stopping to leak X11 code outside
Ozone/X11, use the ScopedEnvironmentVariableOverride
to unset the DISPLAY env variable.

In this particual case, GlDisplayEglUtil will delegate
unsetting the DISPLAY variable to platforms, which
return an uniform ScopedEnvironmentVariableOverride.

This helps to avoid leaking X11 includes outside
X11 folders.
