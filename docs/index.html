[a11y][NTP] Fix focus on deletion of final tile

The issue is that it tries to update the focus before the tiles have
actually been updated. This didn't cause a problem with the non-add
tiles because the tile disappearing focus is actually handled by either
something in polymer or the browser. tileFocus_ was actually only making
it so the focus was switching from the menu to the tile. However, it
would not switch to the add tile, since the tileElements_ list was still
the original length.

To fix this problem, I changed it to setting focus with
afterNextRender().
