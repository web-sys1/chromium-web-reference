Allow projects embedding Chrome to disable libc++ hardening.

This moves enable_safe_libcxx to build_overrides so that external
projects embedding Chrome can disable libc++ hardening if deemed
appropriate for those project's requirements.

In order to ease the transition for projects that utilize Chrome's
//build, c++.gni still defines enable_safe_libcxx as a fallback for
now. Once various external projects have picked up this change, the
fallback will be removed.
