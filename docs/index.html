Fix NetworkHandler init in SimpleGeolocation tests

This CL removes direct NetworkHandler initialization in
SimpleGeolocationTest and SimpleGeolocationWirelessTest
and replaces it with NetworkHandlerTestHelper usage.

This is being done in preparation for enabling eSIM cellular
network features in NetworkHandler. This requires additional
initialization in tests that use NetworkHandler.
NetworkHandlerTestHelper ensures that all required DBus clients
and NetworkHandler singleton is initialized properly in tests.
