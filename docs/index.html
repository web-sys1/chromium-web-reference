borealis: Move blocking dev-mode check to an async method.

The check "is you device in dev mode", which we use to prevent
verified-mode devices from accidentally using external disks on
borealis, requires blocking IO but was being performed in a nonblocking
context.

We refactor the code to call the method asynchronously in a MayBlock()
task environment, with the result handed back to the nonblocking thread.
