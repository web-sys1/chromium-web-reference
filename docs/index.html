bruschetta: Use edk2-ovmf-dlc when launching

Because this is intended to be the final state, also remove the legacy
code to support older configurations where the firmware image and/or
pflash files were stored in the user's Downloads directory.

Users that installed after we moved the pflash files to concierge
should automatically migrate to the UEFI DLC. Users from before then
will not have a persistent pflash file, which will probably mostly
work for now, but won't e.g. enforce secure boot.
