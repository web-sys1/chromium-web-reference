[headless] Added --disable-pdf-tagging switch

Since https://crrev.com/c/3255921 headless Chrome always
generates tagged PDFs. Unfortunately, this sometimes results
in substantially increased PDF file size.

This CL introduces a command line flag that allows affected
headless users to disable PDF tagging.
