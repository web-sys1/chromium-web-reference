Show Handwriting hover icon when pointer can be adjusted to start writing

This CL implements showing the stylus writing Pointer icon when hovering over an area where pointer down can be adjusted to start writing in a nearby writable input.

Stylus pointer adjustment was introduced recently in crrev.com/c/4085362 and this CL adds showing the icon for the adjusted cases. Calculate the same hit test writable node nearby and set the pan action as writable to show the handwriting icon.

When hover action becomes not writable again, blink may not send another cursor update as element under pointer could be same. Cache the previous
cursor icon and reset Pointer to that icon.
