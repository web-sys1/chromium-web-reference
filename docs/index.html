media/gpu/v4l2: migrate to finalized VP9 stateless decoder uAPI

There was a recent update with the V4L2 kernel driver (b/226978629) to
support the final VP9 stable API landed on kernel 5.17, which made the
vp9-ctrls.h obsolete. Thus, vp9-ctrls-upstream.h header file is needed
instead to fix it. This CL fixes build errors after using
vp9-ctrls-upstream.h. Another CL will follow to wrap things up to
enable the VP9 stable API.
