Run a subset of identity_apitests on Android desktop.

desktop:
1. IdentityGetProfileUserInfoFunctionTest (integration test for
chrome.identity.getProfileUserInfo)
2. IdentityGetAccountsFunctionTest (integration test for
chrome.identity.GetAllAccounts)
3. OnSignInChangedEventTest (integration test for
chrome.identity.OnSigninChanged notifications)

implemented by custom JS bindings)
