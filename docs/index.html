[bfcache] Use pagehide instead of unload in messaging API test and navigation browsertest

In nagivation_browsertest.cc, it is safe to replace unload with pagehide because pagehide gets fired before the deletion of a window, which is the same as unload.

In messaging_apitest.cc, it is also safe to replace unload with pagehide because the message should be sent once the tab closes, which is when pagehide also gets fired.
