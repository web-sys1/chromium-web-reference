Clean up unnecessary usage of ash::UserMetricsRecorder (CL 4)

UserMetricsRecorder exists to record periodic metrics. It has a
deprecated method that records base::UserMetricsAction metrics.
It's better to record those sorts of metrics close to the code that
implements the action. Eliminate the method and inline the metrics
recording.

Eliminate the user_metrics_recorder.h include from shell.h and
switch to forward declaration. Fix IWYU issues this reveals.

No behavior or metrics changes, just incidental cleanup.
