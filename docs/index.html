[Lacros] Show the welcome screen in the first run for primary profile

This CL adds logic into ProfilePickerTurnSyncOnDelegate for the primary
profile first run. For the interim, it shows the enterprise welcome
screen instead of the lacros welcome screen. This will be fixed by
follow-up CLs.

This CL introduces a separate metric
Profile.LacrosPrimaryProfileFirstRunOutcome to track the outcome of this
flow. It reuses the same enum as Profile.AddSignInFlowOutcome, only
renames it to reflect the now broader scope.
