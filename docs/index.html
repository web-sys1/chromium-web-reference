[GridNG] Fix usage of invalid reference in GridItemsIncludingSubgridded

After we call `ReserveCapacity` to append the subgridded items of a
subgrid, `grid_items` is effectively reallocated, making any previously
created reference to its grid items invalid. This patch fixes the issue
by ensuring no reference is used after we reserve the new capacity.
