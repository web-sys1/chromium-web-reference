Replace ListHashSet with a Vector wrapper class in CoreProbeSink.

ListHashSet is deprecated in favor of LinkedHashSet.
The main difference is that ListHashSet allows modification of the container during iteration, while LinkedHashSet does not.
Modification during iteration happens in InspectorDOMDebugger because the DevTools session may get detached while paused at a breakpoint.
Instead of making a copy before at every time before iteration (crrev.com/c/1584109), AgentRegistry is implemented as a Copy-On-Write vector to avoid copy overhead.
