[mathml] Add fuzz target for StretchyOperatorShaper

This CL adds a new fuzz target [1] for the StretchyOperatorShaper class.
It currently calls each API function but may need future adjustments.
Using the corpus of math fonts from WPT [2] helps fuzzers find existing
paths within StretchyOperatorShaper.
