service worker: Support retry for disabling storage

This CL adds retry support for ServiceWorkerStorageControl::Disable().
Disable() is changed to return a response so that ServiceWorkerRegistry
can check whether the call is finished or not.

This CL also removes Disable() direct calls from tests to reduce
direct dependencies of ServiceWorkerStorageControl. It helps us to
encapsulates ServiceWorkerStorageControl with the retry mechanism.
