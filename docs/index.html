[Autofill] Add UMA Autofill.DataUtilization.{All, Selected}FieldTypes.*

The metric records if a field was autofilled and not edited on form
submission, or if the field's final value was entered manually by the
user and found in their stored address profiles / credit cards. The
latter includes cases where Autofill was used before the user edited
the field. Emitted on form submission.
