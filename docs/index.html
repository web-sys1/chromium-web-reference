Fixed Text Decoration Offset Calculation

The WPT test for zero text underline offset [0] is failing.

Based on the references spec [1], when text-underline-offset is zero,
the underline should be drawn at the alphabetic baseline.

The test is failing because the underline is drawn below the baseline,
because of a gap that gets added between the baseline and the offset.

The proposed fix is to have no gap
when the underline offset position is fixed.

[0] wpt/css/css-text-decor/text-underline-offset-zero-position.html
[1] https://www.w3.org/TR/css-text-decor-4/#line-offset-zero
