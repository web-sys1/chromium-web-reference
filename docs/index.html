OOBE: Update Network Error metrics

* Use patterned histograms to consolidate OOBE.NetworkErrorShown.*
  and OOBE.ErrorScreensTime.* metrics.
* Extend the expiry date for OOBE.NetworkErrorShown.* and
  OOBE.ErrorScreensTime.* metrics.

  OOBE.ErrorScreensTime.* metrics.

  OOBE.ErrorScreenTime.* metrics, since they are emitted but not defined
