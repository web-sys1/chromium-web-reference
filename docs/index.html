gpu: Add GetUsageForMailbox on SharedImageManager

This change exposes a function SharedImageManager::GetUsageForMailbox.
SharedImageFactory already has a GetUsageForMailbox method, but it only
searches mailboxes that the SharedImageFactory created. The
SharedImageManager knows about all mailboxes.

This change also makes the existing SharedImageFactory variant fall back
to the SharedImageManager version.
