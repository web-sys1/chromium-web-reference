Fix policy validation errors in M145

This CL removes the creation of the CloudPolicyStore for Extension install policies for Chrome versions under M146 since the feature is not supported on those versions.

This fixes policy validation errors that have been fixed in M146 and that appear in M145.

Additional checks for the feature being enabled are added for the stores created at user and profile levels. Those checks cannot be added on the machine level because the store is created before finch is initialized.
