Temporarily disable JSON modules in ServiceWorkers

To complete JSON module support in ServiceWorkers with import
assertions, there is still some nontrivial work to do to plumb expected
module type across to the Browser process and store it in the
service worker database.

In order not to block overall JSON module support on this issue, we can
disable JSON modules in ServiceWorkers, ship JSON modules initially
without support in ServiceWorkers, and later enable them in
ServiceWorkers as the necessary work is complete.

To that end, this change removes the conditions in the Browser and
Renderer processes that would allow a JSON module to be imported in a
ServiceWorker.
