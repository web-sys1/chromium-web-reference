Update Inline Reply for CrOS

As a part of the notification refresh effort...
1. A slight refactor to enable the refresh for CrOS
 - Pull out the inline reply(NotificationInputContainer) from
   NotificationViewBase.
 - Add an Init() to NotificationInputContainer to ensure derived
   class functions are called during view construction.
 - Move a function used by NotificationInputContainer and
   NotificationViewBase into a notification util file.
2. The meat of the UI Refresh changes for CrOS
 - Create an AshNotificationInputContainer for chromeos notifications
   with styling that matches the spec.

In another CL I will...
 - Change the send icon
 - Change focus highlights
 - Adjust actions row font and spacing
 - Animations for to/from showing inline reply
