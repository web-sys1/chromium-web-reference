Add is_interrupt boolean to OnActionRequested signature

This allows implementers of the API to be able to know whether the action being executed is an interrupt without having to keep state through the `OnInterrupt*` methods. It also makes it more appearent that interrupt ExternalActions can happen. See (go/nested-external-actions) for more context.
