SpeechRecognitionPrivate: Implement the onError event

This change implements the onError event, which is dispatched to
the relevant extension when a speech recognition error occurs.

Additionally, this change adds code in FakeSpeechRecognitionManager
to send fake speech recognition errors. The implementation mirrors
the pattern used to send a fake speech result. The new function is
then used from the API tests to verify error handling and behavior.

For more technical details, please see the design doc attached to the
bug.
