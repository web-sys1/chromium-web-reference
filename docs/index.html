Do not build printer_manager_dialog_linux.cc on Chrome OS

This file contains Linux-specific code that is not applicable to Chrome
OS. The reason it is built on Linux is because the `is_linux` GN build
variable used to imply `is_chromeos`. Now that the 2 variables have been
separately, it is more obvious that the current rule is incorrect.
