bindings: Introduce ScriptWrappable::ToMostDerived (and DowncastTo)

Since the following pattern is error-prone,

  ScriptWrappable* script_wrappable = ...;
  if (script_wrappable->GetWrapperTypeInfo() ==
          V8Foo::GetWrapperTypeInfo()) {
    Foo* foo = script_wrappable->ToImpl<Foo>();
  }

where we use V8Foo for type check and Foo for casting, and
ToImpl<Foo> doesn't perform any type check, so it's possible that
a type mismatch (like V8Foo and Bar) would happen.

This patch introduces the following pattern to prevent a possible
type mismatch.

  if (Foo* foo = script_wrappable->ToMostDerived<Foo>()) {
    // foo->something();
  }
