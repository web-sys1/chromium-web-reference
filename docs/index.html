[DomStorage] Run session_storage_impl_unittest.cc with SQLite

LevelDB backends. Updates the `CorruptionOnDisk` test to replace the
SQLite database file with plaintext to simulate corruption. Implements
`SessionStorageSqlite::MakeAllCommitsFailForTesting()`, which simulates

`SessionStorageSqlite::UpdateMaps()` by force returning a
`DbStatis::IOError()`.
