[Sync] Suppress NTP promo after X impressions behind a feature flag

This CL adds the feature flag `SyncAndroidLimitNTPPromoImpressions` and
adds `SyncAndroidNTPPromoImpressionsLimit` as a feature parameter, then
it suppresses the NTP promo after `SyncAndroidNTPPromoImpressionsLimit`
impressions when the flag is enabled, currently the feature flag is
disabled by default.
