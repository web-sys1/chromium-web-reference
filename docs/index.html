[webauthn] Fix PasskeyModel for guest profiles

PasskeyModel instantiates the AffiliationService on creation so it can
report passkey creation. However, the AffiliationService does not expect
to be instantiated in guest profiles, and it DCHECKs. This CL updates
PasskeyModel to avoid doing that.
