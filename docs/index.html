shortcuts: Hide/remove some duplicate accelerators from display

* Raw accelerator data from kAcceleratorData may result in the same
  shortcut being displayed multiple times in the frontend.

* e.g.: For TOGGLE_APP_LIST, accelerators includes:
    - VKEY_BROWSER_SEARCH.  => search_icon
    - VKEY_BROWSER_SEARCH + EF_SHIFT_DOWN => search_icon + shift
    - VKEY_ALL_APPLICATIONS  => search_icon
    - VKEY_LWIN => Meta
    - VKEY_LWIN + EF_SHIFT_DOWN => Meta + shift

* This CL use GetHiddenAcceleratorMap() to hide certain duplicate
  shortcuts that have a similar behavior as the TOGGLE_APP_LIST.
