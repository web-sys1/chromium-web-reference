WebNN: Implement the synchronous compute method of MLGraph

This CL implements MLGraph::ComputeSync() that is called by MLContext to
implement the computeSync() method for the dedicated worker.
ComputeSync() has the similar function as ComputeAsync(). The difference
is if there are no validation errors, it calls ComputeSyncImpl()
implemented by an MLGraph backend, such as MLGraphXnnpack, and executes
the compiled platform graph synchronously in the caller's thread.

This CL also implements MLGraphTestBase::ComputeGraph() that helps test
the validation steps for both asynchronous and synchronous compute
methods.
