Record Autofill.SkippingFormFillDueToChangedFieldCount histogram

If the number of fields to be filled in
BrowserAutofillManager::FillOrPreviewDataModelForm is different than the
number of fields registered in the form cache, the BAM skips filling the
form. This CL adds a UMA metric to count the prevalence of this. Also it
adds chrome://autofill-internals logging for better discovery.
