Lacros: Support checking out specific toolchains for compiling Lacros

Currently, Lacros uses the simple chrome cros toolchain that is updated
on a roughly daily basis, however, there has been quite a few unfixable
issues due to version skew that results in release blockers.

This CL implements a short term workaround to allow Lacros use a
different version of toolchain for compilation.

This CL depends on: https://crrev.com/c/3308954.
