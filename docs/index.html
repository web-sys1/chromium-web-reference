Correctly handle move to another display

OnWindowBoundsChanged may be called with post-adjusted bounds when
the window is being moved to another displayy (parent window), which
sends the wrong bounds to the wayland client.

With this CL, ash-chrome/exo updates the bounds correctly when
the window is moved to another display.
