[white-space] Fix `NOTREACHED` when `CSSWhiteSpaceShorthand` is off

This patch fixes the serialization may reach `NOTREACHED` when
the `CSSWhiteSpaceShorthand` runtime flag is off.

The failure was found in tests for when the `insertParagraph`
command copies styles, which ensure changes in `EditStyle` is
as expected, such as crrev.com/c/4378117.

The behavior of `execCommand` is not defined[1] that this test
is not in `external/wpt`.

[1] https://w3c.github.io/editing/docs/execCommand/
