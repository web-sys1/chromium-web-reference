Reset FontPerformance timing accumulators after recording to histogram

This patch changes |LocalFrameView::OnFirstContentfulPaint()| to call
|FontPerformance::DidReachFirstContentfulPaint()| to reset timing
accumulators to record each page navigation of main frame instead of
accumulating during renderer process life.

Before this patch, recorded timings are accumulated during process life
time. Thus, Nth recorded timings is sum of 1st to (N-1)th. We want to
have timing of each FCP instead of such timing.
