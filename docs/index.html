blink::Color: Add ToSkColorDeprecated

This was previously a cast operator, which made sense when blink::Color
was just an RGBA32 (aka uint32 aka SkColor).

Change this from a cast operator to a function. Give it the name
ToSkColorDeprecated, because we should never be producing an SkColor
from a blink::Color (we should always use SkColor4f).
