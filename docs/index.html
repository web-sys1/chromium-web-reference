Don't compile cronet_test_instrumentation_apk on non-Cronet bots

Compiling Cronet requires a special set of GN args. Non-Cronet bots
don't use these (they use Clank's), so it doesn't make sense for them
to list cronet_test_instrumentation_apk as an additional compile
target.

Listing cronet_test_instrumentation_apk there also stops Cronet from
hiding it's GN targets from non-Cronet builds.
