[PA] Move `use_partition_alloc`

This CL trivially moves the declaration of the build arg
`use_partition_alloc` from `//base/allocator/allocator.gni` to PA's
`partition_alloc.gni`. Some other `BUILD.gn`s are changed to import
`partition_alloc.gni`, in order to continue using `use_partition_alloc`.

This CL introduces no behavioral change, but is intended to allow PA to
express the `use_partition_alloc` concept in embedder GN recipes. Much
as Chromium already does, the idea is to allow embedders to gate
inclusion of the PA sources on `use_partition_alloc`.
