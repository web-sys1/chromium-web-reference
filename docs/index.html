[iOS] Handle clear data operation for supervised account FRE sign-in.

There is an empty handler when clearing data in FRE, which causes the app to stop responding for supervised sign-in. This patch provides the missing command handler.

This will unblock supervised user FRE sign-in when the feature is
enabled by default in sign-in configurations.
