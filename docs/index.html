[iOS] Add swipe-to-delete to omnibox prototype + preview suggestions

Basic support for swipe-to-delete is added to popup matches, with
preview matches. A separate PopupMatchRowView struct is created, with
two main advantages:
- `.deleteDisabled` can be now set explicitly on the row view rather
  than the `HStack` it currently uses to layout its content.
- The list view code is cleaner, making room for sections and nested
  `ForEach` constructs.
Also, the trailing button callback code was modified so we don't need to
forward the popup model as a whole to the row view.
