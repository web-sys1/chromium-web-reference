Refactor ProcessMetricsDecorator before adding CPU measurements

* Moves common code that splits memory measurements between frames and
  workers to a helper function that can be reused for different types
  of memory, and will be used for CPU measurements.
* Adds NodeDataDescribers to display the current measurement state on
  the system node, and metrics estimates on frame and worker nodes.
* Move the unit test to the same directory as the implementation.
* Removes PageNodeImpl::usage_estimate_time() and
  PageNodeImpl::private_footprint_kb_estimate() which seem related to
  ProcessMetricsDecorator but are actually unused.
* Adds a non-renderer process to the unit test.
