[PolicyLogs] Policy Logs Page Added with chrome.send()

This CL adds the policy logs page for chrome://policy/logs on Android.
The page has a simple web UI to display the version information
and the policy event logs from PolicyLogger.

The CL contains the basic skeleton for the WebUI page which displays
the logs and version information as well as the communication
between the browser and the renderer.

PolicyUI will initialize the page's html source for the
chrome://policy/logs path, and its script (policy_logs.js)
will make the requests to the browser when the page is first created and
when the Refresh Logs button is clicked.

A button is also added to download the page contents as a JSON file.
