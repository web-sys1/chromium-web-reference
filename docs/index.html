Implement navigation for the visual guided default browser setter

Introduce the ability for the visual guided setter to navigate to a
specified URL, facilitating a guided flow for users to set Chrome as
their default browser.

This change includes:
* Adding a feature parameter to define the visual guide URL, defaulting
  to "chrome://default-browser/".
* Updating the DefaultBrowserVisualGuidedSetter class to require a
  Profile instance, allowing it to perform browser navigation.
* Implementing the Execute method to open the guide URL in a new
  foreground tab.

See: go/windows-default-browser-eng-doc
