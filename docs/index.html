Add perf test for dir=auto when text node is changed

This is a follow-up CL of CL 2683596 that improved the performance by
avoiding a tree walk for resolving directionality, is a test that
changes the text content of 10,000 children to the element having
dir=auto attribute and measures the performance by changing a text
node to the last child so that we can see the exact figures of the
performance improvement from the CL and can track changes in the
performance in the future.
