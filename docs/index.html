Convert RemoteFileSyncService.h to OnceCallback

This CL converts OriginStatusMap, StatusMapCallback and ListCallback to
OnceCallbacks. It also deletes the unused RemoteVersionsCallback and
DownloadVersionCallback.

The CL also changes CallbackTracker to support OnceCallbacks only,
which is in line with the guarantees offered by CallbackTracker.

There are also a few opportunistic clean ups to use std::make_unique,
instead of calling "new".
