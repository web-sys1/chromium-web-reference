Reland "DEPS: update ninja to 1.12.1"

This reverts commit 10df808911580e41aefff1f53334bf1384aba77e.

Reason for revert:
known missing dependencies were fixed in https://crbug.com/40253918

1.12.1 is mostly faster than 1.11.1 on chromium builds.

# noop build
* 4% faster on Linux
* 3% slower on Windows
* 49% faster on macOS

# clean build
* 2% faster on Linux
* 9% faster on Windows
* 9% faster on macOS


Confirming build health in build perf builders
* https://ci.chromium.org/ui/p/chromium/builders/build.shadow/linux-build-perf-developer/5/overview
* https://ci.chromium.org/ui/p/chromium/builders/build.shadow/win-build-perf-developer/2/overview
* https://ci.chromium.org/ui/p/chromium/builders/build.shadow/mac-build-perf-developer/5/overview
