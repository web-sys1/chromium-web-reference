[PwdMessageDialog] Chevron in edit password dialog should only appear
when a change has been made to the username

When there is only one username in the dropdown's list, the purpose of
the dropdown is to help users to get back to their previously saved
username when they make changes. The chevron & dropdown should only
appear once the user makes a change to their username. The dropdown and
the chevron are hidden again if user resets to the initial username.
