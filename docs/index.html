[iOS] Add "Close Other Tabs" action to Tab Grid context menu

This CL adds a "Close Other Tabs" action. This action allows users to
close all other regular tabs except the selected one.

Key features:
- Available for both regular and pinned tabs.
- When triggered from a pinned tab, all regular tabs are closed, and the
  pinned tab remains open.
- When triggered from a regular tab, all other regular tabs are closed.
- Hidden if there are no other regular tabs to close.
