Make destructor of `WTF::SharedBuffer` private (friend of `RefCounted`)

This CL helps to ensure that `WTF::SharedBuffer` can only be destructed
via `base::RefCounted` (e.g. that `SharedBuffer` cannot be
stack-allocated).

One motivation for this CL is a future, hypothetical Rust interop, which
may need to provide an explicit comment with a justification of why an
`unsafe impl RefCounted for wtf::SharedBuffer` won't undermine Rust
memory safety.
