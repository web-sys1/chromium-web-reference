[DevTools] Report beacons as resource type ping to DevTools

This will allow the devtools to filter on beacons in the network panel.

We infer the beacon type based on initiator_info.name, which is set to
fetch_initiator_type_names::kBeacon by PingLoader::SendBeacon. The
DevTools front-end can then be filtered via the resource type Ping
added in https://crrev.com/c/2372387.
