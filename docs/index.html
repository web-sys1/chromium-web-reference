[reland] Pass GLDisplay pointer between gl_display init and gpu_init.

This is a reland plus a fix for the component build on all platforms
(discovered on Mac Debug):
https://chromium-review.googlesource.com/c/chromium/src/+/3621555

When a GLDisplay is initialized, the pointer needs to be passed up to
gpu_init or other code that calls InitializeGLOneOff. This pointer can
be used to initialize extension settings, and shutdown GL.
