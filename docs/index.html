Android: Add script to be used in autorolling lint

Script used in: http://review.skia.org/558306

Allow lint to continue successfully when new baseline files are created
since that is the primary way to update baseline files.

Also build chromecast and cronet so that all baseline files can be
updated at the same time.

Example (Updating Chromium):
$ third_party/android_build_tools/lint/rebuild_baselines.py

Example (Updating Clank):
$ third_party/android_build_tools/lint/rebuild_baselines.py --prefix clank --git-root clank
