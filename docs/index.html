[iOS] Add snapshot spinner when the tab is loading

This CL adds a spinner to the tab grid cells to indicate that a tab is
loading. The spinner is shown when the tab's web content is being
realized, providing immediate feedback to the user and making the UI
feel more responsive.

This change includes:
- Adding `{show,hide}{Favicon,Snapshot}ActivityIndicator` properties to `GridCell`.
- Updating `BaseGridViewController` to show/hide the activity indicator based on the tab's loading state.
