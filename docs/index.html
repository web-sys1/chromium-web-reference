[CrOS Cellular] Add ONC toplevel validation for unique admin APN Ids

Checks that the IDs of APNs provided in the toplevel kAdminAPNList
field are unique.

Note that kAdminAssignedAPNIds for managed eSIMs and
kPSIMAdminAssignedAPNIds for pSIMs map to APNs in the kAdminAPNList,
so the IDs must be unique.
