Reports aqs= and gs_lcrp= using {google:assistedQueryStats} replacement

Agassi and Enterprise policies use out-of-date DSE templates that do
not feature {google:searchboxStats}. Due to the nature of these features
it is not currently possible to guarantee they stay up-to-date with the
//components/search_engines/prepopulated_engines.json in the Chrome
binary. This change allows reporting both aqs= and gs_lcrp= params using
{google:assistedQueryStats} which is an old replacement and likely
present in virtually all DSE templates out there.
