[AutofillDataModel] Set up heuristic predictions overrides for new types

For recently introduced types, sometimes we want to prioritize local
heuristics over the autocomplete attribute or predictions returned by
the server. It is possible that the newly introduced types do not do not
have a clear corresponding HTML type or that the server might have
learnt incorrect predictions for those fields.

This CL introduces a fixed set of local heuristics overrides to do
just that.
