[side panel] Move tooltip updates to the SidePanelButtonHighlighter

Currently the google lens and reading list side panels attempt to
directly set the tooltip text on the side panel toolbar button.

The current code is hard to follow and would require another
set of calls to SetTooltipText() in side search code to support
the side search panel.

This CL removes the bespoke SetTooltipText() calls from the
side panel specific callsites and moves the logic that
controls this text into the SidePanelButtonHighlighter which
already tracks the open state of the lens, reading list and
side search panels.
