Apply ConvertingAudioFifo to AAC encoder

This CL stores data incoming into the AudioToolboxEncoder into a
ConvertingAudioFifo. This FIFO will rebuffer frames to match the
expected AAC frame size, and remix channels, if ever the input had a
varying number of channels.
