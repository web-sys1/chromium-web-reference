Settings: Remove dependency from a browser proxy to a UI element.

Browser proxies should not depend on UI elements, only vice-versa.
Fixing this reverse dependency makes it easier to experiment with the
"limited sharing" mechanism between CrOS and Browser Settings, by
allowing deploying it to non-UI files first which are simpler cases to
decouple (no dependencies to shared CSS styles).
