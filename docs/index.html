[CrOS Bluetooth] Add FetchDevice() method to DevicePairingHandler.

Update DevicePairingHandler mojom definition and implementations with
a FetchDevice() method that queries the Bluetooth adapter for a device
based on its address. This will be used by clients that only have the
address of a device that need the device's ID in order to initiate
pairing with it.
