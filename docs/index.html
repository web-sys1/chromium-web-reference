Fix painting background of block-in-inline

Before block-in-inline, |PaintLineBoxChildren| could early
return for |kBlockBackground| and
|kDescendantBlockBackgroundsOnly| phases.

This patch stops the early return so that these paint phases
go into the block-in-inline children.

and ~60 more.
