iwa: Update ContentSettingBubble to use new API & Display IWA App Name

UrlIdentity creates a user-identifiable string for a URL.

We update ContentSettingBubbleModel to use UrlIdentity to format display
name. After this change, normal site and file URLs will keep their
current formatting logic, while Isolated Web App URLs will use app name.

In this CL, we also add FileFormatOptions for UrlIdentity. This new enum
is used to help preserve the current File URL display logic.
