Add a device policy to control the availability of Hindi Inscript Kbd.

Adds a new device policy, DeviceHindiInscriptLayoutEnabled, to control
the availability of the HindiInscript layout.

In the following 2 CLs (crrev.com/c/3998142 provides
an easier-to-understand combined view of the 2 CLs),
HindiInscriptLayoutEnabled was added as a user policy.

-  crrev.com/c/3935574
-  crrev.com/c/3989499

However, the way the kbd availability is checked (at the Chrome start-up
before the user login) makes the user policy not working for the
purpose. So, the new device policy above is added, instead.
