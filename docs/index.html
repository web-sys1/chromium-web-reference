Update the internals page for the productivity launcher.

The launcher internals page previously reflected the old launcher's UI,
and results were split by zero state and search based on the content of
the previous query.

Launcher infrastructure has changed now and there are three separate UI
surfaces: recent files (also known as "continue"), recent apps, and
search. This CL updates the launcher internals page to split results
into these three categories based on their display type. The display
types are hardcoded but stable strings.

Also update the results table to use trusted types if available. See
https://source.chromium.org/chromium/chromium/src/+/main:docs/trusted_types_on_webui.md
for more details.
