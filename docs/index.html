PSM libs: p5. Removed unused functions

shell-encryption and private_membership leaked some configs
which disable some default warnings.

In order to re-enable all these warnings for chrome/browser/*
in this CL crrev.com/c/3303510, all warnings need to be fixed.

This CL is fixing the unused function warnings by removing
these functions to re-enable the warning.

These are all the remaining listed unused functions.

Some functions should not be removed, but should be guarded by
the correct build flag to exist or not. These can be noticed
when tracing out the callers which are only going to call
these functions if a specific build flag is set (i.e. specific
OS). These functions are:
1) ShouldShowProfilePickerAtProcessLaunch
2) GetProfilePickerStartupProfileInfo
3) SharingHubDisabledByPolicy
