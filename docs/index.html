fusebox: factor out fusebox_server.cc

This splits the existing fusebox_service_provider.cc file into two: one
half focused on "business logic" and one half focused on "D-Bus protocol
logic". For example, this makes it easier to confirm that D-Bus requests
and responses have the same "shape" (the same number and type of
arguments) regardless of the code path taken to serve that request.
