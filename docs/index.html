[OutOfBlinkSandbox]: Stop pulling from blink.

The previous reland:
https://chromium-review.googlesource.com/c/chromium/src/+/2741587
made the browser process to push the sandbox flags to commit for
the new document toward the renderer.

This patch removed the plumbing that was used to pull the sandbox flags
from blink.

Now, the responsibilities in between the browser and the renderer have
been fully inverted:
- Compromised renderer can't fake sandbox_flags anymore.
- The DidCommitNavigation IPC is one step closer to be removed.
  See the [swap-rfh-at-commit] project.

[swap-rfh-at-commit]:
https://docs.google.com/document/d/1RXGDkrSWTfaeio76joXKWBLGDHAbiB48sHUdcTX6rdc
