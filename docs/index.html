[DBSC] Convert BoundSessionThrottlerParams to an array

Pass an array of BoundSessionThrottlerParams to GoogleUrlLoaderThrottle
to support multiple bound sessions running simultaneously.

It won't be possible to pack all running bound sessions into a single
BoundSessionThrottleParams in all cases because the sessions might have
non-overlaping domain/path locations. This is why an approach with an
array of params was chosen.

The total number of session is expected to be less than 5 at all times.
