Moving app installation events from WebAppRegistrar to WebAppInstallManager.

This refactoring change moves the installed/updated/uninstalled
from the AppRegistrarObserver to a new WebAppInstallManager class.
This way the OnWebAppInstalled event can be triggered from the
WebAppInstallManager after the install task is completed.
This change corresponds to phase 1 of the linked design doc.
All new WebAppInstallManager events are triggered at the same time
as their corresponding WebAppRegistrar events, except for the
WebAppInstalled event which is now triggered after the install task
is finished.
