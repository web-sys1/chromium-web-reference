Add simplified multi-instance check for Incognito windows

We cannot use the existing MultiWindowUtils method directly because
checking the manifest `launchMode` requires access to
ChromeTabbedActivity, which is not visible to this module due to
dependency restrictions.

Instead, this method relies solely on the SDK version check
(Android 12L+) as a proxy for determining if the multi-instance feature
is supported.
