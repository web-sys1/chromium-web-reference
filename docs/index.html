[blink scheduler] Clean up kThrottleTimedOutIdleTasks feature

This CL removes the kThrottleTimedOutIdleTasks kill switch, which was
added in M134. It also removes PrioritisationType::kJavaScriptTimer,
which is no longer used (it used to affect queue policy, but that was
removed in crrev.com/c/3791718). I'm including this in this patch to
simplify setting up the throttleable types.
