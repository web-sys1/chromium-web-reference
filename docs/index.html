system-extensions: Implement CrosWindowManagementContext

CrosWindowManagementContext is a class that manages CrosWindowManagement
instances. It implements CrosWindowManagementFactory which the renderer
uses to bind to WindowManagementImpl, the implementation of
CrosWindowManagement.

CrosWindowManagementContext holds a set of receivers for
CrosWindowManagementFactory and a unique associated receiver set
for WindowManagementImpl and CrosWindowManagement. The lifetime of
WindowManagementImpls is tied to that of the
CrosWindowManagementFactory receiver which itself is deleted when the
pipe is closed, which should happen when the service worker is
destroyed on the renderer side.

This is slightly problematic since the pipe could be closed after the
process and profile are deleted, but WindowManagementImpl checks that
both the process and the profile at the start of its functions. A
followup CL could ensure that WindowManagementImpl is destroyed
when the process gets destroyed to make this more safe.
