[PA] Remove use of TimeTicks/TimeDelta at public API boundary

PA library is soon going to use
partition_alloc::internal::base::Time{Ticks,Delta}, which are
different types from base::Time{Ticks,Delta}.  Thus, this patch
stops use of these classes at API boundary.

Disclaimer: This patch uses deprecated APIs of
TimeTicks::{From,To}InternalValue due to its very unique
situation.
