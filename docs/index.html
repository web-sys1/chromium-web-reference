dlp: Extend DlpClient::Observer interface with OnFilesAddedToDlpDaemon

Currently only the who adds files may be notified of a successful file
addition. This CL enables the possibility to notify a generic observer.
As an example use case, we might want to notify an open Files app
instance that files have been added to the daemon and thus they should
now show the managed icon.
