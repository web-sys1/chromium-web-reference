[ZPS] Extends suggestion groups to all providers

Refactors the zero-suggest header info (i.e., header text map and the
default visibility state) into a single struct representing a basic
suggestion group info.

Also generalizes the logic so that every provider can now provide a map
of suggestion group IDs to group info; which will be accumulated in the
AutocompleteResult and used to group and demote matches with group IDs.

Also allows suggestion groups without header texts to exist.

This CL should not impact the zero-prefix suggestions which are the
only suggestion types with groups.
