[AW] Add a NetworkFetcher, NetworkFetcherFactory.

Component updater code expects a NetworkFetcherFactory, which can
create a NetworkFetcher for downloading dynamic config. The default
implementation relies on use of network service which can't be used by
the WebView non-embedded process. Create a Webview-specific one.

A test has been added to verify the logic. To be able to mock out calls
to native, the test has to be a junit test.
