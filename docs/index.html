[ios][PhishGuard] Add PasswordProtection calls to PasswordManagerClient

This CL continues to port over PhishGuard functionality to iOS.

Code in this CL can be compared to the /content counterpart
(from which it was ported) in //chrome/browser/password_manager/
chrome_password_manager_client.h/cc.

* Calls PasswordProtectionService methods from
IOSChromePasswordManagerClient.
* IOSChromePasswordManagerClient owns PasswordReuseDetectionManager.
* IOSChromePasswordManagerClient observes WebState so it can pass
on navigation events to the detection manager.
* IOSChromePasswordManagerClient observes input events from
PasswordProtectionJavaScriptFeature so it can pass key presses and
pastes to the detection manager.
* Sends JS input events to InputEventObserver callbacks.

* Changes to /autofill, /sync, and /ui/settings are due to resolving
circular dependencies.
