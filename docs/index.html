[composebox] Delete file in query controller cache

* WebUI sends token associated with a file to the handler.
* The handler passes this deletion message on to the
  |ComposeboxQueryController|.
* Query controller deletes file from cache if it exists, returning a
  bool of whether it was able to delete the file or not.
* If the controller failed, the handler throws a bad message, since
  the WebUI sent an invalid token.
