Replace HandleIntent with the AppService LaunchAppWithIntent interface.

This CL extends the intent, and replaces the mojom HandleIntent in
StartSmartSelectionActionMenu with the AppService LaunchAppWithIntent
interface to:
1. Have a central place for UMA to record app launching.
2. Save the app launch info for the full restore project.

Fix the display issue for ARC app launch with smart selection menu. If
the text from Chrome is shown from the external monitor, the ARC app is
still launch from the labtop internal screen. This CL adds the cursor
point to launch the ARC app from the same screen as the text selected.
For example, if the text from Chrome is shown on the external monitor,
launch ARC app on the external monitor as well.

The mojom HandleIntent interface is modified to add the display id and
the window info in CL:2739291.

Add unit tests to verify the intent functions.
