Allow conversion from base::span<T, N> to std::span<T, N> for const N

std::span(Range) constructor does not deduce N in the type system from
base::span<T, N> so it constructs a std::span<T, dynamic_extent>. We
want to ensure that any code that accepts std::span<T, N> can be
passed a compatible base::span<T, N> and it will implicitly convert.
