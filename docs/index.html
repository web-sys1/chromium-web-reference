[blinkpy] Fix pylint using presubmit canned check

This change replaces check_blink_style.py's broken pylint check with a
presubmit version, which can check files in parallel by using pylint's
`--jobs` parameter and is more consistent with other parts of Chromium.

For the presubmit to run cleanly, this change also fixes or disables
issues revealed by reenabling pylint. Many were left over from the
Python 2-to-3 migration. Others will be dealt with in follow-up
changes.
