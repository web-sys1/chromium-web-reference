Add toggleable line wrapping to view-source document

Enabling line-wrapping on the view-source page causes a performance
problem in layout for very large documents. However, removing line
wrapping causes developer pain, especially for pages with minified
code or very long lines.

This CL adds a "Line wrap" checkbox to the top of the view source
page, which defaults to unchecked (for speed). When checked, it
adds line wrapping to the source table.
