Add an integration test framework, and an initial set of tests.

integration_test.py can be invoked without arguments to run tests, and
with the subcommand 'record' to record a new testcase. Testcases are run
against the test disabling tool, with the results recorded. Then when
running tests we set up a hermetic environment to ensure
reproducibility, and assert that the data written by the tool is the
same.
