[Autofill] Consistent icons for popup settings

In the Autofill popup, currently there is only a icon shown in the
settings footer for managing passwords. For a more consistent experience
there should be a consistent icon.

This CL does two things:

a) The 'options' suggestions in the popup footer is equipped with a
consistent 'settings' icon for all autofill types included credit cards,
addresses and passwords. This behavior is feature-flagged.

b) If all credit cards are server cards, currently the suggestions are
showing a trailing GPay icon in the settings footer.
This icon is moved to the store indicator icon which was added to
the functionality of the footer.
