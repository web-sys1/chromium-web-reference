FakeAudioInputStream: Swap g_fake_input_streams_are_muted to std::atomic

g_fake_input_streams_are_muted is a boolean that tracks whether the
FakeAudioInputStreams are all muted or not. Convert from Atomic32 to
std::atomic<bool> - they are broadly supported, do not implicitly
convert to/from integers, and we are slowly converting code to use
them.

No behavior change expected.
