[Extensions][Al] Show signout dialog if account extensions are installed

By account extensions we mean extensions the user installed while signed
in. If the user installed extensions before signing in we do not need to
show the dialog as they are saved onto the device.

We should always show a sign-out confirmation dialog box with the
'uninstall extensions' checkbox when account extensions have been added,
including in cases where:
- No confirmation dialog would show up otherwise
- The 'unsaved data' confirmation dialog shows up

Example screenshot: https://crbug.com/502977184#comment18
