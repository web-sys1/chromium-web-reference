Use SetPreinstallUrlAllowListForTesting() instead of removing kDisableDefaultApps in tests

This CL adds an allow list for preinstalled web apps that need to be
covered by tests. Instead of enabling all preinstalled web apps this
allows tests to be specific about which.

When all preinstalled web apps are enabled in a test some will require
internet access to install and wait 30s, with multiple of these queued
up in a global install queue this can cause the tests to timeout.

The addition of NotebookLM pushed these tests over the line such that
they started breaching the test timeout threshold.
https://chromium-review.googlesource.com/c/chromium/src/+/6389898

This CL updates the tests allow list the apps they actually care about
and avoid being affected by unrelated preinstalls.

On my machine this CL takes
All/GeminiAppInteractiveUiIneligibilityTest.AbsentFromAppList/kFeatureFlagDisabled
down from 26 seconds to 4 seconds.
