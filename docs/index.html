Fix some drag drop issues in web ui tab strip

This fixes the screen coordinates uses for synthesizing mouse events
to start and end touch drag drop, and when drag drop is finished,
moves the cursor to where the drag drop started. Windows doesn't like
doing touch drag drop if the mouse cursor isn't in the window starting
the drag drop.
