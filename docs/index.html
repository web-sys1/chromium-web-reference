media/gpu/vaapiWrapper: Fill |data| in vaCreateBuffer()

VaapiWrapper::SubmitBuffer_Locked() executes vaCreateBuffer() with
null |data| and then VaapiWrapper::MapAndCopy_Locked().
vaCreateBuffer() fills |data| into a created buffer if |data| is not
nullptr. This CL removes MapAndCopy_Locked() in SubmitBuffer_Locked().

Test: video_encode_accelerator_tests
