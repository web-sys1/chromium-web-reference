personalization: bugfixes for full screen

Better way of exiting full screen preview when anything closes or
minimizes the window. This fixes alt+tab, ctrl+w, and any other way
that the user can get out of full screen. Forces preview wallpaper to
end and exits the full screen view in the wallpaper SWA.

Also small fix for setting the correct initial layout option when
user selects and previews a custom image. WallpaperController expects
a newly set custom image to always be kCenterCrop.
