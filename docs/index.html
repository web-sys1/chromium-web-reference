[Android Build] Use sys.executable in lookup_dep.py subprocess call

lookup_dep.py is used in the build when compiling Java, and is run with
the user-preferred Python interpreter set in GN by --script-executable.
It should propagate this setting when calling another Python script via
subprocess, to avoid breakage when the system python binary picked by
the shebang in list_java_targets.py is not appropriate. Use
sys.executable for this.
