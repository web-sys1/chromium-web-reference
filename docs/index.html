[blink_apis] Sort `depends_on` list.

`blink_api_proto.py` serializes Blink API metadata using as a textproto.
Some repeated fields may wind up with non-deterministic ordering thereby
causing the resulting API list of to be unstable.

To address this, ideally we should enforce a deterministic ordering on all
repeated fields of the API metadata message. However in practice the
ordering is enforced implicitly for most of the fields other than
`IDLType.depends_on`.
