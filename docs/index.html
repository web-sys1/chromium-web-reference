[infra] Update mb try to correctly specify subprocess input for python3.

The process-handling code in mb.py was not correctly updated to handle
specifying stdin for the created subprocess. That particular detail is
only used for the try subcommand. This changes the process creation code
to use subprocess.run, which provides the necessary functionality for
specifying input data, capturing output and handling text conversion for
input and output. The buffer_output and stdin arguments were renamed to
capture_output and input to match the subprocess argument names/.
