Privacy Sandbox: Enable all V4 prefs for testing

This CL creates a method that enables all privacy sandbox APIs prefs for
testing for V4 and updates all tests which enable privacy sandbox by
`SetPrivacySandboxEnabled` to use
`SetAllPrivacySandboxAllowedForTesting`.
This change only covers tests which don't care about granular access
control, other tests that call `SetPrivacySandboxEnabled` to disallow
Privacy Sandbox will be updated to allow/disallow the specific APIs in
the following CL.
