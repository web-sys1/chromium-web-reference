[Print] Align PDF default print scale to HTML print

The default PDF print scale was CUSTOMISED 100, which does not scale or
centre the content. This could cause content to be cropped in the print
preview if it extended beyond the paper size. To address this, this CL
aligns the default scale of PDF print to HTML print, which is
`kCenterShrinkToFitPaper`, which shrinks the overflow content and centre
the smaller content.

This change is controlled by flag kAlignPdfDefaultPrintSettingsWithHTML.
