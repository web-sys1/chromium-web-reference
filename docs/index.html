[media] Remove SharedImageFormatType::kLegacy checks

All usages of SharedImageFormatType::kLegacy also check for
SharedImageFormatType::kSharedImageFormatExternalSampler and treat them
similarly. Thus it is now okay to use kSharedImageFormatExternalSampler
as the default value within VideoFrame and remove
SharedImageFormatType::kLegacy. This removes the last few usages of
SharedImageFormatType::kLegacy before removing the enum value in a
follow-up.
