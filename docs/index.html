[Files app] Replace window.open with private API for lacros compatibility

window.open (used by Files app to navigate to URLs) is run by Files app
in ash-chrome, and therefore opens URLs in ash browsers.

Change all calls to window.open to a private API function that routes
the "open url" request to the primary browser, either ash or lacros.

the gear menu opens in lacros if lacros is primary, or ash otherwise.
