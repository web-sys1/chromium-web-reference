File System: Fix race condition in browser test.

Before this CL, FileSystemBrowserTestWithLowQuota::SetLowQuota() posts a
task on the IO thread to configure QuotaManager, but returns before the
task completes. This means that the task is racing with other test
steps. This adds unnecessary complexity to the test's timing.
