Introduce WebappUninstallSource for web app uninstallation log

Web app has a WebappInstallSource that is used for code flow as well
as logging. Uninstallation, however, doesn't have its counterpart
values, so the change introduce WebappUninstallSource that logs its uninstallation
source, but doesn't affect code flow (yet);

It is reported that respective platform or browsers has its own unique
uninstallation source and they want to track the its uninstallation source like
its installation source.

The change doesn't add UKM yet that is used by Android and it will be added in next iteration.
