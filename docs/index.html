[GlicMI] Always bind pinned tabs in GlicInstanceImpl

Add kGlicAlwaysBindOnPin feature flag (defaulted to true) to bypass
the check that prevents binding/showing a tab if it is already bound
to a Glic instance. This ensures that pinning a tab will always
attempt to bind it to the current instance or show the inactive side
panel UI, regardless of its previous state.
