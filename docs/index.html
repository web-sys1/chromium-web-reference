Move ClientDestroyedWidget to DesktopWindowTreeHostWin

This CL moves the ClientDestroyedWidget() interface from
BrowserDesktopWindowTreeHost to DesktopWindowTreeHost. This
allows invalidation of Widget-specific state on all window tree
host subclasses, not just the Browser specific subclasses.

This is purely a refactor with no intended behavioral changes.
