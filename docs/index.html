Remove menulist special casing from RenderAccessibilityImpl

The original code added in CL:769674 ensured that kHidden states were
properly updated in <select size=1> when the popup was toggled or the
active descendant changed. This CL moves toward the goal of
having RenderAccessibilityImpl be a generalized layer that doesn't
need to know about specific roles.

This CL adds tests and moves the code into AXMenuListPopup.
RenderAccessibilityImpl shouldn't need to know about menu lists.

TODO in a follow-up: remove ax::mojom::Event::kChildrenChanged and
just mark objects dirty. Tests that wait for kChildrenChanged can
wait for the generated CHILDREN_CHANGED event instead.
