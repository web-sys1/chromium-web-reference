Remove IAMF Descriptor OBU prepend

Now that the descriptor OBUs are stored in the `extra_data` field of the
AudioDecoderConfig, they no longer need to be prepended to every IAMF
sample. This will allow decoders to read the sample directly, without
calculating and skipping the descriptor bytes when they're not needed.
