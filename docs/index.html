Move FormatMacAddress to CloudPolicyClient.

CloudPolicyClient is the class that knows which format of MAC address it
needs for a request while factory method was taking care of formatting.

Now CloudPolicyClient accepts an array with MAC address bits and formats
it internally. The factory now parses MAC address string obtained with
StatisticsProvider into bits.

Additionally, empty or ill-formatted MAC address is visible in code now
with using optional.
