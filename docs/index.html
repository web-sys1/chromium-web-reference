[Jellybean] Allow for json5 color files to force preblending.

https://chromium-review.googlesource.com/c/chromium/src/+/4349545 made
the CSSGenerator output blended colors as color-mix calls. However this
assumes that the target browser supports color-mix which as of writing
is untrue for non chrome browsers and chrome pre M111.

This CL adds the option for a json5 file to force preblending for all
of it's colors, allowing our legacy colors to remain preblended and
usable on older and none chrome browsers. Note that newer cros tokens
do not specify preblending as a dynamic palette has a hard requirement
on color-mix calls.
