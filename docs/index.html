[omnibox][search_engines] Disable make default for starter pack tab search.

The starter pack @tabs search mode is not a "real" search engine in that
there's no landing page for search results.  It therefore does not make
sense to allow tab search to be a default search engine for the omnibox.
This CL disables the ability to make the @tabs search engine default.
