Add retry logic for creating Android and ChromeOS driver.

In http://crrev.com/c/7261982 we added retry logic for Desktop.
We observe similar issues on Android Dev and ChromeOS Beta,
therefore we add similar mechanism for those platforms.
Please notice that we do not reuse the timeout logic, since
those platforms have longer start up time due to being run
in emulator or vm.
