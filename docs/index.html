Lacros: Pass flag to disable gpu in Lacros in testing.

Since hardware accelerated rendering for Lacros is not possible
yet as Ash on X11 backend can't import dmabufs, so passing
additional flag to disable gpu in Lacros running with ash
browser tests. Without disabling gpu, gpu process crashes
repeatedly before switching to software mode, and this
extra delay can cause tests to timeout.
