[iOS][Thumb Strip] Dismiss keyboard when the thumb strip is Revealed

When the thumb strip transitions to Revealed state, the keyboard
should be closed if it is open onscreen. To facilitate this, the
-willAnimateViewReveal method now has both the current and the next
state instead of just the current state. This is necessary because when
the current state is Peeked, the transition could proceed in either
direction.
