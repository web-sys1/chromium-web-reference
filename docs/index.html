[iOS][Omnibox] Add Omnibox Carousel

Omnibox Carousel for most visited tiles.
Base Architecture:
- CarouselItem represent a tile in the UI.
- OmniboxPopupCarouselCell represent a cell in the popup tableview, it
  contains the scrollview and stack view to display the tiles.
- OmniboxPopupCarouselControl is a UIControl displaying a tile.
