[Autofill][Android] Improve save address flow closing animation

Prior to this change, done callback in EditorDialog was triggered only
after the dialog was dismissed with animation. This caused the
sequential closing of the edit dialog and an address modal dialog.

Now with instant done callback enabled, the done callback is triggered
right away, which allows closing both dialogs simultaneously.
