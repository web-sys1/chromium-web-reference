Fix viewport height for interstitial screen

This change adds a fix to deal with the wrong viewport height
that is calculated on Android.
This is a work-around for the problem described in crbug.com/1400469.

The change restores the default Chrome pre-108 resizing behaviour (https://developer.chrome.com/blog/viewport-resize-behavior/#opting-in-to-a-different-behavior).
