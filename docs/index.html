[webauthn] Use dwTransports from attestation for supported transports

WEBAUTHN_CREDENTIAL_ATTESTATION v8 adds a dwTransports field that
contains a bitmask of all transports supported by the credential.
Use it to populate the response's transports set.

For older API versions, fall back to the previous behavior of only
setting transports to {kInternal} when the used transport was internal.
