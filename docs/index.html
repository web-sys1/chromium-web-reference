arcvm: Implement WorkingSetTrimmerPolicyChromeOS::ArcVmDelegate

The new class, WorkingSetTrimmerPolicyArcVm, tells the Chrome OS
policy class when it is safe to reclaim memory from ARCVM. More
specifically, its IsEligibleForReclaim() function returns true
when all the following conditions are met:

- ARCVM has fully booted.
- ARCVM app window is not focused.
- |arcvm_inactivity_time| has passed since the last user interaction
  with ARCVM. "user interaction" is recorded on all of the following
  events:
    - One of Arc.UserInteraction UMA metrics is recorded.
    - ARCVM has fully booted.
    - ARCVM app window has lost focus.
