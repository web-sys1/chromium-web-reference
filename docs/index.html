[identity] Remove the old consent flow (issueAdvice)

This CL removes all the code handling the old consent flow for minting
access tokens for Chrome extensions.

According to the Signin.OAuth2MintToken.ApiCallResult histogram, the
server no longer returns issue advice response in the OAuth mint token
flow for new Chrome versions (>= M87).

Most of the tests covering the old consent flow were modified to
simulate the new consent flow instead.
