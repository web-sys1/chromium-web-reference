media/mojo/MojoVEAService: Fix the encoding duration trace event

The trace event about encoding duration uses
TRACE_EVENT_NESTABLE_ASYNC_END_WITH_TIMESTAMP with setting id to
TRACE_ID_LOCAL(this). If there are multiple ids, the tracing are
confused to match them. The id must be unique. This fixes it by
using timestamps for id.
