Extend base::Extend to support a generic range as input

This replaces the overload that takes a `base::span` and adds support
for projections, similar to `base::ToVector`. Note that this does not
update the overload that consumes a `std::vector<T>&&`, and that the
legacy `std::vector<T>&&` overload does not support projections. The
right way to implement this is to probably add an adapter to treat the
values in the range as rvalues.
