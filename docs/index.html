[Cast Receiver] Add Renderer Side of UrlRewrite Support

This CL makes the following changes:
- Adds support for UrlRewrite to the cast_receiver component
- Applies that and the remainder for the component's Renderer side code
  to the existing //chromecast/renderer code
- Splits the Renderer code in cast_receiver into a public interface and
  its implementation to limit the details component users need to be
  aware of.
