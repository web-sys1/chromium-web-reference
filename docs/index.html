[iOS] Create ChromeIdentityService::IsServiceSupported()

IsServiceSupported() should returns false for ChromeIdentityService and
true for all subclasses.
ChromeIdentityService is used only for Chromium which doesn't support
sign-in.
In this patch, ChromeIdentityService needs to return true until
ChromeIdentityServiceImpl::IsServiceSupported() is implemented.
