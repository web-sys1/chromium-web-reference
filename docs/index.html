Do not attempt to detach the same AXObject twice

When the parent of a text node calls ClearChildren(), it needs to
use get the AXObject*, if it exists, for each child DOM node.
This allows these AXObjects to be detached from their parents.
Doing so must not try to immediately Remove() irrelevant AXObjects
as this can lead to double-detaching the same object.

This CL defers removal by invalidating the object to be removed.
