Disable background video optimizations for captured video.

This adds a 5s window after a video frame has been captured which
prevents background video optimizations from being applied. The window
refreshes with every frame capture.

There's no mechanism to re-enable optimizations if capturing stops
while the page is in the background, so we'll miss some optimization
cases, but I expect it's an insignificant amount.
