Replace RunLoop::QuitWhenIdle* with RunLoop::Quit* in Kiosk tests

RunLoop::QuitWhenIdle* will break if the work it's waiting moves to
another process (e.g. Lacros). We want to avoid using them generally.
There are 2 places in Kiosk tests that break on RunLoop:Quit*. We need
to do more investigation later.
