[Autofill] Detect GIFT_CARD on all CREDIT_CARD_NUMBER fields.

This CL enables the GIFT_CARD pattern on all field types that could
match a CREDIT_CARD_NUMBER. Otherwise, a field like
  Gift card number PIN: <input type="password">
does not match the GIFT_CARD pattern (because of its type) but does
match the CREDIT_CARD_NUMBER pattern (because of "card number").
