[Region Capture] fix capture hangs

This patch fixes a complete capture hang that occurs sometimes when
region capture. The root cause turns out to be that we sometimes create
a frame identifier but then decide to not output the frame, causing the
delivery queue to get stuck and no longer successfully emit frames.
