Update touch target size for new tab and model selector buttons

This CL updates the new tab button and model selector button to be consistent sizes with a 48dp touch target. Replaced the new tab button drawables with a 24x24 svg drawable. The previous resource did not have a consistent height and width so was difficult to get a consistent touch target.
