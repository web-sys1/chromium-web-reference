Measure potentially dangling <iframe name> values.

This patch adds two use counter values to help us understand the
content of <iframe name> attribute values. In particular, we're
looking for `\n` and `<` characters that might represent unclosed
attributes which could be useful in attacks relying on dangling
markup. These characters match the heuristics we're using for
URL-based attributes like <img src>, and should help us make
decisions about mitigations we could put into place.
