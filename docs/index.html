[zlib_bench] Allow user to print results in CSV format

This adds a new option (i.e. --cvs) that will print the results
of a benchmark in a tab separated format.

Usage: ./zlib_bench gzip --cvs ~/corpus/* > zlib_perf.csv

Makes it easier to import the results of a bench run into Google
Sheets (no more typing).
