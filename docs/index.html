ash: Restore and fix orphaned app_service_app_window_browsertest.cc

Restore the orphaned `app_service_app_window_browsertest.cc` to the
`browser_tests` build target in
`chrome/browser/ui/ash/shelf/app_service/BUILD.gn` along with its
required dependencies to pass `gn check`.

https://chromium-review.git.corp.google.com/c/chromium/src/+/5881739.)

test was failing because user-installed Chrome Apps (which includes the
legacy hosted app used in this test) are now blocked by default under
the Chrome App Deprecation mechanism introduced in 2025 (CL 6633387).
