Cleanup printing preferences files

Update prefs_util files so that they only contain functions that
directly read preferences.  Move wrapper functions which encapsulate
feature helpers for out-of-process printing for checking both feature
flags and policies into new oop_features files.

Expose the previously encapsulated CheckOopPolicy() function from
prefs_util.h.  Update its signature to return an optional, so that
callers can distinguish explicit preference values from no preference
specified.
