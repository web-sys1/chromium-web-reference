Isolate the Video LCP experiment from animated images.

The kAnimatedImagesReporting feature was previously used to enable
treating the first frames of both animated images as well as videos
as LCP candidates. That makes analysis of the effect of these changes
difficult, so this CL introduces a new flag specifically for video.

The runtime flag which controls whether these modified LCP candidates
are exposed to performance APIs still affects both.
