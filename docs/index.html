WebUI: Add ListPropertyUpdateMixin to replace ListPropertyUpdateBehavior

Polymer Behaviors are deprecated in favor of mixins. Moreover,
ListPropertyUpdateMixin is using TypeScript, so is more convenient for
WebUIs that are already migrated to TypeScript. A few such UIs
(print preview, bookmarks) are updated to the new mixin in this CL.
Also removing references to the legacy behavior in the read later UI,
which imported and inherited from the behavior but did not appear to
actually use it.
