Revert "[ios] Move the logic to select API to SnapshotGenerator"

This reverts commit a7a77903add76f057d79c002c663fd95b31a3c85.

Reason for revert: This caused the issue crbug.com/1514555 because
calling `[self updateSnapshotStorageWithImage:image];` in
`updateUIViewSnapshotWithCompletion:` is wrapped by a callback
and is deferred by posting a task. It delays to update the snapshot
storage. The old image is used when the storage is not updated yet.

Original change's description:
> [ios] Move the logic to select API to SnapshotGenerator
>
> This CL moves the logic to select which APIs should be used for
> taking snapshot images from
> SnapshotTabHelper::UpdateSnapshotWithCallback to SnapshotGenerator.
> The callers of SnapshotGenerator don't have to take care of which
> APIs are used.
>
