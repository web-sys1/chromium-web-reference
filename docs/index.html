Add new histogram Installer.PowerwashDays in histograms.xml

When the device has powerwash completed, it will save the powerwash time
into file. After the device restarted in next time, the upstart script
will read that powerwash time and calculate how many days before current
time and send the days to UMA. For majority cases, this value should be
1 (which means device restarted just after powerwashed immediately).
