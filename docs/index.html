Rename TrustToken to PrivateToken in blink

Rename TrustToken to PrivateToken in developer facing blink side.
New fetch API is as follows.

fetch('<issuer>/<issuance path>', {
  privateToken: {
    type: 'private-state-token',
    version: 1,
    operation: 'token-request',
    issuer: <issuer>
  }
})
