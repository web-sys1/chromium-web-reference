Exposes computed node attributes where available to all code under ui/accessibility

In https://crrev.com/c/2953640 computed node data was exposed to some parts of the codebase,
especially AXPlatformNodeBase. This patch exposes any such attributes to all classes in ui/accessibility.

This patch also exposes HasState and GetRole in AXNode, in preparation for making some states / roles
computed on the browser as well. We might want the ignored state to be computed,
for example.
