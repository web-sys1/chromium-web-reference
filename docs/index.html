FIELDSET: Don't reattach on descendant reattach

Since crrev.com/893709, a FIELDSET has reattached whenever any
descendant node is reattached.  It caused crbug.com/1240414 because
SELECT closes its popup on detach.

This CL makes the reattach condition stricter; Reattach a FIELDSET only
for reattach of a direct child LayoutObject, not a descendant node.

Bug: 1240414
