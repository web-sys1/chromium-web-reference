Create cc::SolidColorLayer for solid color PendingLayers

This is behind feature SolidColorLayers.

Compared to cc::PictureLayer with solid color drawing,
cc::SolidColorLayer is more lightweight in many ways. For example,
it doesn't need a cc::DisplayItemList, solid-color analysis or
blink-side raster invalidation.

This CL is a spinoff of the solid color background work for LCD-text.
It won't affect LCD-text.
