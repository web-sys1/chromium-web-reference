[Ash Pixel Diff] A python script to detect flakiness

This CL adds a Python script to detect flakiness in any Skia
Gold based pixel tests including ash_pixeltests.

In detail, this script runs the specified test executable, such as
ash_pixeltests, multiple times. Then this script compares the
screenshots generated by neighboring iterations through the file hash
code. If one screenshot's hash code changes, the flakiness is detected.

Because this script uses hash code for comparison, this CL can only be
used in the pixel tests that use precise match.
