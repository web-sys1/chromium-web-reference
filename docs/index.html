Add more histograms for dice migration dialog

This CL adds the following histograms:
1. Signin.DiceMigrationDialog.TimerStarted: Tracks whether the timer was started or not.
2. Signin.DiceMigrationDialog.PreviouslyShownCount: Records how many times the dialog has already been shown.
3. Signin.DiceMigrationDialog.DaysSinceLastShown: Records the duration from the last time the dialog was shown.
4. Signin.DiceMigrationDialog.Shown: Tracks whether or not the dialog was shown after the timer was fired.
