Fix ContextualSearch not triggering when CCT uses mayLaunchUrl()

When mayLaunchUrl() is used, CCT may create a tab very early in startup
before the contextual search manager has been created. This change
allows ContextualSearchTabHelper to wait for the manager and add the
hooks when it is ready.
