CodeHealth: Remove use of DictionaryValue::size()

Since DictionaryValue is going away and base::Value
exposes APIs for all types, existing usages should be
replaced with Value::DictSize().

This change removes the use of DictionaryValue::size()
and replaces with Value::DictSize(). It's the first step
before removing DictionaryValue::size().
