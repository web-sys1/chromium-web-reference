Simplify chrome/test/data/webui/settings/BUILD.gn setup.

Remove distinction between preprocessed and non-preprocessed files, as
it adds complexity without any significant benefit. Preprocessing all
files by default, makes it easier for WebUI authors to add <if expr>
statements in tests files since they don't have to modify any build
configuration.

This is also done as part of experimenting with possibly standardizing
WebUI tests build configuration with a build_webui_tests() rule, similar
to what build_webui() does for prod code.
