[Nearby][WLAN] Restrict TCP server socket port range

Add a class that restricts the eligible TCP server socket ports to
[49152,65535]. Restricting the port range was a request from security
review.

We also add an analogous mojo interface. This will be used in
follow-up CLs that add new mojo services for opening TCP sockets and
firewall holes.

This CL is part of the effort to add WLAN as a Nearby Connections
transfer medium on Chrome OS. See http://go/cros-nearby-wlan for more
details.
