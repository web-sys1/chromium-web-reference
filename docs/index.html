[Extensions] Implementation of chrome.action.set/getBadgeTextColor.

Attempts by callers to set transparent text colors will be considered an
API error.

Prior to this extensions could not set badge text color and it was
automatically chosen for them based on contrast. This lead to bugs per
the linked bug.
