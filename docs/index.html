[IOS] Create WebSelectionJavaScriptFeature

Create a WebSelectionJavaScriptFeature to retrieve the
page selection.
This will be used for the actions in the edit menu that
simply need to get the selected text.

This feature is heavily inspired by the
LinkToTextJavaScriptFeature, but LinkToTextJavaScriptFeature
has a lot of specificities (only acts on allowlisted pages,
on amp frames,...) that makes it hard to reuse it.
