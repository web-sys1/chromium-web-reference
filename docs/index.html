Change the public key sent in DBSC registration request to a JWK

Change the public key included in the payload of the JWT sent to the
server in DBSC's registration request from the SPKI format to the JWK
format.

A key conversion helper is added to translate public keys in SPKI format
from /components/unexportable_keys to the JWK format, which only handles
ES256 and RS256 algorithms for now. It can be expanded as more signature
algorithms are supported by DBSC.
