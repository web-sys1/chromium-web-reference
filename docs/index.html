media: Update expiration dates for EME histograms

Set the new expiration date to 2024-04-11 based on "If the histogram
is actively in use and seems useful for an indefinite time, pick 1
year."
https://chromium.googlesource.com/chromium/src/+/main/tools/metrics/histograms/README.md#how-to-choose-expiry-for-histograms

To avoid updating the expiration dates 2~3 times a year, the
expiration date for all EME histograms owned by Alcatraz team changed
from:
```
2023-05-07
2023-06-04
2023-07-09
2023-07-16
2023-07-30
2023-08-13
2023-09-10
2023-09-17
2023-10-01
2023-10-08
```
to:
```
2024-04-11
```
