Shared Storage: Add selectURL call limit

We limit the number of calls to `sharedStorage.selectURL()` to 3 per
origin per page load (Finch-configurable).

As some Fenced Frame web test files relied on exceeding this limit, we
also split the affected test files into smaller files that respect the
new limit.
