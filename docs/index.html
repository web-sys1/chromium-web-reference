ForeignObject NG: Port UpdateLogicalWidth() and ComputeLogicalHeight()
of LayoutSVGForeignObject

This CL corrects the size of generated physical fragments.

LayoutNGSVGForeignObject implements UpdateLogicalWidth() and
ComputeLogicalHeight() as SetOverrideLogicalWidth() and
SetOverrideLogicalHeight() in UpdateBlockLayout() because
UpdateLogicalWidth() and ComputeLogicalHeight() are legacy-only
functions.

This CL has no behavior changes because of a disabled runtime flag.
