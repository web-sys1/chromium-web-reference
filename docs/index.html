Error Page Refactor, fix offline page navigation.

This CL:

1. Plumbs loadSimulatedRequest down from web_state, exposing it to
OfflinePageTabHelper.

2. Fixes incorrect double backForwardList population by removing offline
chrome:// URL(s) across the entire app when
web::features::IsLoadSimulatedRequestAPIEnabled().

3. Fixes infinite app hang when pressing forward nav button from offline
page.

4. Properly reloads the "real" page when navigating to an offline page,
if proper conditions are met (i.e. user has good network connection).
