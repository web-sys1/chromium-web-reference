Add SWADelegate::IsUrlInSystemAppScope()

Currently SWAs will not show the Custom Tab Bar for any
chrome:// or chrome-untrusted:// URL. This change allows SWAs to
explicitly set additional URLs to be considered in-scope.

This change will allow Terminal SWA to host nassh extension in a tab
and have it considered in-scope and not show the Custom Tab Bar.
