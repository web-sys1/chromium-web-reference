Add an animation registry to holding space.

The holding space animation registry is a singleton registry which
lifetime independent of holding space views. This allows it to own
animations which can also live independent of holding space views as
well as allows for sharing of holding space animations across various
parts of UI. This makes it possible to keep animations in sync which
refer to the same underlying objects.

Currently only progress ring animations are added to the registry. These
animations are keyed in one of two ways:

- For cumulative progress, the animation key is a pointer to the holding
  space controller.

- For individual item progress, the animation key is a pointer to the
  holding space item itself.
