[privacy sandbox settings] Add custom scrollbar

Add custom scrollbar for the privacy sandbox settings dialog.
The scrollbar on Windows doesn't fit with the style of the dialog, it
also causes width to jump to make space for the scrollbar when the
"Learn more" section is expanded.

To avoid animating the scrollbar on the section collapse, hide scrollbar
for not-scrollable dialogs. Without it, the scrollbar thumb grows
until it fills the track and disappears. Also, add showing a divider if
the dialog is scrollable from the beginning.
