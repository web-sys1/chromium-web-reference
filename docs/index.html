Add capability for privacy sandbox restricted measurement notice

This CL adds a capability function, with a temporary implementation
that allows callers to know if an account is subject to be shown
a notice before enabling privacy sandbox restricted measurement APIs.

Once the account capability is define on the server-side, this method
will be updated to use the corresponding account capability.
