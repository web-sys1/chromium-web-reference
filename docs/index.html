[Task Nudge] Fix label alignment in GlicActorTaskIcon

TabStripNudgeButton calculates default margins that account for a close
button. In the context of the GlicActorUiNudgeRedesign, these defaults
resulted in the label text being misaligned with the absence of the task
icon.

This CL explicitly sets the views::kMarginsKey on the label to a fixed
value when the redesign feature is enabled. This overrides the base
class behavior to ensure correct symmetric padding.
