[anchor-position] Add |NGAnchorQuery|

This patch adds the |NGAnchorQuery| class, which keeps a map
of anchor-names to their bounding boxes.

|NGPhysicalFragment| has this map for its descendants. A WIP
prototype to resolve `anchor()` functions[1] uses this map.

This patch has no behavior changes, as the changes are under
a runtime flag.
