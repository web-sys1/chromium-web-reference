Record metrics for drive suggestion subtypes

To get better idea of what kind of drive suggestions users launch from
continue section, record impressions and user actions for different
types of drive and local file suggestions. To support this, pipe down
the files' justification types (currently used only to generate
justification strings) to app list search result type (as result
subtypes), and record the result subtypes with other app list search
impression metrics.
