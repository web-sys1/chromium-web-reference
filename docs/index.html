SET/viz: Introduce callback for transition directives.

This patch adds callbacks that are invoked when viz is finished processing
transition directives. This is important for both prepare and start.

In the prepare case we need to know when we received the copy output
request. If we call animate too soon, it will fail because the copy
output request has not yet been satisfied.

For start, this is important since we plan to expose (via a promise)
when the animation is finished.
