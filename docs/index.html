Minor refactoring to StyleEngine's font face invalidation

Currently, StyleEngine::Apply(User)RuleChanges does things
in this order:
1. Reuild FontFaceCache
2. Add active style sheets
3. Call FontSelector::FontFaceInvalidated() using information
   obtained in step 1

This patch changes the order into 1, 3, 2 to make the code
more compact.
