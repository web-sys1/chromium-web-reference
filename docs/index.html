[PWA] Fix the tab helper to not consider a 'suggested app'.

The WebAppTabHelper (and other places) have been 'seeing' suggested
apps that aren't installed, where they likely should not. This was
likely a problem in a recent metrics CL. This solves the root cause
by making the `FindAppWithUrlInScope` call to no longer use the
SUGGESTED_FROM_ANOTHER_DEVICE install state.

This fixes some tests that were OK to change, and one place where the
code actually did want to see if a SUGGESTED_FROM_ANOTHER_DEVICE app
was installed.
