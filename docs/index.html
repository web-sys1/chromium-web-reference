Use structuredClone() instead of JSON.parse(JSON.stringify())

It is common to perform deep copy of JS objects by
const clone = JSON.parse(JSON.stringify(myObj));

A native structuredClone() API has been introduced and should be
preferred. It also results in better TypeScript type checking.

This is in preparation of enforcing this guidance using ESLint during
presubmit checks.
