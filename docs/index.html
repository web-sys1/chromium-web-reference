[Files]: Fixes ignored attempt to cancel a touchmove event

Files app is generating an ever increasing number of runtime errors
steming from an attempt to cancel an event while scrolling is taking
place. During scrolling the events cannot be canceled. The fix is to
check if an event can be canceled before trying to cancel it with the
preventDefault call.
