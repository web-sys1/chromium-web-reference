[Vertical Tabs] Lay out vertical tab container when not collapsed

This CL does the following:
 - Lays out VTS along the far left side of the window
 - Places the THSP below the bookmarks bar and adjusts the shadow box
   accordingly
 - Correctly positions VTS, THSP, and contents in immersive mode
 - Creates a placeholder GetPreferredSize() for VTS to ensure it is
   reasonably-sized

This CL does not:
 - Properly handle overlap of VTS and caption buttons
 - Handle collapsed VTS
 - Properly draw toolbar color under caption buttons in VTS mode
 - Fully test any of this behavior

The "does not" entries will be addressed in follow-up CLs.
