Add KerberosUseCustomPrefilledConfig policy

This policy is an utility addition for KerberosCustomPrefilledConfig
(used to be called KerberosDefaultConfiguration).
KerberosDefaultConfiguration was a string policy, which had unset
value different from empty string. While it was working correctly
for local tests, DPanel treats unset string as empty. Now the
KerberosCustomPrefilledConfig's unset value is "" and the previous
default value is introduced via a boolean flag
(i.e. KerberosUseCustomPrefilledConfig).
