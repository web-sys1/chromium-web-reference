Original change's description:
> ash: Video conference effects manager, abstract effect interface
>
> VideoConferenceTrayController now owns a
> VideoConferenceTrayEffectsManager. Classes that want to host one or
> more effects must implement the VcEffectsDelegate interface and
> register themselves with the effects manager.
>
> An effect can be of type "toggle" or "set value," and consists of one
> or more states. A state contains everything needed (icon, text,
> callback, etc.) to construct the UI control through which its parent
> effect is exposed to the user. For construction of the UI, the effects
> manager provides methods that return the effects and their states
> arranged for layout in the bubble.
