Delay async script execution until next LCP candidate is found

LCP is decided from multiple LCP candidates. The final LCP is determined
by the timing when the largest object was painted afterward. This CL
delays running async scripts until the next LCP candidate is found so
that processing for LCP candidates is prioritized over running async
scripts.
