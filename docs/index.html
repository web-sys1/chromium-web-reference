desk_templates: Add inactive tabs to the DesksTemplatesIconContainer

When the DesksTemplatesIconContainer was first implemented, it only
considered the favicons of active tabs for it for the sake of
simplicity. Ideally, the DesksTemplatesIconContainer would display the
favicons of inactive tabs as well.

This CL makes it so the favicons of inactive tabs can also be added to
the DesksTemplatesIconContainer as well.
