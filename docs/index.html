Revert "Flags UI: Remove jstemplate."

This reverts commit 13a42b4af1cfb3e08725bb6634b89bdf5d85d261.

Reason for revert: It is breaking chrome://flags on iOS.

Original change's description:
> Flags UI: Remove jstemplate.
>
> jstemplate is deprecated and no longer used. This CL removes
> its usage in chrome://flags, which also makes the page less
> wasteful while rendering content.
>
> The template rendering strategy attached all available experiments into
> the DOM twice, one in a section for non-default experiments and one in a
> section for default experiments. It then toggled the visibility of each
> element depending on whether it was default or non-default.
>
> This approach instead only attaches each experiment to its respective
> section depending on its status as default. Therefore, no extra elements
> need to be attached and toggled to 'hidden'.
>
> This CL brings the time it takes to render experiments down from
> 1.5s to 1.25s on Linux.
>
