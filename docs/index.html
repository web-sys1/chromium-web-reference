[WebLayer] Bring up ads page load metrics observation

This CL brings up AdsPageLoadMetricsObserver in //weblayer and ports
a sufficient subset of //chrome's browsertests to test the integration.
This CL also changes AdsPageLoadMetricsObserver's callback for the
application locale to return the string by value rather than const
reference, as //weblayer's getter for the application locale returns it
by value and would not be easy to change.

Bringup and testing of ad interventions will follow.
