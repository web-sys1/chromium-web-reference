[mullet m3] Remove trigger source for AED::DidSelectSuggestion.

AutofillSuggestionTriggerSource (ASTS) defines what triggered the
suggestions (i.e. a form field click, manual fallback, etc.). There
can be only 1 popup with suggestions at a given time. Thus, the trigger
source can't change between 2 calls to AED::OnQuery.

This change removes trigger source from AED::DidSelectSuggestion by
using the trigger source stored inside the AED.

Next steps: remove trigger source from AED::DidAcceptSuggestion.
