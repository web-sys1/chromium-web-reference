Implement KioskInput API in Lacros

The KioskInput API was added for Ash in Q4/23
(see go/gb-kiosk-input-api).
This API is a kiosk-only extension API that allows an extension
to change the current keyboard input language.

This CL also makes this API available in Lacros, to ensure feature
parity between Ash and Lacros.

For the crosapi I opted for a generic `InputMethods` API rather than
a kiosk specific `KioskInput` API, since the functionality the API
exposes ('change selected keyboard input language') is in no way
kiosk specific, even though it is currently only used for a kiosk
specific extensions.
