Enable curtain mode in It2MeDesktopEnvironment

This CL contains 3 changes to make curtain mode work in CRD on ChromeOS:
   - Invoke ash::curtain::CurtainModeController::Enable() when starting
     the curtain mode.
   - Update AshProxy::TakeScreenshotOfDisplay() to not capture
     the curtain when using the AuraDesktopCapturere.
   - Added AshProxy::MakeWindowCapturable() which allows capturing
     the non-root uncurtained window when using the
     FrameSinkDesktopCapturer.
