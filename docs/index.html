Drop enrollment from DeviceCloudPolicyInitializer.

Remove the enrollment flow from DeviceCloudPolicyInitializer and move it
to its only user EnterpriseEnrollmentHelperImpl. The change is not
expected to bring any functional changes but refactoring.

Unpredictable lifetime of the initializer caused crashes in the
enrollment helper which depended on it. Now the initializer is fully
internal part of BrowserPolicyConnectorAsh and may have any lifespan.

Before this change the initializer did not connect the policy manager if
enrollment was in progress. It also did not reconnect the manager once
connected. With this change the initializer is not blocked by the
enrollment and the enrollment helper does not reconnect the policy
manager if it has been connected previously.
