Reland "Improve a11y for Crostini "Turn on" button in settings"

Reland of crrev.com/c/3167116, which was reverted as the Crostini Tast
tests needed to be updated (already done in crrev.com/c/3312664).

As per the linked bug, ChromeVox currently reads out the following four
strings when focussing the "Turn on" button:
1- Developers
2- Button
3- Run Linux tools, editors and IDEs on your Chromebook. Learn more
4- Press Search plus Space to activate

This CL removes the aria-label which makes the main description be
"Developers", and instead be the actual text "Turn on".
