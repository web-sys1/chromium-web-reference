emoji-picker: add flex layout to emoji-group

To allow removing emoticon-group component, its main functionality
which is providing flex layout is now added to emoji-group. With
this new change, emoji-group with "use-flex-layout" attribute will
provide the same view as emoticon-group.

It means that all the styles associated with emoji-group and its
elements must stay the same in the current version emoji-picker.

Note: The CL must not impact the user view of emojis presentation.
