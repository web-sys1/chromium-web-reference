Add backdrop to supported pseudo elements

Currently DevTools only add and bind ::before, ::after, ::marker pseudo
elements when initializing. This CL adds ::backdrop to the list to avoid
the confusion: when we have ::backdrop in the DOM tree before DevTools
is open, DevTools won't bind ::backdrop and therefore cannot show it.
