[ios] Return early if no valid return to recent tab WebState

This fix will ensure that the WebStateList is not updated to an invalid
index value because there is no valid Return to Recent Tab WebState.
Nulling out the returnToRecentTabItem will also ensure a reload will
not have the return to recent tab tile.
A UMA action will be logged to still keep track of these invalid cases.
