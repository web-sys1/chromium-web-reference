Remove the parameter to `PaintLayer::SetNeedsCompositingInputsUpdate`

The `mark_ancestor_flags` parameter does nothing but make the method
a noop if it is `false`. It is also set to `true` by default, but no
callers pass the parameter at all. This change removes it, without
change in functionality.
