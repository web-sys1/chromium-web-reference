Fix presentation callback in WidgetBase::ForceRedraw()

`WidgetBase::ForceRedraw()` forces a redraw and then uses a presentation
callback to determine when the redraw is completed by being presented on
the screen. However, the callback it currently uses might be called even
if the submitted frame fails to present successfully on the screen. This
CL fixes it by using callbacks that are called after a frame is
successfully presented on the screen.
