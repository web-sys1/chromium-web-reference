Skip MathML <mtr> in ancestor wrapping

When copying MathML table content, <mtr> elements were being serialized
twice. Once during DOM traversal and again during ancestor wrapping
resulting in duplicate tags. Skipping <mtr> in the ancestor wrapping
loop to prevent this duplication while preserving MathML structure.
