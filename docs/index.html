Introduce AutofillUseBRAddressModel and AutofillUseMXAddressModel

This CL introduces AutofillUseBRAddressModel and
AutofillUseMXAddressModel as additional feature flags.

The affected logic is already gated by AutofillUseI18nAddressModel. The
two new feature flags enable a more fine-granular rollout of only one of
the countries.
