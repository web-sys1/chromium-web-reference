Android: Replace onBackPressed with OnBackPressedDispatcher (2)

As a follow-up on crrev.com/c/3421446, replaces all the overridden
method |onBackPressed| in *Activity classes in Chrome with
OnBackPressedDispatcher/OnBackPressedCallback.

When a back press event is not handled, we disable the current callback
and call the dispatcher again to let other parties get a chance to
handle it. This was not necessary in 342146 for ChromeActivity which
is always at the bottom of the activity stack.
