[FilesBanner] Implement timeLimit banner functionality

State banners have the added functionality of only appearing for a
finite period of time (unless otherwise specified). This requires
maintaining the time the banner has been visible in the Files app. This
adds that functionality as well as the ability to return
Banners.INFINITE_TIME_LIMIT if the banner needs to always persist.
