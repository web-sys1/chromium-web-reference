WebView: Replace Context#bindService calls with ServiceHelper#bindService.

Android platform disallows calling Context#bindService() from within a
BroadcastReceiver context. If we call this method anyway, then bindService()
throws a RuntimeException.

In order to avoid this RuntimeException from occurring,
the Context#bindService calls are replaced with
ServiceHelper#bindService calls that circumvent this exception.

The set of WebView instrumentation tests were used to test the changes since no new functionality was added.

Test: `out/Default/bin/run_webview_instrumentation_test_apk`
