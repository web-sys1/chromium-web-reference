[WebUITabStrip] Crash @CrosapiNewWindowDelegate::WindowObserver::SetWindowID()

When a new window is created, as a result of a drag 'n drop operation
with WebUITabStrip ON, the closure procedure
CrosapiNewWindowDelegate::WindowObserver::SetWindowID() is invoked.
It calls out to CrosapiNewWindowDelegate::DestroyWindowObserver()
which destroys the WindowObserver/callee instance itself.

This CL returns right after the destruction, so avoid
use-after-free crashes. Logically, it is also correct to return since
the execution of the closure is complete.
