[motionmark] Disable injecting some extra code during telemetry.

The web-page-replay system injects some extra javascript before the
tests are run, which override the default behaviour of some functions
(e.g. Date functions). This interferes with the motionmark stories,
specifically the Design test, and makes the motionmark story noisy.  So
disable the injection of this script for the motionmark stories.

It may make sense to disable this injection for many additional stories
in the rendering benchmark, although it is not yet clear whether that
will help with de-noising any of the other stories. So the default for
the benchmark is still to enable the injection of the custom script.
