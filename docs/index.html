[Android][UPM] Reenroll on toggling password saving settings

When users are unenrolled, they see the old password settings page.
Toggling the "save password" button off and on again will reset any
previous unenrollment.

This is a niche use case and makes it a simple way to reset the
enrollment status. If users don't like the UPM in general, they can
still use the chrome:flag to permanently disable the UPM.
