[Settings] Do not finish activity on clearing browsing data

In the new single-activity mode of the settings UI, a single instance of
SettingsActivity is used throughout navigating the settings hierarchy,
so settings should not use getActivity().finish() just to go back to the
previous page. Use getActivity().onBackPressed() instead.
