Migrate LongScreenshotsTabService off MemoryPressureMonitor

In the future, there will not be a global memory pressure state. Each
listener will be told how much memory they are allowed to use, allowing
chrome to react in a more controlled manner to memory pressure.

This requires all callers of GetCurrentMemoryPressureLevel to be
migrated to MemoryPressureListener.
