[CrOS Cellular] Track APN migrated networks in ManagedCellularPrefHandler

This change enhances ManagedCellularPrefHandler so that it can provides
a way to store cellular networks that have been migrated to the APN
Revamp feature.
The migration process will happen for existing cellular networks after
the APN Revamp flag has been toggled. The migration will happen only
once, and it cannot be reverted, thus the requirement to track
migrated networks.
