Report skipped Windows WebShare tests as SKIPPED

The various Windows WebShare tests currently have early returns to skip
their execution from non-supported environments, but this deceptively
means it looks like these tests are running on these platforms.

This change updates these tests to report these tests as SKIPPED (which
still rolls up to 'all tests passed'), as well as adding 1 test to run
in the non-supported environment to verify the behavior there.
