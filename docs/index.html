[Fontations] Remove FreeType hb_font backend in HarfBuzz

Remove the build dependency from HarfBuzz to FreeType in our build
files. hb-ft.c mostly contains an hb_font_t implementation backed by a
FreeType FT_Face. We do not seem to use that in Blink. If we use
hb_font_t functionality, we rely on HarfBuzz' own implementations, which
it has gained in recent years.
