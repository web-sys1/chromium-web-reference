[MSan] Ensure observers outlive BrowserManager::observers_

Observer implementations typically unregister themselves from the
observer list at destruction time. Since `BrowserManager` owns things
that are also `BrowserManagerObserver`s, reorder `observers_` to be the
first field so it is destroyed after all other fields to avoid MSan
use-after-dtor errors.
