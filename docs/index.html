Reland "Convert media_app_guest_ui_browsertest to TypeScript."

This is a reland of commit becc38d509e0debd1fb3c2eb7186b10716831c07

Additional dependencies are now fed through build targets so that the
logic in ts_library.py that clears out stale generated files can be
engaged, should an incremental builder ever return to this version
after a follow-up converts a JS file to TS.

See https://crrev.com/c/2934619 for context.

To keep a lid on the gymnastics, separate the staging folder in two: One
for rollup that only contains JS, and one for TypeScript that only
contains TS (plus .d.ts for unconverted JS files along with their
corresponding .js files).
