content: Remove WebContents from render_widget_host_impl.h.

renderer_host/ should not know the WebContents type. The forward
declaration bypassed the DEPS check.

The forward declaration was just for a test function to be able
to `friend` the class, in order to call a public observer method which
was overridden as private. Since it's a public observer method and we
can just upcast to the base class to call it, this CL just makes the
overrides public (some discussion of the private override technique is
at [1] with mixed opinions).

[1] https://groups.google.com/a/chromium.org/g/chromium-dev/c/XdsXTHy9lis/m/yTzbG0fqAAAJ
