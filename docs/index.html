dlp: Synchronize Drag and Drop source metadata from Ash to Lacros.

As part of a move to support DLP DnD restrictions on Lacros, drag source
metadata (e.g. origin URL) need to be synchronized between Lacros and
Ash. This CL modifies Ash's DataOffer to serialize the drag
source metadata. This encoded metadata is then sent as a custom JSON
MIME type to Lacros. Lacros then reinterprets the drag source.
