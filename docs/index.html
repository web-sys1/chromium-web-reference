Enable the read buffer size switching in WebSocket

Read buffer size switching algorithm was added to the code and protected
by a feature which was disabled by default. This CL aims to set this
algorithm default.

The changes in this CL include:
- Remove some base::Features and base::FeatureParams and make the size
switching default
- Add a class to manage the read buffer size
- Add tests for the size switching
- Modify WebSocketBasicStreamSocketChunkedReadTest.OneMegFrame, which is
affected by the change of the initial read buffer size.
