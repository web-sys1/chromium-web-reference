[PA] Reserve memory for the shadow GigaCage on init

This CL allocates system pages for shadow GigaCage in
PartitionAddressSpace::Init(). The offset to the shadow GigaCage is kept
in GigaCageSetup and this CL also provides a function to get the offset,
which will be necessary in calculating the shadow metadata address.

Two GigaCages are mapped to different physical addresses in this CL, but
his should be modified to the same physical addresses in a later CL.
