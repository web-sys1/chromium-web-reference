Specialize policy priority enum for browser priorities.

The policy priority enum is specialized for priorities which can be
applied to the browser. This is done to isolate the complex
source-to-priority mapping in Chrome Browser from the simple 1:1 mapping
in Chrome OS.

As a result, the CompareEntryPriority(...) method also behaves
differently for browser (scope and source are used to determine the
priority, which is then compared) and OS (scope and source are compared
directly).
