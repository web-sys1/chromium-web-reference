[GPU] Eliminate AHardwareBufferImageBackingFactory::format_info_

|format_info_| is an array that for each ResourceFormat type stores an
internal |FormatInfo|. We want to eliminate this usage of
ResourceFormat, but there is no corresponding way to obtain "all
single-planar SharedImageFormats" (and we don't want to add one).

This CL replaces the usage of |format_info_| by a map that goes from
SharedImageFormat to FormatInfo. We put an entry in this map for each
SharedImageFormat type that AHBImageBackingFactory supports. We then
adjust the usage to account for the fact that there are now entries only
for supported formats, whereas |format_info_| has entries for all
formats that detail whether they are supported or not.

A followup CL will move the construction of FormatInfo into a helper
method to eliminate the need to insert the constructed FormatInfo into
the map in three separate places - I didn't do that in this CL to avoid
introducing noise. After that, I'll eliminate ResourceFormat from this
class altogether.
