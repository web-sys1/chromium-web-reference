[Lacros] Add support for temperature reading for system.cpu extension API.

This CL makes chrome.system.cpu.getInfo() on Lacros return the CPU
|temperature| field, like Ash. The underlying implementation uses Linux
sysfs-interface, and should be stable. So for simplicity, we just have
to make cpu_temperature_reader.{cc, h} available for Lacros.
