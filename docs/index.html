HttpCache: Remove NOTREACHED in RemoveEntry

net::disk_cache::BackendImpl::RemoveEntry would call NOTREACHED() if the
value it decremented went below zero. Since this value is on disk and
not under complete control of Chromium it is not appropriate to use
NOTREACHED() here.

Change it to STRESS_NOTREACHED() so that the invariant is still checked
when stress testing.

BUG=1431875
