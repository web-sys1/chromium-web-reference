[AF] Extract BAM::WillFillCreditCardNumber into an utility function.

BAM::WillFillCreditCardNumber is used to check if the form passed to
the BAM contains a credit card number field. This is useful for the
cases when the credit card number needs to be fetched.

The credit card fetching process is going to be decoupled from the
filling logic in BAM. This CL makes progress towards that by making
the function used to condition the fetching process reusable.
