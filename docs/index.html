app_list: Make some AppsGridView tests work with feature AppListBubble

Two goals:
* Reuse existing tests to get more coverage of AppListBubble
* Prepare for the day when AppListBubble is enabled by default

Therefore:
- Remove test parameterization by "new drag spec", since this feature
  is launched and on-by-default. This reduces the number of
  combinations of tests we need to run.
- Introduce AppsGridViewClamshellTest, which is parameterized by
  feature AppListBubble
- Introduce AppsGridViewNonBubbleTest, to mark tests that are
  explicitly not expected to work with AppListBubble, and can be
  deleted when the flag is removed.
- Move a couple of tests to other suites, closer to the code under
  test.

Future work: Parameterize AppsGridViewDragAndDropTest. I didn't put
it in this CL, since this CL is getting large. My intent is to move
the drag helper functions into the AppsGridViewTest base class, or
maybe into AppListTestHelper.
