[WebLayer] Expose ability to fetch access tokens to C++

This CL adds a public C++ API for fetching access tokens in WebLayer,
as well as an Android implementation of this API that proxies calls
from C++ to the Java-side client. This support will be needed to add
access tokens to Safe Browsing realtime URL lookups.
