Update chrome://version for ARM64 Windows

Chrome on Windows displays the CPU type as (32-bit) or (64-bit) but in
modern Windows there are actually five possibilities:

  (32-bit) - x86 native on 32-bit or 64-bit Windows
  (64-bit) - x86_64 native on 64-bit Windows
  (32-bit emulated) - (x86 on ARM64 Windows)
  (64-bit emulated)  - (x64 on ARM64 Windows)
  (arm64)

It is important to distinguish between native and emulated, and between
x86_64 and ARM64, so this change adds three new options to what can be
displayed in chrome://version/ and chrome://settings/help. The (arm64)
label is shared with OSX, the (32-bit)/(64-bit) labels are kept from
before, and the (32-bit emulated)/(64-bit emulated) were inspired by
OSX but using Windows nomenclature.

intended to be translated.
