[iOS] Enable remoting feature when use_blink is enabled

This CL enables the remoting feature on iOS-blink with disabling
a failing test,

  - ChromotingClientRuntimeTest.StartAndStop

To enable the feature on iOS blink, this CL removes the gn args
in setup.gn-config. Then, configure 'enable_remoting' in
remoting_enable.gni via 'use_blink' variable.

Additionally, this CL needs to add 'enable_remoting' item
to //tools/mb/mb_config.pyl. in order to make ios trybots have
`enable_remoting=true` gn args when it's enabled.
