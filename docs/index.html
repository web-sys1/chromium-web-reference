Bubble: remove all SizeToContents() in /chrome/browser/ui/views/passwords

Remove all SizeToContents() and add InvalidateLayout() when necessary.
This changes is used to make the bubbles using autosize mode via
autosize parameter.(autosize mode: When the preferred size of bubble
changes,its widget automatically resizes itself to meet the content's
size)

The ultimate goal of this bug is to remove all SizeToContents() to
reduce developer confusion about when SizeToContents() needs to be
called.
