[iOS][SK] Updates tab group view after collab id updates

This CL adds a `CollaborationGroupIDNotifier` class that notifies the
`TabGroupMediator` when the collaboration ID of a group is updated.
The `TabGroupMediator` then calls its consumer to correctly update
the share state and the facePile of the tab group view.
