Allow snapshot in overview mode for lacros windows

This CL allows snapshot in overview mode for lacros windows. This means
occlusion will be paused during overview enter and exit animation, which
improves performance. We will keep the code supporting disabling
snapshot support until we determine it's fine for overview mode to show
lacros snapshots, since lacros may behave worse than ash here.
