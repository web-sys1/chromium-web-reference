[anchor-position] Add tests for multiple anchors in multicol

Following up a review comment[1], this patch adds a test that
has multiple OOF anchors in different containing blocks in
multicol.

Similar to the `anchor-position-multicol-003.html`, the
current code works only if the containing block of the
querying elements is outside of the multicol, and this test
doesn't include the situation, but makes sure r1040243
crrev.com/c/3856814 works for multiple containing blocks.
