[Pixel Diff Ash] Support the policy wallpaper in pixel tests

This CL introduces a structure named `pixel_test::InitParams` to
customize the pixel test setup.

`pixel_test::InitParams` contains an enum data member to specify
whether a regular wallpaper or a policy wallpaper should be set up.

`AshTestBase::PrepareForPixelDiffTest()` creates default init params.
`AshTestBase::SetPixelTestInitParam()` is created to support the
pixel test setup customization.

This CL also adds a test case to verify the login shelf with a policy
wallpaper. With the policy wallpaper, the solid-colored shelf should
show.
