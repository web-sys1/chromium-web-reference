[Video Tutorials] Fetch fresh data if accept languages have changed

This CL adds support for re-requesting fresh data from server if the
accept languages have changed. This is checked at startup and anytime
after startup whenever the language changes. The new data fetched won't
be available to use until next chrome session. Also restructured the
manager class a bit to allow caching of the calls before initialization.
