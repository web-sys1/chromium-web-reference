[sync] DualLayerUserPrefStore: Notify only if effective value changes

This CL wraps up a few-todos to notify the observers only when the
effective value of a pref changes.

Without this check in RemoveValue(), the usage of DualLayerUserPrefStore
can lead to a recursive callback and thus, a crash. This CL,
consequently, unblocks crrev.com/c/4358058.
