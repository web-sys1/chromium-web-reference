Add toggle in settings for Web Actuation consent.

This change introduces a new toggle to the Gemini settings page
(chrome://settings/ai/gemini) for controlling the web actuation setting.
This new "Let Gemini browse for you" setting allows users to opt-in to
Gemini browser control.

The implementation includes a new settings-toggle-button, an expandable
section with more details. The strings are not finalized and are marked
as translateable=false. Will update with finalized strings after they
are approved on 10/20. The new UI is conditionally rendered based on the
`webActuationFeatureEnabled_` flag. UMA metrics have been added to track
the on/off toggle action as well as "Learn More" clicks for the new
setting.
