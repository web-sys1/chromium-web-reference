Web Apps: Add InstallFromManifestCommand to install a manifest string

InstallFromManifestCommand installs a Web App from a given manifest
JSON string (and URLs indicating where the JSON was fetched from). This
can be used to install a web app using the full manifest data without
communicating with the app origin.

This CL includes the minimum required to install the app. Still to do in
