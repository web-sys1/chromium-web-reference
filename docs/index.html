Move MojoBindingContext to renderer/platform.

This is for 2 main reasons:
- It allows us to make it garbage-collected (GarbageCollectedMixin).
- It does not need to be accessed from content. Instead, if we do need
  a way to refer to it from content, we could provide a static
  MojoBindingContext::FromLocalFrameToken() API or similar, and pass the
  token in from content.
