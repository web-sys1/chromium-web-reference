Parse JSON on Android just like other platforms

Android's strategy for parsing untrusted JSON has been to "sanitize" the
JSON by round-tripping it through a Java (memory-safe) parser, and then
parsing it with the in-process JSON parser. This is to avoid spawning a
service process as is done on other platforms.

With the advent of the Rust parser, controlled both by build and runtime
flags, we want to use that on Android when enabled.

This refactors the #ifdef's to be more intelligible and avoid
duplicating code.
