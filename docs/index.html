Add more unit test coverage for WebContentsAccessibilityImpl on Android

This CL adds to the unit test suite for WebContentsAccessibilityImpl
on Android.

With this CL we add tests for the performAction method for the
actions:
  ACTION_SET_PROGRESS
  ACTION_NEXT_HTML_ELEMENT
  ACTION_PREVIOUS_HTML_ELEMENT

These are added to the WebContentsAccessibilityTest suite.

We also add an extra check to the logic that updates the cached
AccessibilityNodeInfo objects to compensate for a small bug
in the Android framework on L/M.
