Support holding space commands in Files app recents.

Previously holding space commands were not enabled for recents sections
of the Files app. Now, holding space commands are enabled provided that
the selection of entries contains entries from supported volumes.

Per UX recommendation, entries from unsupported volumes are ignored
with respect to command execution.
