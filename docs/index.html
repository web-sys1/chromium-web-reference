Avoid redundant calculation in NGColumnLayoutAlgorithm::CalculateBalancedColumnBlockSize()

This patch changes |CalculateBalancedColumnBlockSize()| to avoid
redundant calculation of tallest content run for improving code health.

|NGColumnLayoutAlgorithm::ConstrainColumnBlockSize(size)| always does
|size = std::max(size, tallest_unbreakable_block_size_)|. Thus, we
don't need to pass |size| less than |tallest_unbreakable_block_size_|.
