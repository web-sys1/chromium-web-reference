Add a flag for SVG raster optimizations

This patch adds SvgRasterOptimizations for the following optimizations:
1) crbug.com/1404998: SVG sprite sheet culling
2) crrev.com/1112415: Increased raster invalidator error tolerance

The error tolerance change applies to non-SVG transforms as well, but
we think it is most common with SVG due to the viewbox transform that
has scale components. This patch also contains a change to
fieldtrial_testing_config and a webview flag to set the experiment up
for experimentation.
