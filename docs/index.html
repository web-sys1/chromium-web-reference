Make the InputMethodSettings mojom object extensible

This is an attempt to essentially reland
https://chromium-review.googlesource.com/c/chromium/src/+/4030016
incrementally.

Japanese settings in decoder initalization inside of the
ConnectionFactory::ConnectToInputMethod function -
