fusebox: define a fusebox MTP Volume in volume manager

This volume is the same as normal MTP volume with the following diffs

  • mount and source path are relative "/media/fuse/fusebox"
   • define "/media/fuse/fusebox" as path_util constant
  • file_system_type shall be "fusebox"
  • volume_id: original MTP volume id with a "fusebox" prefix

For testing the volume label will be "fusebox" prefixed so it differs
from the the original MTP volume when shown in files app. Eventually,
we will remove it.
