[FilesRecentFilter] Add group heading support for file list

Support time period group headings (e.g. Today/Yesterday/Earlier this
week) in Recent view.

* maintain a group by snapshot in FileListModel to pre-calculate group
  result when the list changes (e.g. splice) so the view
  (file_table_list) can use it for redraw.
* override couple of methods from List in FileTableList to make sure
  the height related calculation also takes group headings into
  consideration:
    * when calculating the item begin/end index for the viewport
    * when calculating the item top for a specific index
    * when calculating the height for after filler
* use a :before CSS to show group heading for a list item so we don't
  need to add a physical dom element for it
