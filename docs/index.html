Add kLowPriorityScriptLoading feature (disabled by default)

This CL adds kLowPriorityScriptLoading feature. Before this CL, the
following scripts had a "Low" fetch priority.

- async scripts
- deferred scripts
- in-order scripts

After this CL, if kLowPriorityScriptLoading is enabled these scripts
will have a "VeryLow" fetch priority (there are some exceptions such as
render-blocking scripts). The minimum fetch priority for images is
"Low". So kLowPriorityScriptLoading always prioritizes fetching images
rather than fetching these scripts. LCP is determined by the largest
text or image. So we hope kLowPriorityScriptLoading improves LCP. But we
also observed that some LCP images are dependent on async scripts. In
such cases, we expect that LCP will be regressed.
