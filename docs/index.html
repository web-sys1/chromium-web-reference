[Telemetry] Determine device ownership in LaCrOS

This CL changes the way device ownership is determined in
LaCrOS. Currently device ownership is determined by checking
whether the current LaCrOS profile is the main profile, but doesn't
take device ownership information from Ash into account. This is changed
by hooking into the browser init params that contain the information.
