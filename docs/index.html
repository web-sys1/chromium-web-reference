Update base::win use of IsWow64Process to IsWow64Process2

This change updates the use of IsWow64Process in base/win to
IsWow64Process2, and creates new helper functions to provide information
about common scenarios of interest in regards to WOW emulation.  These
helper functions can be used to determine:
 1) If WOW64 emulation is disabled
 2) If an x86 process is being run on an AMD64 device
 3) If an x86 process is being run on an ARM64 device
 4) If an x86 process is being run on some other 64 bit device.
