WebNN: Enable `kUseDeviceTensor` feature flag by default

We introduced the feature flag `kUseDeviceTensor` in the landed CL[1]
to control whether to use the device allocator. It was disabled by
default.

According to my local tests, there is no WPT regression. Now enable
it by default for test infrastructure doing automatic performance
measurement and comparison.

[1] https://chromium-review.googlesource.com/c/chromium/src/+/6972837
