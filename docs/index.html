dlp: Add scanning IOTask progress values

Adds number of scanned items to the ProgressStatus structs, in order to
show progress for scanning in progress notifications. This is roughly
the percentage of the number of scanned items out of the total items.
In some cases this won't be accurate, like in case of uploading entire
folders, or because some items are not scanned at all. The main purpose
is to show the user that some progress is happening.
