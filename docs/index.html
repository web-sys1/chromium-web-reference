Add a Client Hint for the reduced User-Agent header string

As part of allowing origins to opt into receiving the reduced user agent
string, we are introducing a (temporarily-available) client hint
Sec-CH-UA-Reduced, which if set to "?1", indicates to the origin that
the User-Agent header contains the reduced user agent string.

NB: this CL only adds the client hint.  A subsequent CL will add the
functionality to send the client hint to the origin.
