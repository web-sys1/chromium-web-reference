Make AXScreenAIAnnotator available ScreenAI service's Annotate.

Annotate function of ScreenAI service sends back accessibility tree
data to AXScreenAIAnnotator. To make that possible, when Annotate
function is bound, it is ensured that AXScreenAIAnnotator also exists.

This was previously done through creating AXScreenAIAnnotator if PDF
OCR service was enabled, but now only happens if PDF OCR is
triggered.
