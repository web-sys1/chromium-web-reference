[parallel-text-shaping] Use Persistent<FontFaceList> in CSSSegmentedFontFace

This patch changes `CSSSegmentedFontFace` to use
`Persistent<CSSSegmentedFontFace>` instead of embedded object to avoid
using `LinkedHashSet<Member<FontFace>>` which doesn't work well.

This patch is follow up of the CL[1], where it changes
`Member<FontFaceList>` to `FontFaceList` in `CSSSegmentedFontFace`.

[1] http://crrev.com/c/3670315 [parallel-text-shaping] CSS changes
