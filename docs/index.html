Cleanup NetworkPortalNotificationController

This CL does the following cleanup:
1. Removes NetworkPortalNotificationController from the chromeos namespace
2. Removes the NetworkPortalWebDialog co-dependency, using a Delegate
   interface instead.
3. Uses a private internal Delegate to avoid exposing Show/CloseDialog as
   public methods.

These changes have no behavioral change but will facilitate additional
refactoring where NetworkPortalNotificationController supports multiple
delegates.
