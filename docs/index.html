[iOS] Add workaround for omnibox text length bug

The bug here is that sometimes the |self.text.length -
self.autocompleteTextLength| subtraction overflows and becomes huge.
It's unclear why this happens, so this CL adds a debugging DCHECK and
keeps the code from crashing in this case.
