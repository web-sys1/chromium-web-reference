[Extension]: Add persistAcrossSessions flag for registering scripts

This CL adds a flag which allows extensions to specify whether each
dynamic content script registered will persist across sessions or not.

This flag is set to true by default, and persistent scripts are cleared
when the extension gets uninstalled/updated.
