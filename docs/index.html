color provider: Remove NOT_IMPLEMENTED from NativeColor functions in ChromeOS

Receiving the following errors in the latest ChromeOS build since https://source.chromium.org/chromium/chromium/src/+/af14fc18d01c158912f5aeb3297675aa399cbbb2

/var/log/chrome/chrome
```
2021-05-25T00:18:52.867884Z ERROR chrome[3967:3967]: [native_color_mixers.cc(14)] Not implemented reached in void ui::AddNativeCoreColorMixer(ui::ColorProvider *, bool, bool)
```

These functions are marked as NOT_IMPLEMENTED in CHROMEOS, but there is no plan to implement these currently.

As this breaks the DCHECK enabled builds, we should remove these.
