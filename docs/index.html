[results.html] Accept swarming task ids in SET_TASK_IDS() hook

The archive script appends the call to the end of full_results_jsonp.js
with a list of swarming task ids of the web tests step. Results.html
constructs invocations from the task ids when requesting QueryArtifacts
to query artifacts of the web tests step.

How to test:
1. Download https://chromium-layout-test-archives.storage.googleapis.com/linux-rel/1217030/blink_web_tests%20%28with%20patch%29/full_results_jsonp.js
2. Append the following line to the end of the downloaded file:
SET_TASK_IDS(['5edfc41c05116f11','5edfc419fc0ca611','5edfc418268f7f11','5edfc41601944811','5edfc413584e0611','5edfc41180a63111','5edfc40fa4fdc311','5edfc40e15d2a311']);
3. Open results.html?json=<full path of the downloaded file> in the
   browser.
