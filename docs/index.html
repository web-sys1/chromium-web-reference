[tvOS] Exclude unsupported component unit tests

This CL disables component unit tests that are not supported on tvOS.
These tests either rely on APIs that are unavailable on tvOS or require
including WebKit.h, which is not supported.

To avoid modifying existing dependencies, the unsupported tests are
simply excluded in BUILD.gn.
