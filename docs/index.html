Add base::Feature controlling the sizes used in visual feature computation

This CL adds a base::Feature to control the minimum viewport size, under
which we don't send visual features, as well as the sized used in
computed the blurred image. Both parameters will likely be decreased,
so that visual features are sent more often, with no relative privacy
loss. The exact values will be determined later.
