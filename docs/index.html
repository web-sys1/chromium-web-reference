lacros: Remove unrelated notifications from tests.

Ash shows signin error and multi-device notifications on startup for
Lacros tests. This can potentially interfere with interactive_ui_tests
that use ui_controls, as those send events in screen coordinates that
are not tied to specific windows. If there is overlap between the
notifications and test windows there could be problems with event
routing. This CL disables them for Lacros tests.
