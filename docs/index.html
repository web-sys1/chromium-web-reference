[PM] Make V8Memory data ExecutionContext associated.

Currently this data is associated with frames, but it is also measured
for workers. This moves the tracking class to be associated with
"ExecutionContexts", which encapsulates either a frame or a worker.

This also adds a little bit of syntactic sugar to the ExecutionContext
machinery to make it easier to use.
