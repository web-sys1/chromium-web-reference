File Handling: Move CombinedArg Runtime Feature to Generic Section.

Since https://crrev.com/c/2436756, the kNativeFileSystemAPI
blink::feature no longer exists, and "FileHandling" is enabled using
only one sole flag (kFileHandlingAPI).

Therefore, this feature no longer should be enabled in
SetCustomizedRuntimeFeaturesFromCombinedArgs(), which has a caution
comment to "only add custom enabling logic here if it cannot be covered
by the other functions", and should join the generic feature map at
runtimeFeatureNameToChromiumFeatureMapping.

Refactoring-only change. No intended functional changes.
