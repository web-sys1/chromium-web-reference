[Reland] capture_selfie_cam: Handle an edge case for camera previw

The original CL was reverted only because they needed to cleanly revert
a parent CL (CL:3551789). There are no changes in this CL from the
original (CL:3546842).

This CL handles the edge case while pressing on the resize button on
camera preview but releasing the press outside of camera preview. At
this use case, capture session should pass the events to camera preview
even they're not on top of camera preview, because the press event is
on the camera preview, the following events are all targeted on camera
preview until the press is released.
