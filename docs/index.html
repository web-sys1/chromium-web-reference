Create Promos Manager histograms.

This change creates the following histograms:
- IOS.PromosManager.Promo
- IOS.PromosManager.Promo.ImpressionLimitEvaluation
- IOS.PromosManager.Promo.Type

These histograms enable us to ask the following questions about the
Promos Manager:

1) "What's the distribution of the types of promos being displayed to
users?"

2) "Which promos are being displayed to users?"

3) "What's the distribution for when the Promos Manager decides to
accept a promo for display, versus reject a promo for display (and the
various potential rejection reasons)?"
