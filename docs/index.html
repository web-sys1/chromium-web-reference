[AIAE] Fetch APC on amount extraction

Fetch APC right before amount extraction inference call is made. If
timeout is reached, an error dialog will be shown on UI.

We can do this because fetching APC is very fast, ~34ms on 90 percentile
and very reliable(https://screenshot.googleplex.com/a5ZG5eMCDK7kKWs).
This change simplifies the AI-based amount extraction flow and the code
path becomes much cleaner.
