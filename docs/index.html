Implement a memory ablation study on ChromeOS.

This CL creates a new class MemoryAblationStudy which will ablate memory
up to the finch-controlled ablation amount.

The implementation of the class allocates memory in 10MB chunks until
the desired amount is reached. Every 30s, the class picks a 10MB chunk
and reads one byte from each page to ensure that it's resident.
