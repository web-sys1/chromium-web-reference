CCA: Fix edge-fading effect on mode selector scrollbar

To avoid scrollbar thumb being covered by edge-fading effect, this CL
reimplements the effect by covering the fading area with gradient
background |before|, |after| of |div#mode-selector|, using |calc()| with
extracted |--scrollbar-height| variable to ensure their height won't
overrun onto scrollbar thumb.
