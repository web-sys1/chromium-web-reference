Add new CdmPrefService Mojo Interface

This change adds a new CdmPrefService mojo interface that allows the
cdm code under content/* to interact with the PrefService in order
to retrieve an origin id.

CdmPrefService exposes one method `GetCdmOriginId` which
uses the PrefService to store and retrieve the mapping of origin to
origin id.

`CdmPrefServiceImpl` implements the interface. Other changes in this
CL are all of the piping needed to connect MojoCdmHelper and this new
CdmPrefService.

This change also register a new pref "media.cdm.origins" in the
profile registry.
