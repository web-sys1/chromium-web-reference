web-engine: Disable flaking external intents test

The external intents test is flaking on CI because
it requires launching a new application which may take
a while.

We will disable this test on CI but keep it around to
run locally for testing.

We can also keep the test enabled that checks that
no application is launched if we disable this because
that should fail 100% of the time and a flake on that
test would be useful data (it doesn't flake).
