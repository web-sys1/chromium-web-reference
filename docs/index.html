iwa: Add pref to allow IWAs to focus windows programmatically

This change introduces a new web preference,
`allow_window_focus_without_user_gesture`, to control whether scripts
can programmatically focus windows without a prior user gesture.

This new preference is distinct from the existing
`allow_scripts_to_close_windows`as the ability to close windows is also
granted to extensions and system web apps, sometimes even without
requiring specific window management permissions. To maintain
finer-grained control and avoid unintentionally merging
