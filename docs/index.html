[gpu] Forward memory pressure notifications to the GPU process

The feature was previously only enabled on Windows. Enable it on other
platforms (except for Android) as well, since it has been shown in a
field trial to improve memory pressure on Chrome OS.

More specifically, this shows (see UMA, https://uma.googleplex.com/p/chrome/variations?sid=aafd959c9e4c1e0f988a8d6c2453cfa4):
- Reduced memory pressure (+1.9% more time without memory pressure)
- Fewer tab discards:
  - -9.5% at the 99th percentile
  - -5% total tabs discards (no confidence interval as it's a sum)
- 1-diamond stability improvements
