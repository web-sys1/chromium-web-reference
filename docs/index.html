CrOS Settings: Fork focus_config.ts from Browser Settings.

Given that the files that implement the "focus config" mechanism have
already forked between

 - settings_page/settings_animated_pages.ts and
 - chromeos/os_settings_page/os_settings_animated_pages.ts

sharing the file that holds the FocusConfig type definition does not
make much sense anymore. Forking it so that the two "focus config"
mechanisms are entirely independent, which also reduces the amount of
files that need to properly be shared between OS and Browser settings.
