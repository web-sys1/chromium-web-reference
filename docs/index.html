[CrOS] Add metrics to track shutdown from shelf shutdown button

To understand how many shutdowns came from the shelf shutdown button,
we tracked the user action when the shelf shutdown button is clicked.

There are already existing user action metrics to track shutdown from
tray and the physical power button, Accel_ShutDown_PowerButton and
Tray_ShutDown, but not for shutdown from the lock/loginin screen.
