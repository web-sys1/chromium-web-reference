webauthn: apply RemoteDesktopClientOverride extension to App IDs

CL:3577285 implemented browser-side support for the extension, which
lets authorized sites override the origin used to verify whether the
site is authorized to claim the RP ID for a MakeCredential or
GetAssertion request. This extends ths same behavior for access control
checks on App IDs, which are passed in the `appidExclude` and `appid`
extension for makeCredential and getAssertion respectively.
