Customize price tracking menu for magic stack entry point

A new card is being added to the magic stack which alerts the
user in the magic stack if there is a price drop on their tracked
products. The secondary action 'See more' opens the price tracking
menu, but without the 'Current site' part of the price tracking
menu. This CL customizes the price tracking menu to hide the
'Current site' part from the magic stack entrypoint.
