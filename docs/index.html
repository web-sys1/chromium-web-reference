[QuickUnlockPrivate] Add chrome.quickUnlockPrivate.getAuthToken() stub to Lacros.

This CL makes chrome.quickUnlockPrivate.getAuthToken() available to
Lacros. The Lacros extension impl is currently a stub, which will be
populated once the required crosapi is added. Since the Lacros impl
(using crosapi) will be significantly different from Ash impl, the new
code is added to separate *_lacros.{cc,h} files.
