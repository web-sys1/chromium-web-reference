[TTFCC] Review Touch To Fill surface hiding conditions

The conclusion is to hide the TTF surface:
 - [TBD] on user interaction (from Android side)
 - on renderer events (in OnHidePopup)
 - on navigation (new page loaded, tab gone)
 - [TBD] on TouchToFillDelegateImpl destruction

and NOT hide TTF surface:
 - on Android focus changes (since bottom sheet takes the focus)
 - on page dimensions changes (since the overlay UI doesn't depend on
those)

See [1] for a detailed explanation.
