[bfcache] Evict entry for network redirects/uploads, or datapipe drained

This CL adds bfcache entry eviction mechanism when a page is in bfcache
and  network redirects or uploads happen, or datapipe is drained in
bfcache.

This is needed now that BackForwardCache intends to support pages with
in-flight network requests.
