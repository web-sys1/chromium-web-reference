Fix the website metrics usage time when the tab is closed.

When the tab is closed, SetTabInActivated is called to calculate the
usage time in the past 5 minutes. However, the usage time is not added
to running_time_in_two_hours. SaveUsageTime is responsible for
writing the 5 minutes time to running_time_in_two_hours. Well, if
SaveUsageTime is not called by OnFiveMinutes, when OnTwoHours is called,
running_time_in_two_hours is recorded. Since the tab is closed,
url_info_ is updated to record the opened tabs usage time only, then
the closed tab usage time is lost.

Modify SetTabInActivated to add the usage time in the past 5 minutes to
running_time_in_two_hours for the closed tabs.
