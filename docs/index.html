Reland "[Sharesheet] Refactor SharesheetServiceDelegate." with fix.

This CL was originally reverted due to a heap-use-after-free error
in Linux Chrome OS asan builder. The error occurred in
SharesheetServiceDelegate where is_bubble_open_ was set to false
after the object had already been deleted in the previous line.
(OnBubbleClosed). This has been removed and a comment added such that
this error will not be made again. SharesheetBubbleViewBrowserTest
has also been updated to be more robust.
