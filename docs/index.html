Disable -Wsign-compare for QUICHE code

-Wsign-compare interacts poorly with a lot template magic that
CHECK_*/EXPECT_* macros use; in particular, it requires the caller to
write "EXPECT_EQ(variable, 1u)" instead of "EXPECT_EQ(variable, 1)" if
`variable` is unsigned.  google3 version of gtest does not have this
limitation, causing friction for Chromium merge.
