Pixel snap input.scrollWidth

input.scrollWidth and clientWidth sometimes differed by a pixel.
Root cause is that clientWidth was computed with pixel snapping,
and scrollWidth was not.

Element::scrollWidth does pixel snapping
HTMLInputElement::scrollWidth did not do pixel snapping

This fix adds pixel snapping to HTMLInputElement::scrollWidth
