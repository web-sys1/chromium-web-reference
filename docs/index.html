[iOS,Passwords] Update presaved user input on keyup action.

PasswordManager usually relies on form activity of type "input" for
updating cached user inputted values. However on some websites "keydown"
and "input" activity is never received, only "keyup" is available.
This CL starts to use this form activity type on par with "input" to
properly save credentials in such cases.
