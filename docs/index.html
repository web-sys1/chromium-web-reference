[DomStorage] Update LocalStorageImpl to use database ReadAllMetadata()

Updates `LocalStorageImpl` to use
`AsyncDomStorageDatabase::ReadAllMetadata()` in two locations:

1. The implementation of `mojom::LocalStorageControl::GetUsage()`, which
returns the storage key, last modified timestamp and total bytes for
each local storage map.

2. `DeleteStaleStorageAreas()` uses `ReadAllMetadata()` to determine
which stale local storage maps to delete. A map becomes stale after 400
days without use.
