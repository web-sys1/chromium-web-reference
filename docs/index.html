[ChromeVox] Fix nested list navigation.

When a list item has nested items (or other children), bullet and the
list text are considered two different objects when navigated using
Search+L/R.
To fix the issue, list markers that are immediately followed by a static
text are ignored during navigation, and are appended before the static
text when output is generated.
