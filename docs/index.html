Disable some obsolete tests in AffiliationsPrefetcherTest

This disables the "false" arm of AffiliationsPrefetcherTest (i.e., the
version where kFillingAcrossAffiliatedWebsites is disabled) on Desktop
and iOS, where that feature is already enabled by default.

This is necessary because these tests are blocking us from enabling
kPasswordsGrouping by default on iOS. Rather than going in and figuring
out why enabling kPasswordsGrouping breaks these tests, I opted to
simply disable them. Since they're basically dead code on this platform
anyway, I don't see any value in updating them.
