[Vertical Tabs] Fix ChromeOS hit testing

This CL fixes two bugs:
 - When projects panel was visible, hit tests in the toolbar/tabstrip
   would not have been run (even if the point is not in the projects
   panel).
 - On ChromeOS, the top container hit test happened before the caption
   button hit test, so in VTS mode the caption buttons could not be
   clicked; this has been fixed with a ChromeOS-only exception in the
   top container hit test logic.
