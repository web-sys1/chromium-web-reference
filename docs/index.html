[Stylus Handwriting] Implement Android logic for stylus recognition

This CL adds interfaces and stub APIs to support implementation of
Samsung Direct writing service connection for stylus handwriting
recognition and commit to HTML inputs.

The direct writing service & service callback aidls, along with their
corresponding implementations would be added in follow up CLs with
reasonable number of new APIs.

Add Direct writing service connection handler, event handler, and
supporting classes in //components/stylus_handwriting/ and add
interfaces for communication with ImeAdapterImpl.

Cursor location is used by the Direct writing service to show
animated commit of recognized text towards the cursor. This should
be available to the service by the time writing is done. Get the
Caret position and Focused node bounds when writing is started.
