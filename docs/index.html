Do not report forms that should not be parsed for autofill to UKM

We looked at the distribution of the number of fields of the forms that
were not autofilled, and all the them only have one field, which is a
search box. For these forms, |ShouldBeParsed| returns false and they
are not parsed by autofill, so we should not report them into UKM as
well. We will skip reporting forms to UKM if
FormStructure::ShouldBeParsed() is false.
