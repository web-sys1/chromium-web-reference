breadcrumbs: move BreadcrumbMgrKeyedService test

This change migrates breadcrumb_manager_keyed_service_unittest.cc to
the new breadcrumbs component, and simplifies it.
BreadcrumbManagerKeyedService used to be closely linked with
BrowserState, but as of crrev.com/c/2808046, it now instead just takes
a boolean indicating whether the profile is incognito or not. Because
of this, references to BrowserState in the test can be removed, and it
has become much simpler.
