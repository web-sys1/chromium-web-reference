Migrate variations::GetVariationParamValueByFeature to base counterpart

API defined for variations::GetVariationParamValueByFeature in
variations_associated_data.h/cc is just a wrapper over the equivalent
base API and has been deprecated. [1]
This CL attempts to migrate callers of GetVariationParamValueByFeature to
base::GetFieldTrialParamValueByFeature() as a part of code clean up.
