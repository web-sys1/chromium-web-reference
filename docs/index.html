[Web Payment] Test installer for desktop payment app tests

Before this patch, the desktop (Views) payment app tests for payment
apps was installing payment apps via PaymentInstruments API that is
planned to be removed.

This patch changes the desktop payment app tests to install payment apps
via a test-only installer utility. JavaScript is still being used for
enabling delegation in the already installed payment app.
