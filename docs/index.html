SVG Text NG: Preparation of "8. Position on path"

To implement "8. Position on path", we need to have start/end indexes of
addressable characters in a <textPath>. SVGTextPathRange represents this
information.

* NGSVGTextLayoutAttributesbuilder produces a list of SVGTextPathRange.

* It is owned by NGInlineNodeData through SVGInlineNodeData.

* The list of NGSVGCharacterData is also moved to SVGInlineNodeData.

This CL has no behavior changes.
