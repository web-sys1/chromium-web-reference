Ensure PWM clientside model is used for clients using password change

For password change experimentation in M140, we want to always
user the clientside classification model for clients who are
enrolled into improved password change (for those who have used the
feature in the past, and are still able to use it in the future),
however we do not want to check the feature flag on every page load
to avoid diluting the experiment, therefore we need to check the
enrolment status with the password change service.
