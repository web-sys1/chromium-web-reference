> [Android][Passwords] Get Passwords from ChromeSync
>
> This CL forwards requests to a downstream implementation and forwards
> the responses to the caller of GetLogins(), e.g. in the old settings UI.
>
> To achieve that, an interface is defined and instantiate by an abstract
> factory whose implementation lives downstream. The tests also live there
> since the entire android-specific implementation is unused in public
> builds that keep using the cross-platform, local store backend.
>
> In order to test or run this CL, you will need to include the API
> changes from https://crrev.com/i/4086348 and build with:
>
>   enable_chrome_android_internal = true  # to fetch the API SDK
>   use_signing_keys = true                # to talk to the 1st party API
