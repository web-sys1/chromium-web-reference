Integrate ARC power button menu with ChromeOS power button menu

Currently, if we launch ARC power button menu (via GlobalActions),
it will launch Android's power button menu, which is not usable.
Therefore, we should propagate the call and launch ChromeOS' power
button menu.

We are not using existing API `OnPowerButtonEvent` because in the
current flow it will automatically start pre shutdown after menu
has been opened. For ARC GlobalActions behavior, we only want to
open power button menu (similar to hold and release behavior).
