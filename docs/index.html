[TTFCC] Announce the card network for accessibility

The card network name (such as "Visa" or "Mastercard", etc.) is
announced for those cards that have a nickname.
The cards that don't have a nickname assigned by the user get a title
from the card network name. The title is always announced.

See the screenshot for example:
https://screenshot.googleplex.com/5cfqv7ypLHMxBaE

Here is what the code announces:
1) For cards with a nickname
[nickname], [network name], [4 last digits], expires on [expiry date]

2) For cards with no nickname
[network name], [4 last digits], expires on [expiry date]
