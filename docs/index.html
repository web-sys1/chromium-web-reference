[content] Dispatch OnMediaDestroyed on RenderFrame deletion.

With this CL, OnMediaDestroyed is dispatched for any media player that
still exists when a RF is deleted. A RF may be deleted when it
still has media players if, for example, its renderer crashes. Without
this change, any code that relies on OnMediaDestroyed to clean up the
state associated with a media player (e.g.
MediaEngagementContentsObserver::MediaDestroyed) leaks memory.
