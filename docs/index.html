Multiplanar: Add CreateSkImageForPlane in SkiaImageRepresentation

This change adds CreateSkImageForPlane(plane_index) to
SkiaImageRepresentation::ScopeReadAccess. This is needed when adding
multiplanar support for ReadbackARGBImagePixels to both passthrough
and raster decoder where for multiplanar shared images the client
passes in single mailbox and the plane index to read from the SkImage.
