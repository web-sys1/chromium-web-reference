Use NG painter also when painting table cell backgrounds.

-webkit-background-clip is part of background painting, and, unlike any
other type of background painting operation (?), this will traverse the
subtree. Make sure that we don't force legacy object traversal fallback
unconditionally, as that would fail DCHECKs and also behave incorrectly
when block-fragmented (which currently isn't possible to test, since NG
block fragmentation isn't implemented for tables yet).
