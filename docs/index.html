[Autofill] Enable AutofillPopupCheckHtmlFormPopupOverlap on ToT

This CL changes the default state of
kAutofillPopupCheckHtmlFormPopupOverlap to enabled on top-of-tree.

When enabled, Autofill popups perform an additional check to detect if
they are obscured by top-level HTML form popups (e.g. a color picker).
If so, the Autofill popup won't be shown.

The corresponding entry in fieldtrial_testing_config.json is also
removed.
