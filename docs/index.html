Don't show the restore setting notificaiton after reboot from crash.

The restore setting notification is shown when the restore button of the
full restore notification is clicked.

When the system reboots from a crash, whatever the full restore setting
is, the notification after a crash is always shown. So for the crash
restore notification, we don't need to show the restore setting
notification, which might confuse users.

Modify test cases to verify the restore setting notification is not
shown after clicking the restore button of the crash restore
notification.
