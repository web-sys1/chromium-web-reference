content: Compress navigation screenshots using ETC1 compression.

Use ETC1 compression to compress cached screenshots for navigation
entries on Android. The compression is setup as follows:

1. When the screenshot is created, a task is posted to a low priority
background worker to run the compression algorithm. The compressed
bitmap is persisted on NavigationEntryScreenshot.

2. The cache is notified of the change in compression size if the
screenshot is being managed by the cache. If not, we defer discarding
the bitmap until the screenshot is placed back into the cache.
