[Autofill] Manage list of previewed fields.

With this CL, Autofill maintains a list of currently-previewed elements.
Previously, the field that was last queried for Autofill suggestions was
used to extract the fields that needed to be cleared.
SetSuggestedValue() is called for every element in that list,
irrespective of its type.
