Turn docs/rust.md into a landing page for people interested in Rust

This adds a new rust doc aimed at increasing discoverability of
the Rust team and helping people who are already interested in
Rust. It's the readme for the docs/rust folder so that it renders
automatically in codesearch.

The doc contains a lot of information from docs/rust.md. I intend to
remove docs/rust.md in a followup CL, because there are several references
to it throughout the codebase.

The primary goals of this document are to tell people how to
contact us, and to head off common concerns that might get in the
way (e.g. questioning if they're "allowed" to use Rust).

This CL also has a few opportunistic link fixes, since I realized I'd
been using the wrong syntax for gitiles.
