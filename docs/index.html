CCA: Use photo treatment for continuous video snapshot

For photo taking, we only allow previous photo to finish then can take
next photo, while for video snapshotting, we will queue all user
triggered snapshot request and execute all of them one by one. This CL
consolidate these two behavior to ensure next user triggered snapshot
can only happen when previous one complete.
