media/V4L2: Audit V4L2Device static functions

- V4L2Device::V4L2PixFmtToDrmFormat() is only used by GenericV4L2Device,
this CL moves it there as file static function.
- V4L2Device::V4L2BufferTypeToString() is used only in v4l2_device.cc,
so this CL moves it to file-static function in an unnamed namespace.
- Other V4L2Device::*ToString() are moved to a new v4l2_utils.cc file
(imitating vaapi_utils.cc file) to simplify v4l2_device.cc which is
already pretty large.

No new functionality intended.
