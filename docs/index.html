NaCl on ARM64: Use new toolchain for nacl_helper

Actually build nacl_helper for CrOS builds, using the new
nacl_helper_arm32 toolchain.
Before, we would just build for Linux when targeting CrOS.
This also builds nacl_helper_bootstrap with a new specialized toolchain.

Disable the use of some libraries (cras, nss, libdrm, libsync) that are
neither present nor needed for this build.

In build/config/nacl/host_toolchain.gni, identify CrOS by the default
toolchain instead of is_chromeos to be slightly more precise.
