multipaste: Increase ControlVHistogramRecorder's event handler priority

This CL raises ControlVHistogramRecorder's priority as a pre-target
event handler from the default to the accessibility level. This change
is safe because the event handler observes without modifying, and it is
necessary because when the ClipboardHistoryLongpress feature is enabled,
the AssistiveSuggester stops propagating Ctrl+V release evnets before
this histogram recorder hears about them.
