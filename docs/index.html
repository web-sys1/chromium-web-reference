[iOS] Changing rows in Safe Browsing UI to TableViewInfoButtonItems

To follow the icon image size and margin standards put in place within
the settings UI, we made the decision to change the rows in this page
from SettingsImageDetailTextItem to TableViewInfoButtonItem.

We have also added functionality to hide the info button if needed to
conform to the UI changes needed for Enhanced Safe Browsing. Changes
have also been made in TableViewInfoButtonItem to account for
constraint changes when an info button is hidden. These changes can be
seen in the video below
