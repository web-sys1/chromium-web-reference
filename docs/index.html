[DebugVisualizers] Use MostDerivedType for NGPhysicalFragment

It turned out that `<ExpandItem>` does not work for array
items, but `<MostDerivedType>` does. For that reason, This
patch changes `NGPhysicalFragment` to use `<MostDerivedType>`.

Also:
* Changed to use enum members instead of int.
* Added `<Intrinsic>` to get the casted type to compare with
  the enum members to avoid lengthy expression appearing
  multiple times.
