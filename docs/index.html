AX: Improve bounds calculation efficiency

This patch adds an outline rect collector with two subclasses: one
that gathers the rects, and one that just unions them.

The latter is quicker when there are a lot of rects to gather. For AX,
the code used to do OutlineRects followed by union, now it just uses
the union outline collector.

On an extremely heavy (gerrit) page with ~7000 line diff visible and
AX enabled, the runtime of one of the bounds calls goes from about
1.9sec to about 150ms.
