Add hardware manufacturer filtering to field trial testing config.

This change introduces the ability to filter field trial experiments in
the testing config based on the device's hardware manufacturer. It adds
`hardware_manufacturers` and `exclude_hardware_manufacturers` fields to
the schema, updates the utility functions to process these filters, and
includes unit tests for the new filtering logic.
