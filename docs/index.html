Web Applications: Replace IsChromeOs() for Lacros

We replace IsChromeOs() with IsChromeOsDataRequired()
and AreAppsLocallyInstalledByDefault().

Web apps on both Lacros and Ash have chromeos_data populated,
as both Lacros and Ash browsers run on Chrome OS.

With both Lacros and (pre-Lacros) Ash, web apps are locally
installed on sync.
