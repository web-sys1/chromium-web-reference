[Autofill] Set max width for address suggestion instead of the dropdown

This is a follow up to https://crrev.com/c/3168859 that reverts some of
the added code added and on top does the following:

When the feature AutofillTypeSpecificPopupWidth is enabled:
1- Only the address dropdown has a maximum width (the credit card
dropdown width will not be affected by this experiment).
2- The Dropdown is in principle allowed to shrink to a minimum width of
zero.
3- When the text in the address dropdown entry is longer than the
available space that is capped by the maximum width, the text is elided.

Updated specs:
