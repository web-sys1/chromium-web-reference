[@counter-style] Convert a renderer hang into CHECK hit during UA counter style map initialization

Currently in some cases, we somehow fail to resolve a 'fallback' reference
for a UA counter style, and hence, enter an infinite recursion that tries
to look for 'decimal' from the UA counter style map while the UA counter
style map is being constructed.

This patch turns the infinite recursion into a CHECK hit so that the bug
no longer cause bot timeouts, and will be easier to debug.
