Add option for SWAs to opt in/out of system colors.

crrev/c/3812729 updates SWAs to pull theme and bg color from the OS by
default. However some apps such as terminal should opt out as they
have their own theming system. This CL creates a SWA delegate method
so terminal can communicate this intent.

This value is unused as of now. crrev/c/3812729 will add the code to
read and respect this value.
