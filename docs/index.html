Remove obsolete histograms for `Permissions.FCM.Revocation`

`Permissions.FCM.Revocation` histograms were introduced to track
performance of the Notifications permission revocation feature on
Android. The feature works as intended and its performance is tracked
via other histograms like `Notifications.AppNotificationStatus` and
`PushMessaging.DeliveryStatus`.
