Use GlicUiEmbedder::Show instead of Attach

This CL removes the `GlicUiEmbedder::Attach` method and updates the
callsite to use `GlicUiEmbedder::Show` directly.

The `Attach` method was redundant as it only called `Show`. This change
removes the unnecessary forwarding method, simplifying the interface and
improving clarity.
