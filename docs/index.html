Add Apps.UsageTimeV2 metrics.

Add Apps.UsageTimeV2 metrics to record the app running times in the
last five minutes on Chrome OS.

Chrome apps and web apps are recorded as:
ChromeAppTab: Chrome apps opened in tabs
ChromeAppWindow: Chrome apps opened in standalone windows
WebAppTab: Web apps opened in tabs
WebAppWindow: Web apps opened in standalone windows
