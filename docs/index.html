Update Screen AI presandbox hook.

We don't need to assume Screen AI library does not have presandbox init
function as the component updater is enforced to download the library
with version above 112.0 which includes the required function (through
ScreenAIInstallState::VerifyLibraryVersion).

So the TODO to remove checking for existence of the PresandboxInit
function is removed, but it is still checked as we always need to ensure
the library has the expected functions.
