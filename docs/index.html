Launch access code casting dialog from the system tray if no sinks exist

For users that have the AccessCodeCastEnabled policy set, they should be
able to cast even if there are no cast sinks available for them at the
moment. This change ensures that if that policy is set, the cast menu
item is shown even with no sinks, and if the button is clicked, to launch
the access Code casting dialog directly.
