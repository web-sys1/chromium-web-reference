Use network::IsUrlPotentiallyTrustworthy for protocol upgrade

Currently, this relies on SecurityOrigin::IsPotentiallyTrustworthy()
which essentially implements [1] by calling the corresponding network
API. Since all callers check that url's protocol is "http" or "ws",
this is just equivalent to [2] and can be done by calling
network::IsUrlPotentiallyTrustworthy() directly.

[1] https://w3c.github.io/webappsec-secure-contexts/#is-origin-trustworthy
[2] https://w3c.github.io/webappsec-secure-contexts/#is-url-trustworthy
