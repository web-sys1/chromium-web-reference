CCA: Change spoken messages when a QR code is scanned under Scan mode

This CL changes spoken messages when a QR code is scanned under Scan
mode.
To be more concise and consistent and keep the accessibility, the
strings were updated.
When a url is detected, `Link detected. ${LINK}` became `Open ${LINK},
since the focus is on the url link button. If a user moves the focus to
the copy button, it will say `Copy link address ${LINK}` not `Copy link
address`.
When a string is detected, `Text detected. Copy button` became `Copy
text ${TEXT}` since the focus is on the copy button.
When a wifi is detected, it will read `Join ${WIFI} Network`.
