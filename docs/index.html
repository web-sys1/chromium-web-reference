[UNO] Fix account storage support in openToExternalBookmark

No behavior change unless ReplaceSyncPromosWithSignInPromos is enabled.

Before this CL, BookmarksCoordinator.openToExternalBookmark would
always open local mobile bookmarks, even if the bookmark was added to
the account storage. After this change, it should correctly open
the folder that contains the most recently added bookmark for the
requested URL.
