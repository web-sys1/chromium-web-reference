Implement provideNamedData() and android.consumeNamedDataAsArrayBuffer()

provideNamedData(String name, byte[] bytes) can be called from Java and android.consumeNamedDataAsArrayBuffer(name) can be called from JS. android.consumeNamedDataAsArrayBuffer(name) returns a promise to the ArrayBuffer. The byte array is pushed into a socket pair and the other end is sent over IPC to the receiving process. During consumptions the fd is read and converted to an ArrayBuffer and the result promise is resolved.
This would allow clients to pass in data without worrying about the
Binder transaction limits. Once the ArrayBuffer is received, it can be
used for WASM compilation.
This CL also adds support for returning promises. evaluateJavascript() will wait for the returned promise to resolve and return the resolved value of the promise as a string.
