[PartitionAlloc] Distinguish GigaCage space exhaustion from true OOMs.

The GigaCage should be as small as possible, but no smaller. For the
direct-mapped pool, a crash can happen because of fragmentation in the
pool. Distinguish between self-inflicted OOMs and "true" OOMs.

This was previously possible by looking at the crashing line, but this
makes the signature stable.
