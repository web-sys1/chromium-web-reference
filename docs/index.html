[Passwords] Add metrics for add credential manually

This CL introduces histogram
PasswordManager.AddCredentialFromSettings.UserAction
that will be recorded the following user actions when a new credential
is added manually via settings in Desktop and iOS:
1. Credential is created.
2. Add credential dialog is opened.
3. Add credential dialog is closed.
4. Duplicated credential exists.
5. Duplicated credential is viewed.
