Settings: Password View: A11y: Focus on the row after navigating back

Before this CL, when user navigated back to the passwords main page,
the focus was lost.

With this CL, the focus is set to the icon button of the row. If the
credential is deleted from the view page or if the view
page is opened manually (not from passwords main page), the focus is
not set.

Change is manually tested in Linux.
