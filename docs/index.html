Add Lacros support for tts.pause and tts.resume.

This cl implements the following:
1. Add lacros support for tts.pause and ttsEngine.onPause,
as well as the support for web api speechSynthesis.pause.
2. Add lacros support for tts.resume and ttsEngine.onResume,
as well as the support for web api speechSynthesis.resume.
3. Lacros browser tests for pausing/resuming speech synthesis with
both tts extension and ttsEngine extension running in lacros.
