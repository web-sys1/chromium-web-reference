[Journeys] Fix appearance of TabLayout toggle

Three issues are addressed here:
* The elevation values were accidentally backwards in
  section_tab_background
* Calling selectTab() from onTabSelected is racy. Things are much more
  reliable if we set the correct tab when swapping back.
* Having the TabLayout share a background with the tabs themselves is
  unreliable because of caching; the bounds are not reliably reset.
