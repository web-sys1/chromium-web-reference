[WebAuthn] Attach OAuth token to cloud enclave WebSocket connection

This retrieves an OAuth token that will allow connecting to a production
cloud enclave authenticator and puts it in the Authorization header.

This uses an EventStream to pass the token to the authenticator. It will
only be called once but since fetching the token is asynchronous this
lets us incorporate the wait into the authenticator's state machine.
