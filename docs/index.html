CCA: Call LoadDocumentScanner in DocumentScannerServiceClient

In CCA, there are 2 clients using DocumentScannerServiceClient. One is
CameraAppHelperImpl, and the other is CameraAppDeviceImpl. CCA uses
CameraAppHelperImpl to check the document scanner status and uses
CameraAppDeviceImpl to do the ml method calls via mojo. Since
CheckDocumentModeReadiness will never be called by CameraAppDeviceImpl,
LoadDocumentScanner needs to be executed in constructor of
DocumentScannerServiceClient to support ML service like corner detection
in CCA.

Bug: b:301200244, b:299837974, b:304651471
Test: Manually on DUT and corner detection on CCA.
