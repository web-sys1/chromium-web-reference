AutofillAI: Add departure date to Flight Reservation.

This CL introduces a new attribute in Flight Reservation entity and a
new `FieldType`, `FLIGHT_RESERVATION_DEPARTURE_DATE`, for use within the
Autofill AI framework. This field should never be predicted by the
server. It is added so that disambiguation order of the entity can
include it.
