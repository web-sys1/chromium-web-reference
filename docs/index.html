capture_mode_polish: Fix curesor while adjusting region

While adjusting the capture region, the cursor used
to change to kPointer when it hovers over the area
where the capture bar is. The capture bar is fully
transparent in this case, and the cursor should
remain at the same type used for adjusting the
region at the current point.

This CL fixes the issue and adds a test.
