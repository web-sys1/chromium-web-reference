[Contextual Tasks] More composebox server positioning fixes

positioning of the composebox:
 - Move the forced styles onto `contextual-tasks-composebox` instead of `cr-composebox`
 - Reset the bounds on hard navigation
 - Update the server if there is a mismatch in the server reported height
 - Adjust the composebox positioning to be relative to the viewport instead of the <webview>
