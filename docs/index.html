Android: Passes TabModel the activity type

A new method TabModel::activity_type() replaces |isTabbedActivity|
previously passed to it. It can be used by native layer to tell
which activity a given tab is hosted in without having to cast it
to TabImpl for the method |isCustomTab|.

This helps remove dependencies on TabImpl which is planned to be
package-private.
