Remove file check from open panels

This check for file-ness was added to file dialogs in 2014 with
489a9778c0ab6b2c7566b2cbf87d59bd687ebf67 to prevent selecting packages
for upload. It was a "stopgap" until the packages could be
automatically zipped. That zipping actually happened later in 2014
with 22de64a3653834acad2580ded12857385e3d4d65.

This file check is therefore obsolete and should be removed.
