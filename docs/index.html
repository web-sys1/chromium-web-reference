DeclarativeNetRequest: Add API to get disabled static rule ids

Add an API to get the disabled rule ids of a static ruleset:
- DeclarativeNetRequest.getDisabledRuleIds()

The API returns the list of static rules in the given ruleset that are
currently disabled.

The callback is called with a list of rule ids that correspond to the
disabled rules in the ruleset.
