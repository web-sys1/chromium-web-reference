[Extensions] Enable extensions to fetch file scheme URLs.

In Manifest V2 with DOM-based background pages, extensions XHR to the
resource. In Manifest V3, with service worker-based background contexts,
XHR does not exist and it is largely replaced by fetch(). This CL
enables extensions to fetch file scheme URLS, to be able to access file
scheme resources.
