Convert home button to WebUI

This CL migrates the Chrome Home button to WebUI, fully aligning with
native interaction logic. It implements PointerEvent listeners to
support left-click, middle-click, and modifier keys (Ctrl/Shift) for
flexible navigation (e.g., opening in new tabs/windows). Additionally,
it replicates the native 500ms left-click long-press timer for context
menus.
