[RDS] Unset Request Desktop Site per-tab setting if needed

Unset RDS per-tab setting when toggling from the app menu to the same
layout as the global setting.

Use enum instead of boolean for the per-tab setting to capture all three
states: Default/Mobile/Desktop.

When toggling "Request Desktop Site" from the app menu:
- If toggling to the same layout as the global setting, clear per-tab
setting for the tab;
- Otherwise, set per-tab setting for the tab.
