Check for "Location in Work Profile" setting disabled

This change adds a check for whether location services have been
specifically disabled due to a user restriction rather than for the
device as a whole. This fixes an issue where the Geolocation and Web
Bluetooth APIs mysteriously don't return any results if this setting is
turned off.
