Make CoepReflection independent of AnonymousIframe.

Mozilla and Webkit expressed positive feedback around exposing the
current's document COEP value to itself. However w3ctag expressed the
opposite opinion: https://github.com/w3ctag/design-reviews/issues/742
So it might not ship as it was planned.

AnonymousIframe was originally planned to depend on CoepReflection, so
that libraries could use it. This patch removes the dependency so that
AnonymousIframe can launch independently.

If possible, libraries may be able to use one of the alternatives
described below:
https://github.com/w3ctag/design-reviews/issues/742#issuecomment-1196473390
