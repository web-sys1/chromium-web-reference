Using D3D11ImageRepresentation for shared handle video frame

This CL adds D3D11ImageRepresentation::ScopedAccess when
creating video frame with shared handle and fence support.
Decoder calls BeginAccess before decoding into frame and
EndAccess when other component can access the frame.
