[iOS] Target 10.15 when building WebKit for macOS on macOS 11

The builder bots are now on macOS 11. In order to continue building
with Xcode 11.4 (which is needed since the fuzzer bots are still on
10.15.7), target 10.15.7 explicitly.

This is similar to workaround that we used last year, to build on
10.15 but target 10.14.
