Ensure that forms have the same size in suggestion generation

BAM::GetSkipStatues assumes the passede FormData and FormStructure have
the same size. This is guaranteed in BAM::FillOrPreviewDataModelForm but
not in BAM::GetProfileSuggestions. This CL fixes this by falling back to
the old behavior that marks all fields as relevant or suggestion if the
real statuses cannot be computed.
