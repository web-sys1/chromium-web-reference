PasswordScriptsFetcher: Pass Version into constructor

During the lifetime of PasswordScriptsFetcher[Impl], the Chrome version
(of course) remains constant. So there is no need to pass the version
into each request.
This CL instead queries the version once, in the factory, and passes it
into PasswordScriptsFetcher on construction. This way, users don't have
to pass it in anymore.
