[ComponentUpdater] Add java EmbeddedComponentLoader

Add the java part of the EmbeddedComponentLoader which connects to
the ComponentProviderService under WebView's UID to fetch components
files and then pass them to the native loader classes.

Add an integration test as part of android_webview
instrumentation tests since it needs loading native, launching a
mock service ... etc. Which will be a huge code duplication to
have under components/component_updater/android.

Both embedded WebView and WebLayer can use this class to load components
files during startup.
