Send USB events/telemetry to PERIPHERAL_EVENTS destination.

We, currently sending events to the event report queue and telemetry to the
telemetry report queue. This change creates a new report queue
which uses the PERIPHERAL_EVENTS destination and pushes USB events and
telemetry to that queue.
