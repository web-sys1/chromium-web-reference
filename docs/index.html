[Passwords] Refactor Password(Form)Manager::PresaveGeneratedPassword

This CL deletes an iOS-specific overrides of PresaveGeneratedPassword.

Cross-platform PasswordFormManager::PresaveGeneratedPassword is to
be used on all platforms.
The iOS specific override was:
1) updating *mutable_observed_form() - this is a simple operation
making sense on all platforms, so I moved it to cross-platform code
2) setting generation element and type - this was a redundancy, as
it's called by SharedPasswordController::generatePasswordForFormId
on all password generations anyway, so I removed it

PasswordManager::OnPresaveGeneratedPassword is to be used instead of
iOS-only PasswordManager::PresaveGeneratedPassword. It can be further
simplified in follow-up CLs.

This CL also deletes two PasswordManagerTests:
1) SavingGeneratedPasswordOnIOS - not needed since there is no iOS-
only method override anymore. Tests the same steps as
PasswordManagerTest.GeneratedPasswordFormSubmitEmptyStore
1) PasswordNoLongerGeneratedOnIOS - not needed since there is no iOS-
only method override anymore. Tests the same steps (even fewer) as
PasswordManagerTest.PasswordGenerationPresavePassword.
