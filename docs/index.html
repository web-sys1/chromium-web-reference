Add Page.getUrl() to Android WebView API

This CL implements `Page.getUrl()` to return the last committed URL of
the main document. This exposs the `getUrl()` method on the `Page`
object in the Android WebView Support Library, enabling embedders to
retrieve the URL associated with a specific page instance.
