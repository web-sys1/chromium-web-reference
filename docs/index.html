Fix overflowed toolbar PinnedActionToolbarButton pop out behavior.

Buttons with anchored bubbles, when triggered from the overflow menu,
weren't appearing in the toolbar and the bubble was anchoring to
nothing. This was due to an incorrect update in flex priority. Also
inconsistently the button would not appear due to the anchor being
changes by some logic in ToolbarView to prevent anchoring to a
non-visible location bar. This has been updated to any
PinnedActionToolbarButton is use regardless of visibility.
