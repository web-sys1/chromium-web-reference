Fix dangling pointers in callback in WindowsSpellChecker

`metrics` was passed and used after free in a callback.
It didn't crash, because the method where it was used could be made static, and `this` was not really dereferenced.

This patch removes the argument, and turn the member function into a
static one. This will prevent future folks doing UAF by using `this`.
