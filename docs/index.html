[Passwords Grouping] API to refreshes cache in AffiliationService

This change adds a new function UpdateAffiliationsAndBranding() to
AffiliationService which will clear existing cache and request new
branding/grouping information from the affiliation service. This is
useful when PasswordsGrouping feature was just enabled and cache
doesn't contain any grouping information.

Callback will be invoke immediately if there is no internet connection
and after the response is received. It doesn't matter whether the
request was successful or not.
