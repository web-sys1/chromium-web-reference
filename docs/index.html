RubyLB: Create LogicalLineItems for ruby annotations

  LineInfo to a LogicalLineItems with a new InlineLayoutStateStack
  and a new LogicalLineBuilder.

* Its result is stored in a LogicalRubyColumn.

* The top-level line creation determines inline positions of all
  LogicalRubyColumns.
