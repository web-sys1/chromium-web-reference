Fixed crash in OpenSavedTabGroupHelper.

Previously we use a std::move operation with in the function calls (OpenSavedTabGroup and OpenSavedTabGroupHelper to preserve the lifetime of the saved tab group being passed. This was not a good idea as sometimes when attempting to reopen saved groups using the buttons, the reference would be lost.

Accordingly, we check if the saved group is not null before we do any operations with it, in addition to checking if we are able to obtain a keyed service for saved tab groups to further ensure we are safe to query the id of our saved group.
