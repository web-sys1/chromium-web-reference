Stops running profile migration due to lacros dir not existing.

Previously even if profile migration is marked as completed i.e.
`IsProfileMigrationCompletedForUser()` returning true, profile migration
was run if lacros data dir did not exist. This is to run profile
migration in case if lacros was disabled and re-enabled again. By simply
calling `ClearProfileMigrationCompletedForUser()` if lacros is disabled,
there is no longer a need to check if the data dir exists or not.
