[iOS FRE] Add basic "More" button support

Adds support for a "More" button in the shared FRE screen view
controller for screens where scrolling to the bottom is mandatory. When
this behavior is enabled, the primary button is replaced with a "More"
button until the user scrolls to the end of the content, at which point
the primary button's text is reset to the intended string. Also, until
the content has been scrolled to the end, tapping the "More" button will
scroll the content instead of notifying the delegate of a primary button
tap.

The final, localized text and adding an Arrow icon to the More button
will be done in follow-up CLs.

Also updates the fake FRE pages in the Showcase project to use this new
logic.
