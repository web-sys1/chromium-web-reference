[Fuchsia] Signal last release fences in ~OutputPresenterFuchsia

ImagePipe may not signal the release fences submitted in the last
Present() call, but ~ExternalVkImageBacking() waits for the
corresponding Vulkan semaphores. This was causing the timeout for
semaphore wait to expire. This change updates OutputPresenterFuchsia to
signal the semaphore explicitly on destruction.
