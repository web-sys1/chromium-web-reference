[Structured Metrics] Notify recorder when full hardware class is
available.

Because hardware class is required for most structured metrics,
upload of events will be delayed until hardware class has been loaded in Ash Chrome and will available to be attached to the payload.

This will be monitored by comparing the total event count with the
feature flag enabled vs disabled.

See b/224883594 and https://chromium-review.googlesource.com/c/chromium/src/+/3609102 for more details.
