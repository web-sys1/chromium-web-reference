Require at least 3 different field types for local heuristics

Today, local heuristics produce field classifications if at least 3
fields get fillable field types assigned. With the new experiment
AutofillMin3FieldTypesForLocalHeuristics, at least three *different*
field types need to get assigned to fields. We hypothesize that this
reduces the number of false positives.
