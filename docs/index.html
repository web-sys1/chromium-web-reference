Oilpan library: Support V8-only debug mode

V8 supports a mode where it gets compiled with debugging features in
an otherwise Release build for Chromium. This results in various
debugging information being present when Chromium thinks it's running
in release mode.

gn arg:
  v8_enable_debugging_features = true
