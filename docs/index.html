Transform collapsed spaces to U+3000 for text-transform: full-width

Per CSS Text Level 3, the full-width transformation happens after
whitespace collapsing. This means that when this property is active
and whitespace collapses to a single space, it should become
Ideographic Space (U+3000).

Previously, InlineItemsBuilder always inserted U+0020 when collapsing
whitespace, ignoring the value of text-transform. This CL ensures
that the correct type of space is used for the current style.
