Repair accidental early exit in clean layout callback processing

The intention of this code is to not attempt a callback for an object
that is no longer valid. However, the early return is accidental.
Instead, the loop should continue and process the next callback.

This resulted in a double echo of characters in Google Sheets, because
a role change was never processed. It likely resulted in a lot of other
flaky behavior as well.
