WebGPU: Add UKM for GPUAdapter requestAdapterInfo() method

This CL updates kV8GPUAdapter_RequestAdapterInfo_Method use counter to
be measured via UKM.  This will be helpful to investigate some specific
origins that use the GPUAdapter requestAdapterInfo() method that we're
going to deprecate in favor of the GPUAdapter info attribute.

Its usage is gated by GPU requestAdapter() method which is small (0.3%)
so it should be okay to add it to UKM:
https://chromestatus.com/metrics/feature/timeline/popularity/3888
