Key management file changes.

I've modified a few of the key rotation files to include the network
delegate as a parameter. I've also moved the exponential back off
code to the windows network delegate and updated some tests to
accommodate for this change. I also modified setup mains call to
rotate to include the win network delegate as a parameter. Because the
linux network delegate takes in the url loader factory as a parameter
and the windows network delegate does not. There is no longer a need for
a generalized network factory. We can instead directly pass the network
delegate in our calls to create the rotation manager.
