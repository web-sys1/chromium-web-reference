[Profile customization bubble] Handle invalid input and input deletion

This CL is a speculative fix for a range of crashers, making
ProfileCustomizationBubbleSyncController more robust.

Firstly, it deals with invalid input parameters and returns early.
Secondly, it listens to deletion of the profile and of the anchor view
and skips the bubble.
