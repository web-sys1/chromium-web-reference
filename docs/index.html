Remove shared preference file helpers

The GVR tests that were recently removed were the only consumers of
shared preference files that allowed for setting/restoring some device
specific state during tests. Now that those tests and the shared
preference files themselves have been removed, we can remove the
utilities that supported them.

This was done as a separate change because there is one cleanup where
a script was leveraging a helper from the shared preference files
that was actually a bit more generic of functionality, however, the
bulk of this functionality is actually a no-op on Python 3. As such the
previous behavior was inlined.
