PageLoadMetrics: Add utilities for PLMOs and Prerender2

To make PLMOs support prerendering, we have to consider time origins of
metrics. For example, the time origin of First Contentful Paint is
navigation start. Such kind of metrics can be arbitrary large as the
activation delayed. These metrics should represent how long the users
"wait" for those events. Prerendering shorten these waits. So, we should
correct those metrics.

For the details, see
https://docs.google.com/document/d/1Mg44THmNtUsK2NiT6sze65LdcdGBRaZrkh9ko1-ez-0 .

As a first step, this CL adds utilities for prerendering to
PageLoadMetricsObserverDelegate and page_load_metrics_util.
