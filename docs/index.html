Modify GetIconKey to use the non-mojom App struct.

Modify IconLoader::GetIconKey to return a null icon key as the default
implementation, then the GetIconKey implementation in other test code
can be removed.

Modify ExtensionAppsBase::OnExtensionLoaded to call CreateApp to create
the non-mojom App struct, because during the init phase, only
OnExtensionLoaded is called to add Chrome apps to the non mojom App
struct to AppRegistryCache, so we need to build the whole App struct,
including the icon key to load icon.

Modify NoteTakingHelperTest to set the user for the second profile to
prevent creating the arc app pref, which may cause crash during the
shut down phase due to the default app added asynchronously after
shutting down.
