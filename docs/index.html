[rust] Rearrange stdlib source code layout

gn can inform IDEs such VSCode to find Rust code. That includes
the Rust stdlib, which doesn't currently get found by IDEs in Chromium.

This is the first in a series of two CLs which make that work.
In this CL, we rearrange the layout of stdlib source code (taken from
the Android Rust toolchain) to correspond to that of Fuchsia's Rust
toolchain, which is what gn assumes.

A subsequent CL will absorb the revised CIPD package and adjust
gn rules too.
