Move IOS AutofillShowTypePredictions check

This change moves the check for the AutofillShowTypePredictions
option from fillFormDataPredictions to SendTypePredictionsToRenderer
to ensure consistency with the non-IOS implementation.

This allows for an earlier exit and improves code maintainability
across platforms.

The change also removes a redundant const in base::span argument.
