Add WebDxFeature UseCounter to readable-byte-streams

The Web Platform Telemetry project aims to provide better usage data to
teams working on the WebPlatform in Chrome. This change adds a new
kReadableByteStreams WebDXFeature use counter to account for the use
of readable-byte-streams feature
(https://webstatus.dev/features/readable-byte-streams) as part of this
initiative.
