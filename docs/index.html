[GridNG] Subgrid GetComputedStyle and filtering

This change completes basic lined-name support for subgrids. There are
two main parts of this change: updating to serialization of subgrid
named grid lines in accordance with the spec, and a second piece that
filters out named lines from the parent that are out of the subgrid's
range. Line names are not inherited for the non-subgridded axis. There
wasn't an existing test validating this condition, so I added one.

The basic concept is that all named lines are relative to index 0
internally. This is how non-subgrids currently work. During placement,
line names are resolved relative to index 0, and then get shifted by
the subgrid's position.

This change regresses `parent-repeat-auto-fit-001.html` and
`parent-repeat-auto-fit-002.html` because the reference files now
render properly. I will fix these in a follow-up, by handling
`auto_repeat_named_grid_lines`.
