Support verification of multiple fingerprints

Currently only one app fingerprint is verified against a list of fingerprints defined in the `assetlinks.json` file, however, apps can have several fingerprints. This CL adds the option to verify several fingerprints against this list, the check only passes if all app fingerprints are in the list of `sha256_cert_fingerprints` in the `assetlinks.json` file.

This CL does not change the current behavior.
