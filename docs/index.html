[launcher] Add details text for search and assistant results.

This test adds two details texts:
1. Assistant results are appended with "Google Assistant".
2. Omnibox search results are appended with the default search engine
   name, e.g. "Google Search".

Extra info about a11y:
If an accessible name is not set, it will be computed by concatenating
the title and details fields. Previously, Omnibox search results set the
accessible name field to append the search engine name.
- For rich entities, continue to do this.
- For non-rich-entities, this is not required anymore since the
  accessible name is in the details field now.
