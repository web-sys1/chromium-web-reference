[Editing] Fix backspace to delete emoji tag sequences as a single unit

The BackspaceStateMachine did not handle emoji tag sequences (used for
subdivision flags like England, Scotland, etc). Instead of deleting the
entire flag with one backspace, users had to press backspace multiple
times to delete the invisible tag characters individually.

Emoji tag sequences are defined in UTS #51 [1] as:
  emoji_tag_sequence := tag_base tag_spec tag_end
Where:
  - tag_base: emoji_character or emoji_modifier_sequence or emoji_presentation_sequence
  - tag_spec: one or more tag characters (U+E0020â€“U+E007E)
  - tag_end: U+E007F CANCEL TAG

These sequences should be treated as a single unit for editing purposes.

This CL adds two new states to the BackspaceStateMachine:
  - kBeforeTagTerm: entered when CANCEL TAG (U+E007F) is seen
  - kInTagSequence: accumulates tag sequence characters

[1] https://unicode.org/reports/tr51/#def_emoji_tag_sequence
