wm_mode: Build skeleton of the feature

This is the first CL of WM Mode; an experimental feature
that belongs to the CrOS Labs effort. This feature aims
at making it easier for users to layout, resize, and
position their windows especially using touch gestures
without having to be very precise.

This CL just builds the skeleton:
- Adds the feature flag,
- Adds the controller class with basic logic, and
- Adds a unit tests file.

Later CLs will build up on this one.

A demo of a prototype of this feature can be seen at:
https://crbug.com/1348416#c6.
