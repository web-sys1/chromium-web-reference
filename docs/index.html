[media] Create VideoToolboxVideoDecoder

VideoToolboxVideoDecoder is a VideoDecoder implemented using
VideoToolbox platform decoding. It is intended to replace
VTVideoDecodeAccelerator which was written using VDA interfaces.

VideoToolboxVideoDecoder is composed of five major pieces:
  - VideoToolboxVideoDecoder implements the VideoDecoder interface and
    handles frame ordering.
  - H264Decoder parses bitstreams.
  - VideoToolboxH264Accelerator packages bitstreams for VideoToolbox.
  - VideoToolboxDecompressionInterface configures and operates the
    VideoToolbox platform APIs.
  - VideoToolboxFrameConverter produces VideoFrame objects backed by
    VideoToolbox IOSurfaces.

Enable with --enable-features=VideoToolboxVideoDecoder,UseMultiPlaneFormatForHardwareVideo,PassthroughYuvRgbConversion
