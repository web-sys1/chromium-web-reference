[fuchsia] Include //fuchsia/cipd in all build configurations

Although there is no need for CIPD packages except in official builds on
official build bots, there is no reason that the GN rules shouldn't
build in unofficial builds. Thus, remove this difference between
official and unofficial builds. Since CQ/CI build "all" in an unofficial
configuration, this will also provide additional coverage on CQ/CI and
help prevent changes from landing and later failing on official bots as
happened in https://crbug.com/1309864.
