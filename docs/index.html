[WebLayer] Create WebLayerSafeBrowsingBlockingPageFactory

This CL moves creation of //weblayer's SafeBrowsingBlockingPage from
SafeBrowsingBlockingPage::CreateBlockingPage() to a new
WebLayerSafeBrowsingBlockingPageFactory class. This will facilitate
changing //weblayer to use the //components-level
SafeBrowsingBlockingPage class rather than its own custom fork. At that
point WebLayerSafeBrowsingBlockingPageFactory will become a subclass
of safe_browsing::SafeBrowsingBlockingPageFactory.

As part of this change, we move metrics logging into the
SafeBrowsingBlockingPage constructor for consistency with that logging
being in the constructor of the //components-level
SafeBrowsingBlockingPage.

There are no functional changes.
