[iOS] Fix Avatar resizing

The resizing was taking into account the scale of the image but with
a projected size in pt (which is considering the scale of the screen).
This CL changes the resize utils to always take the size of the screen
into account.

It also unifies all the places where the avatars were used to make sure
they use the ResizedAvatarCache, allowing for a more consistent
user experience.

The changes to the resize util impacts the Thumbnail generator used
when sending a thumbnail of the page to the share util. This side
effect shouldn't impact the user experience.
