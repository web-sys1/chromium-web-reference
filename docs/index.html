[FLEDGE] Add sendReports argument to deprecatedURNtoURL().

We recently made FLEDGE only send reports when the result of an auction
is loaded into a frame, in the case an auction succeeds, which better
aligns with the spec.

This has caused issues for some folks experimenting with FLEDGE who are
not actually using it to show ads yet, but are testing its performance
and functionality, so want to be able to send reports, but not actually
use the ads.

The new argument, added to a temporary method for early
experimentation, provides this functionality without adding much
complexity.
