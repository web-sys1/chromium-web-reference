[BroadcastChannel] Use long timeout for cross-partition tests

It seems like the cross-partition tests are right at the threshold
of the default WPT test timeout, causing them to timeout
intermittently (especially when the test machine is under load).
This plus some other changes seems to fix the flakiness in Chrome
and in Firefox when testing locally, so let's see if this change
alone is sufficient for now.
