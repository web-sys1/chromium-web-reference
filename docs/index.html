Reland: "PWA on Desktop Windows: Implement uninstallation via OS settings."

This reverts https://chromium-review.googlesource.com/c/chromium/src/+/2459309

Reason for revert:
Test, All/SystemWebAppManagerUpgradeBrowserTest.Upgrade/WebApps, failed often.
It was indeed test bug that it mistakenly called install operation without
waiting uninstall operation. so it ends up with early signal, which
try to launch non-existent web app. The original CL
exposed this error due to increased uninstall time. The testing bug
was fixed now https://chromium-review.googlesource.com/c/chromium/src/+/2503374.

note. there is no additional change except merge conflict fix
in the chrome_features.*, app_launcher_handlers.cc.
