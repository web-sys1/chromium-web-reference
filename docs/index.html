apps_service: Prevent crostini_tracker from claiming borealis windows

Borealis windows were incorrectly using the crostini_tracker during
shelf id registration, which became problematic after
crrev.com/c/3439335 as the tracker would statefully modify the borealis
window's associated IDs to crostini ones.

We will prevent the crostini tracker from accidentally using borealis
windows. This may introduce regressions if the old behaviour of tracking
borealis windows via crostini was actually being used in some other way,
but we will only see that after landing this CL.
