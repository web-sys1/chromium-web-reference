Downloads: Small cleanups to facilitate Typescript migration.

 - Remove usage of addSingletonGetter(), replace with explicit
   getInstance/setInstance. Fix newly discovered type errors, as a
   result of the improved type coverage.
 - Leverage EventTracker helper to track listeners in manager.js,
   remove custom tracking code.
 - Use a Map in icon_loader.js, instead of a plain Object, which is
   not a parameterizable type in TypeScript.

These are done because it makes migrating to TypeScript a bit easier,
but they are good improvements anyway.
