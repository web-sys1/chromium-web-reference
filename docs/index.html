Merge CreateBackendAndDatabase and CreateBackendAndDatabaseAllowFail

Both CreateBackendAndDatabase and CreateBackendAndDatabaseAllowFail had
the same implementation, the only difference being that one DCHECKed on
error while the other did nothing.

The two functions can be merged by returning a bool and letting the
caller decide how to handle the error. This will be useful in a
follow-up CL, where a test will expect a database to fail opening.
