[launcher] Adjust deduplication between Open Tab and Drive results.

All Omnibox Drive results were being filtered in order to deduplicate
them with results from the Drive search provider. This CL updates the
filter so that Open Tab results don't get removed.

The CL also changes GetDriveId slightly, because the document prefix can
vary (e.g. "/document/d/", "/presentation/d/", "/spreadsheets/d/"). Then
extract GetDriveId into a shared util file to avoid duplication.
