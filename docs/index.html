Introduce more specific WiFi security settings

Introduce more fine-grained security options for the WiFi:
- WPA2
- WPA2-WPA3
- WPA3
- WPA-Enterprise
- WPA-WPA2-Enterprise
- WPA2-Enterprise
- WPA2-WPA3-Enterprise
- WPA3-Enterprise
- WPA3-Enterprise_192

Note that there are no explicit modes for all possible WPA versions
(e.g.  WPA-WPA2-WPA3). If network configuration indeed requires that
kind of setting, user can still use legacy value WPA-PSK for that.
