Bump minimum version supported for App Container to RS5.

This is because it is not possible to set process mitigations
on an App Container process until RS5.

Since the number of users in the range RS2 <-> RS4 is small
it's not worth trying to fix or support this configuration.

Better to bump the minimum version to RS5 and all the APIs
are fully supported.
