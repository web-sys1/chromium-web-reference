[ImprovedDownloadWarnings] Add "Turn on Safe Browsing" link

This adds a link in the download bubble security dialog offering to
open the settings page so the user can turn on Safe Browsing. This link
is displayed when the user has no Safe Browsing protections but is
able to turn it on. The link goes to the chrome://settings/security page
and also displays the IPH for Enhanced Safe Browsing.

The link also goes along with special warning text on the security
dialog and button/status text calling the file "unverified".
