Improve owner user detection

Implement InstallAttributes::IsFirstSignIn() which is similar
to InstallAttributes::IsDeviceLocked(), but is less time sensitive.
It remembers whether the device was not locked "earlier" and can
be used to more reliably detect the first user.

Use InstallAttributes::IsFirstSignIn() to decide whether a user
should establish the ownership (the first user on a consumer
device should).

Save into local state who is the owner in case Chrome crashes or
gets interrupted before the ownership is fully established.
This will be used in a follow up CL that will make Chrome
responsible for owner key generation.

Implement UserManager::RecordOwner() and
UserManager::GetOwnerEmail() helper methods.
