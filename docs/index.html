[infra][gpu] Switch macOS GPU testing in CQ to arm64 (Apple M2)

The M2 builder + tester have been very green for the past week [0, 1],
so:
a. Remove `mac-gpu-rel` (x64) from CQ.
b. Have `mac-rel` mirror the CI builders.
c. Enable gardening for the CI builders.

The same CL needs to do (b) and (c) because gardened builders must have
a try mirror, and CQ bots must mirror a gardened CI builder.

`mac-gpu-rel` will be removed in a follow-up CL so that this CL itself
can pass CQ.

During peak hours, `mac-gpu-rel` currently uses 200 Intel bots [2] (with
similar numbers for AMD). [3] shows there is enough M2 capacity (200+
always ready M2 bots) to handle the new traffic.
