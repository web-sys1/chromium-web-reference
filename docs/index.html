Move Cocoa scrollbar state behind MacScrollbarImpl

Move the BlinkScrollbarPainterDelegate and BlinkScrollbarObserver to be
owned by a MacScrollbarImpl.

Previously, BlinkScrollbarPainterDelegate was owned by
MacScrollbarAnimatorImpl, and BlinkScrollbarObserver was owned by a
map accessed in ScrollbarThemeMac.

This change is created with the goal of having a minimal diff. The next
patch will remove all remaining Cocoa code from scrollbar_theme_mac.mm.
After that, all Cocoa will be put into a separate implementation.
