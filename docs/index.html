[iOS][Thumb Strip] Add more simple eg tests for thumb strip

This CL adds three new tests:
1. Tests that swiping down twice transitions the thumb strip to the
revealed state. This means the toolbar is no longer visible and the web
content is hidden behind a hider view with the current location text.
2. Tests that scrolling in the web content causes the thumb strip to
open and close.
3. Tests that scrolling on the NTP causes the thumb strip to open and
close.
