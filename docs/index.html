PreloadingDevTools: Store error of SpeculationRules and enrich CDP events

This CL

- ensures that SpeculationRuleSet::Parse returns a rule set. If an
  error occurred, it is stored in the rule set.
- makes empty rule set with parse error to be stored in
  DocumentSpeculationRules (for a CDP event).
- adds error information to Preload.ruleSetUpdated event.
- makes SpeculantionRuleSet::Parse reports error if a rule set for a key
  (e.g. ruleSet["prefetch"]) is not undefined nor an array.
