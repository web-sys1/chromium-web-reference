[UseCounter] Use std::bitset for ukm_features_recorded

Unlike `features_recorded_` and `main_frame_features_recorded_`,
ukm_features_recorded_ uses std::set instead of std::bitset to
track recorded features. Consider that the upper bound of enum type
`WebFeature` is well defined (Currently Under 4000), std::bitset should
outperform std::set.

This CL replaces std::set with std::bitset on
`UseCounterPageLoadMetricsObserver::ukm_features_recorded_`.
