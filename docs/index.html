Enables visual overflow DCHECK when objects are invalidated

crrev.com/c/2825972 added a DCHECK to ensure visual overflow
is computed before its first use. This patch enables it for
when existing objects are invalidated.

The invalidations are added in the major code paths that lead
to |PaintLayer::SetNeedsVisualOverflowRecalc|.
