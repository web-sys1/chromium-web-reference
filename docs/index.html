Add AXAnnotationsSubpage to ChromeOS settings.

We add a new subpage to ChromeOS settings which is currently an
identical copy of the subpage in browser settings. The reason we do
not share the subpage is because there may be features going forward
which launch on browser but not on ChromeOS, so we want to be flexible
to that.

Also, clean up some PdfOcr settings, which was removed in an
earlier effort (crrev.com/c/5600000).

In a follow-up, will rename AxAnnotationsSubpage to
AxAnnotationsSection in ash and browser.
