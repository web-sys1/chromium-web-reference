Migrate Value::Set*Path() calls in /components/error_page/common.

Value::SetPath() and Value::SetStringPath() are deprecated in favor
of Value::Dict::SetByDottedPath(). This set of CLs migrats all
remaining consumers of those two APIs.

Some callsites aren't passing in paths with dots, so switch those to
Value::Dict::Set(). In some CLs, also migrate a couple other nearby
deprecated Value methods.

This CL was uploaded by git cl split.
