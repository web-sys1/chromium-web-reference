[ios] Replace SnapshotGeneratorDelegate in Swift

This CL replaces SnapshotGeneratorDelegate written in Objective-C with
Swift. WebState, used as a parameter of methods in
SnapshotGeneratorDelegate, is not usable in Swift so this CL
introduces WebStateSnapshotInfo to wrap a WebState. It will provide the
information needed for taking snapshots in a follow-up CL.
