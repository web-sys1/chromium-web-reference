Add FilesCleanupHandler

This CL adds the files cleanup handler for the chrome.login.endSession
API.

Files under the profile's My Files directory will be deleted. Directories
other than the Downloads directory will be deleted (only the Downloads
directory itself is not deleted, everything under it will be deleted).

If there is an error in deleting any file or directory, or an error in
enumerating all files and directories, the cleanup handler will return
an error to the CleanupManager.
