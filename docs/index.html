Fix crash when back gesture is performed after tab modal dialog is dismissed

Steps to reproduce:
1. trigger a tab modal dialog
2. navigate back to dismiss that
3. navigate back again: observe a crash

This is because when ModalDialogManagerObserver#onDialogDismissed is
triggered, Presenter#getCurrentModal does not return null. Then the
TabModalLifetimerHandler assumes by mistake that there is still a
visible dialog on the screen.
