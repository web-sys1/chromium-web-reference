[fuchsia] Disconnect element.Controllers when last window closes

User of element::Manager expects the Controller to be closed when the
element they started finishes. As chrome can starts new browser window
independantly of the element API, there is no element/view mapping.
Instead, as a workaround, this CL close all connected controller when
the last window is closed.
