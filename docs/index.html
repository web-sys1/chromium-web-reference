wm: Fix lacros crash when drag to unmaximize.

We had a DCHECK to check for restore bounds if a window is snapped or
maximized. This is not always the case in lacros currently as restore
bounds are not always available, even for regular unmaximize. See
https://bugs.chromium.org/p/chromium/issues/detail?id=1238928.

Once the above has been fixed drag to unmaximize should work after
reopening a closed maximized lacros browser, so we can revert this
change then. But for now introduce this to stop the DCHECK getting hit.
