Add metric Startup.Android.Cold.TimeToFirstVisibleContent2

As TimeToFirstNavigationCommit2.Tabbed looks like an improvement over
the version without the "2", make the TimeToFirstVisibleContent2 that is
based on it.

The current metric is roughly:

  TimeToFirstVisibleContent = min(
      TimeToFirstNavigationCommit.Tabbed,
      Browser.PaintPreview.TabbedPlayer.TimeToFirstBitmap)

The newly introduced metric is as follows:

  TimeToFirstVisibleContent2 = min(
      TimeToFirstNavigationCommit2.Tabbed,
      Browser.PaintPreview.TabbedPlayer.TimeToFirstBitmap)
