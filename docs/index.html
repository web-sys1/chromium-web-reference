media/RegisteredMailboxFrameConverter: migrate to tracking_token

|VideoFrame::oopvd_mailbox_| is currently being used as an identifier
for looking up an original FrameResource by the ChromeOS-specific
RegisteredMailboxFrameConverter. However, use of bare mailboxes has been
deprecated.

This migrates the RegisteredMailboxFrameResource's identifier from
|VideoFrame::oopvd_mailbox_| to |VideoFrameMetadata::tracking_token|.
This unblocks removal of |VideoFrame::oopvd_mailbox_|.

MailboxFrameRegistry and RegisteredMailboxFrameConverter will be renamed
in a follow-up CL, separate for reviewability.
