[PartitionAlloc] Disable freelist hardening on perf bots.

To investigate whether freelist hardening has a performance impact,
temporarily disable all of it on perf bots, which use the following GN
args:
- Official build
- No PGO

This is narrowly targeted, as freelist hardening is useful to detect
unintentional Use-after-Free in the codebase, so we don't want to ship
even a canary build without it.
