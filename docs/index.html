base: Remove SupportsUserData's SequenceChecker in release builds.

This CL switches base::SupportsUserData from direct use of
base::SequenceChecker to the SEQUENCE_CHECKER() macro and its
associates. This way, the SequenceChcker member doesn't take up memory
inside SupportsUserData instances in release builds.

This CL also uses GUARDED_BY_CONTEXT() instead of equivalent
documentation in a comment.
