Window Placement: Delay showing exit bubble content

This CL delays showing exit bubble content until fullscreen action has
finished. Especially for Mac platforms as the fullscreen action is
asynchronous.

After these changes are landed:
1. the user won't notice the exit bubble content during a fullscreen
intermediate state, which is more in line with expectations.
2. the exit bubble won't disappear until the fullscreen action is
finished and will keep for some time to be noticed.
