PA: Add address space diagram

This CL trivially adds a model of how address space (specifically pools
in 64-bit environments) is divvied up. Access is mediated by the
singular `AddressPoolManager` class. Pools are large contiguous runs of
address space, but the carve-outs given to each root (to each partition)
in the form of super pages may not be so.
