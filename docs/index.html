[Sensors] Add tests for fusion sensors

There has not been dedicated unit test for
PlatformSensorFusionAlgorithm::IsReadingSignificantlyDifferent() at all.
This change adds a test for it.

Also there has not been test which would call
PlatformSensorFusion::OnSensorReadingChanged() and fusion sensors
IsReadingSignificantlyDifferent() function at same time. This change
introduces a test which creates fusion sensor using two low-level
sensors.
