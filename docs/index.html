Exo: Change reactive frame submission to do eager submission.

When a BeginFrame request is received, within the send deadline, respond
immediately as soon as:
 * a frame is available; and
 * there is no in-flight frame between Exo and Ash GPU process.

This change improves dropped frame rate, see
https://docs.google.com/document/d/1Rfq8T1yOBFhnFGGxiQdBWstxAL-x0IfLG7i45rR222o/edit?resourcekey=0-fjo5hr_HkVs-MQaaIjOSLg#heading=h.zdlavartcrrw
