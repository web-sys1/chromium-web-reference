webauthn: Add new request dialog illustrations

This adds new header illustrations for the WebAuthn request dialog.
While the old illustrations were sized to fill the entire top section of
the dialog, the new ones are smaller and supposed to be centered in the
top section after applying regular padding. Hence,
AuthenticatorRequestSheetView needs to account for the difference when
creating the view.

Screenshots:
https://docs.google.com/presentation/d/1c4ZBf23FFGJWS3ez7awpKoDQYqLoQulmR4pqF0f8oYc/edit

All new behavior is flag guarded by
`device::kWebAuthnNewDiscoverableCredentialsUi` (default off). No
behavior changes if the flag is disabled.
