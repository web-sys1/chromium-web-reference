Fix g_allocator_buffer's alignment.

This CL fixes "Chrome 88.0.4300.3(pulse canary) crashes on launch".
g_allocator_buffer is used as an instance of PartitionRoot.
So its alignment must be base::kAlignment(i.e.sizeof(std::max_align_t))-aligned.
If not aligned, we will see "Signal 7 (SIGBUS), code 1 (BUS_ADRALN)" when accessing the PartitionRoot instance.
