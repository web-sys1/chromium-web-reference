[accessibility] Fix various leaks in AXPlatformNodeWin unit tests

- Use Reset rather than Release to free resources on ScopedSafearray.
- Replace calls to CoTaskMemFree with use of ScopedCoMem to hold the
  memory returned by various calls that are documented as returning
  memory allocated by CoTaskMemAlloc.
- Use ScopedVariant::Receive() to pass an instance as an out argument to
  various calls rather than ScopedVariant::AsInput and ensure that
  instances are Reset as needed before subsequent use.
- Introduce ReleasePointers() to call IUnknown::Release on all elements
  in an array of pointers returned from various calls.
- Use ComPtr as appropriate to hold the return value of various calls.
- Use ComPtr to manage the lifetime of instances created by tests.
