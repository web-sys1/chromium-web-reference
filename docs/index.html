Add 'AllFrames' to GetRenderDocumentLevelNameForTestParams()

This makes tests running with the "AllFrames" level of RenderDocument
use "RDAllFrames" instead of "RDSubframe" in the test param name. This
was taken out of crrev.com/c/4202277 because the test name changes
causes some bots mistakenly think existing flaky tests are new flaky
tests introduced by the CL, even though they're the same test, just
with a different name due to the param name change.
