[iOS] Log omnibox device switcher result

Add logging for device switcher result.
DeviceSwitcherResult are checked at every startup and when the users
leaves NTP on FRE. The result determines the default omnibox position.

When result are not available, log the status:
- NotNewUser: device switcher result are not checked.
- Unavailable: device switcher result are not available.
When the result are available, log the resulting omnibox position:
- TopOmnibox: user has the top omnibox by default
- BottomOmnibox: user has the bottom omnibox by default.
