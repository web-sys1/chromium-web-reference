Generate software-backed RSA keys in chaps

Add support for generating software-backed RSA keys in chaps
to PlatformKeysService.

This is implemented by calling PKCS#11 functions on libchaps.so
directly. A few layers were introduced to support this:
- ChapsSlotSession:
  Loads libchaps.so, resolves relevant PKCS#11 functions on it, and
  opens/closes a read-write PKCS#11 session for the slot. Allows calling
  C_GenerateKey, C_GetAttributeValue and C_SetAttributeValue.
- ChapsSlotSessionFactory:
  Creates ChapsSlotSessions. Useful so ChapsSlotSession creation can be
  replaced with a fake for unit tests.
- ChapsUtil:
  Implements generating a key pair by performing the actual PKCS#11
  calls. Also sets CKA_ID of the generated key pair to the value NSS
  would set.
  Uses ChapsSlotSession{Factory} to perform the chaps PKCS#11 calls.
