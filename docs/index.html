This CL makes changes as follows:
- Stop observing the prerender activation earlier to avoid receiving
  uninterested navigation events. Before this CL, those events are
  ignored, so this doesn't change the behavior.
- Return early in WillProcessResponse() if the page is not in
  prerendering mode.
- Replace a non-null pointer on function arguments with a const
  reference as this is recommended by the style guide.
- Update comments about the same-origin checks.
