Create TabContextMenuDelegate to include context menu actions for tabs.

Created TabContextMenuDelegate to replace RecentTabsContextMenuDelegate
for recent tabs and to be used for the actions in the TabGrid context
menu as as well.
In general the actions needed for all tabs are similar. So the plan is to
add all actions that can be tab related in the protocol and they can be
implemented by whoever want to create context menu for tab items.

Updated the usage in recent tab files to use the new delegate.
