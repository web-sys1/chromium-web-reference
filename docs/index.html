CrOS OOBE: Remove a few more CSS Mixins usages.

--oobe-subheader-font and --oobe-modal-dialog-header-font are only used
in a single place each, and are inlined instead.

CSS Mixins are deprecated (never actually launched) and rely on the
shadycss polyfil (baked into Polymer) to work. They can be replaced by
native CSS variables instead.

Moreover, as part of optimizing Chromium's copy of Polymer, exploring
fully removing the shadycss polyfil, which requires all CSS mixins
usages to be eliminated.
