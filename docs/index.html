build: Remove -Wno-module-file-config-mismatch

The -Wno-module-file-config-mismatch flag was intended to suppress
module build errors related to incompatible configurations. However, it
resulted in cryptic failures without adequate explanations. Given that
we've now disabled module builds for targets using incompatible flags
(such as RTTI or exceptions), this suppression flag is no longer needed.
