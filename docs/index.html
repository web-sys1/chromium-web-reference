[Sensors] Add rounding & significance test for ALS

Threshold and rounding check added to test framework. Currently used
only with ALS.

Threshold check tests the requirements specified in here:
https://w3c.github.io/ambient-light/#ambient-light-threshold-check-algorithm

Rounding check tests the requirements specified in here:
https://w3c.github.io/ambient-light/#ambient-light-reading-quantization-algorithm

Removed requirement that readings and expectedReadings must be same
length as above changes require that they are different length.
