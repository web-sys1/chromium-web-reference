Add new scroll latency metrics to rendering benchmark

Old Event.Latency scroll latency metrics are deprecated and replaced by
new EventLatency metrics. This CL adds the new ones to the rendering
benchmark. In a few weeks we would need to update alert config files to
file regression bugs based on the new metrics. Then we can remove
alerting based on the old metrics and remove the old metrics from
rendering benchmark.
