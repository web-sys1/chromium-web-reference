[PCCT] startActivityForResult opens PCCT with black background instead of transparent activity

In the case the developer has requested that a PCCT is opened we need to validate that the package name can be retrieved either from the intent or over the service connection. This check happens as part of the CustomTabIntentDataProvider#configureIntentForResizableCustomTab method. When this is called we need to ensure that the EXTRA_CALLING_ACTIVITY_PACKAGE intent extra has already been set. Moving the setting of this intent before #createCustomTabActivityIntent is called.
