[UniqueFontSelector] Stateful memory pressure behavior behind feature

In https://crrev.com/c/7412889, UniqueFontSelector was changed to modify
the cache max size depending on the memory pressure level.

This change is a partial revert that puts that stateful behavior behind
the kStatefulMemoryPressure feature flag, to ensure that we can test
out the behavioral change with an experiment.
