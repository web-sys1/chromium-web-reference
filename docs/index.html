dbus: move :common to a separate package

Currently lacros can directly pull in the dbus dependencies it needs by
adding them into chromeos/lacros/dbus/BUILD.gn. But it doesn't want to
pull in the whole //chromeos/dbus (and some packages can't be built in
lacros currently). However, some dbus clients depend on
//chromeos/dbus:common, like u2f_client, and as a result lacros can't
directly pull it in as dependency because then the whole //chromeos/dbus
would have to be built. Separate the :common component out as a separate
package so that the dependency can be separated. Now lacros can use
those dbus clients that need :common.

Except moving the files and the portion of chromeos/dbus/BUILD.gn to the
common directory, other changes are all search and replaces.
