[cleanup] Store Vectors of indices instead of counts.

This patch changes the logic within DistributeExcessBlockSizeToRows
to store Vectors of row indices instead of recording if we have any
rows of a particular type.

This has the advantage that we only iterate over the rows of a
particular type rather than all rows, then filtering on a particular
type.

There should be no beahviour change.
