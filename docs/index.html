[dPWA] Disable policy install of web apps on guest profiles

This CL adds logic that limits policy installs to:
1. ChromeOS guest sessions.
2. On profiles where web applications are allowed.

On other use-cases (like incognito/guest profiles), policy installs
are skipped.
