[DownloadBubble] Change download notifications in exclusive_access

Exclusive access shows notifications to inform users.
If a download notification overrides an existing notification,
the notification also informs how users can exit the bubble.
To inform about the override, for each different type of regular notification, there would need to be a corresponding download
notification type.

This CL makes changes in exclusive_access to better track download
notifications, and what kind of existing exclusive_access
notifications they override. Instead of using enums, this CL adds
fields in exclusive_access_bubble to better categorize notifications.
