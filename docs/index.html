Support array's count for @MAC-SCRIPT

This CL introduces a new array count attribute for accessibility script
tests, so that one can verify the size of an array retrieved via AX
attributes. This is tested for AXChildren and will be used for testing
AX attributes needed for MathML (AXMathPostscripts, AXMathPrescripts,
AXMathRootRadicand...) [1] [2].

[1] https://w3c.github.io/mathml-aam/
[2] https://github.com/w3c/mathml-aam/issues/4
