Disable MutationObserver-document.html on all platforms

The MutationObserver-document.html web test is currently disabled on
Linux and Mac12 only. It was observed in Edge that the test is flaky on
other platforms as well. The error pointed out in this comment
(https://bugs.chromium.org/p/chromium/issues/detail?id=1342718#c4) is
what we are specifically seeing. This change expands the disablement of
the test to all platforms. The bug number in TestExpectations is also
being updated since the originally linked bug has been marked duplicate.
