ozone/wayland: remove eglImageFlushExternalEXT.

This flush was inherited from ozone/drm's GbmSurfaceless when
GbmSurfacelessWayland class was implemented. Moreover, this is for
older devices. And according to dnicoara@, newer devices may not
even advertise it.

Running lacros with that call removed doesn't seem to result in any
graphics problems (no tearing, etc).

Thus, try to remove this flush and see if it results in any problems
for users.
