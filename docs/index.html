[iOS][Omnibox] Scroll carousel on item focus

Scroll carousel when focusing a carousel control with the keyboard or
VoiceOver.

When the control becomes focused with VoiceOver and when it becomes
selected with keyboard, send a message to its delegate.
The delegate `OmniboxPopupCarouselCell` updates its scroll view to make
the control visible .

Additionally fix the array of accessibility element to return only visible elements. The visible elements are retrieved in the getter and
invalidated when hiding a control or setting up controls.
