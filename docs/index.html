[ios] Set default `configs` value for swift_source_set

As `swift_source_set` expands to a `source_set`, it gets the
same default, however, it is not possible to use `configs +=`
to add new configs since the default is applied after the
template has been expanded, while the `configs +=` happens
before.

Call `set_defaults` for swift_source_set with the same value
as is used for source_set in //build/BUILDCONFIG.gn.
