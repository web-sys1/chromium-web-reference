[Media Router] Update DIAL metrics.

1.  Replace MediaRouter.Dial.ParsingError with
MediaRouter.Dial.DeviceDescriptionParsingResult, and record the new
metric for both successful parses and errors.

Currently, the histogram does not count successful parses, so there is
no baseline to compute error rates.

2. Remove MediaRouter.Dial.ParseMessage.  The number of of errors
recorded is not meaningful enough to make it worth recording -
currently 0.0052% error rate.
