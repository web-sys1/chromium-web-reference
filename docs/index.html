Cleaned up not-used method for filling on page load IOS.

PasswordFormHelper's fillPasswordForm:inFrame:completionHandler: used to be a method which, when was called, always returned early because the argument formData.wait_for_username is always true now.

This means that the method was never used for actual filling, except for tests.

This CL's deletes the method and makes all the other necessary changes.

In some tests, formData.wait_for_username was false, and the method was used for filling. For those tests I had to find alternatives for filling the forms.
