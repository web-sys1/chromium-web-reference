[Telemetry] Add event service interface definition

This CL adds a TelemetryEventService interface definition
that is going to be part of crosapi.

a copy of cros_healthd's exception type that they plan to
use in a multitude of interfaces.

This is part of a bigger effort to add an event interface
to the Telemetry Extension. For more context visit
go/design-sketch-event-api.
