[dPWA] Allow FakeOsIntegrationManager to also Synchronize()

This CL introduces the ability for FakeOsIntegrationManager to run
Synchronize() with the new sub managers by skipping the Execute step
and directly writing to the DB.

This also modifies the OsIntegrationSynchronizeCommandTest to use
the FakeOsIntegrationManager to verify that all sub managers are being
properly written to the DB.
