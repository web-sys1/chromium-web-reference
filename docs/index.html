Default enable Emoji via GMSCore

Experiment analysis shows that stability (crashes/hangs) seem not
significantly different between control and enabled population,
while at the same time emoji cluster brokenness is reduced.
87% of recorded values show zero emoji brokenness in a page
in the control group vs. 93.6% of recorded values in the enabled
group, which means ~7 percent points more of the
recorded page views exhibit correct emoji display.
