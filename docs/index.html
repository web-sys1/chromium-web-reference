media: Remove Widevine test license server infrastructure

This CL removes the Widevine Portable License Server infrastructure.
This infrastructure was Linux-only and had become difficult to
maintain while providing little value.

Encrypted media tests that previously relied on this server for
successful Widevine playback now expect failure or are skipped,
matching the behavior on other platforms and ensuring that general
Widevine EME flow and UKM recording can still be verified.

Changes include:
- Removing src/third_party/widevine/test/license_server from DEPS
  and .gitmodules.
- Deleting the generic and Widevine-specific test license server
  implementation in chrome/browser/media/.
- Updating chrome/test/BUILD.gn and third_party/widevine/cdm/BUILD.gn
  to remove references to the deleted files and targets.
- Updating chrome/browser/media/encrypted_media_browsertest.cc to
  remove usage of the test license server while preserving general
  Widevine tests with adjusted expectations.
