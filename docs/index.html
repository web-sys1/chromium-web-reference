Fixed several potential dangling pointer issues.

If a child view has `SetNotifyEnterExitOnChild` set and the pointer
hovers over its child elements, and we then move the child view's parent
view or an ancestor view using a keyboard shortcut or other means (e.g.,
moving it to another widget), it will cause a crash because the original
child view no longer belongs to the current RootView.
