[Nearby Share] Adjusting Nearby Share settings dialog heading levels

This CL adjusts some <h> tags so to correct the header tag hierarchy
on in some Nearby Share dialogs.


The dialog boxes all began their header tag hierarchy with an <h1> tag
which meant that the full subpage when compiled had 2 <h1> tags.

The hierarchies in these pages were adjusted to start at <h2> so that
the fully compiled page has only one <h1> tag, the "Settings" title.
