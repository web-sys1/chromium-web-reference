input method settings: Remove unnecessary GetPrefsForEngineId helper

It contained an "default dictionary" that was being passed which was a
quirk of the CreateSettingsFromPref function from where it was called,
making the function not generalizable.

The generalizable part of the function can be written in just one line
(prefs.GetDict(kLanguageInputMethod...).FindDict(engine_id)) with the
only caveat being that it returns a pointer that could be nullptr rather
than a const reference.

Hence removing that helper that is only used once would make the code
easier to follow.
