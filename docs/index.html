Add a temporary header for each file moving to base/task/

A multitude of task-related headers currently live in the general
base/ folder. To improve the organisation of the files in the base
library, those files will be moved to base/task/.

Those temporary headers, whose purpose is to redirect the include to
the real location of the header, will facilitate the mass-migration of
all the references to the moved files.
