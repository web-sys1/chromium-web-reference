[iOS] Show blue dot on NTP tools menu button

This CL fixes an issue where the blue dot was not appearing on the tools
menu button (new in NextIA) on the NTP.

More specifically, this CL:
* Adds a `setNTPBlueDotVisible` command to the NewTabPageCommands
  protocol
* Extracts the blue dot drawing and masking logic from `ToolbarButton`
  into a shared `blue_dot_util`.
* Replaces the generic tools menu button in `NewTabPageHeaderView` with
  a new `NTPToolsMenuButton` that uses the shared utility to properly
  render the blue dot and its background mask.
