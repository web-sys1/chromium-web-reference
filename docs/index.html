Update min framerate in applyConstraints()

The min capture frame rate can be specified in the call to
applyConstraints, however, the capture format does not include
the min frame rate.

This CL fixes a bug where the min frame rate was not updated when
applyConstraints() was called if the other parameters remained
unchanged.
