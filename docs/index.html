Move kPdfMimeType to //components/pdf/common/constants.h

Move the PDF mime type constant from //chrome/common/pdf_util.h to
//components/pdf/common/constants.h. A future change in //extensions
will need this constant and cannot have a dependency on //chrome
(https://crrev.com/c/5072998). Instead, have a dependency on
//components/pdf.

Remove the enable_pdf assertion for the constants build target, as some
existing files may not have PDF enabled, but will still need the
constant (e.g. Android doesn't have PDF enabled, but
//chrome/browser/download/android/download_controller.cc uses the
constant).
