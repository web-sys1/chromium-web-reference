Add SharedImageTestBase

state needed to run shared image tests. Also put some test helper
functions into SharedImageTestBase to avoid duplication. Slight changes
to Vulkan dependencies are needed since SharedImageTestBase will compile
Vulkan code if ENABLE_VULKAN is true.

Use the new test base with a few existing tests to make sure it's
suitable as well.
