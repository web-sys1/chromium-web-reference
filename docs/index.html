[lacros shelf] Handle tab drag in BrowserStatusMonitor

Redirect tab remove/insert events when a tab is dragged to a separate
OnTabMoved callback. For now just delegate to OnTabInserted, as it
handles both insertion and reparenting. This will help handling tab move
event separately, simplify the insertion case later, and also validate
that the tab lifecycle event order is correct.
