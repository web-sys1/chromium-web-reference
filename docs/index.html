[Extensions c2s] Move HasActiveTabAndCanAccess() to //extensions

HasActiveTabAndCanAccess() is used to determine an extensions site
access and site interaction. We want to compute this values in the
//extensions layers, since permissions are not specific to //chrome.
Thus, to move these site permissions computations we have to move
from //chrome to //extensions:

1. kDisableExtensionsFileAccessCheck switch
2. AllowFileAccess() util
and finally
3. HasActiveTabAndCanAccess() from SitePermissionsHelper to
PermissionsManager

Next CL will move the site permission computations.
