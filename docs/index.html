[mathml] Add fuzz target for OpenTypeMathSupport

This CL adds a new fuzz target [1] for the OpenTypeMathSupport class.
It currently calls each API function but may need future adjustments.
Using the corpus of math fonts from WPT [2] helps fuzzers find existing
paths within OpenTypeMathSupport.

[1] https://chromium.googlesource.com/chromium/src/testing/libfuzzer/+/HEAD/README.md
[2] https://github.com/web-platform-tests/wpt/tree/master/fonts/math
