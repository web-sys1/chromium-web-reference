MTECheckedPtr: Disarm select members

This CL introduces no behavioral change, since MTECheckedPtr is not
enabled for anybody. This CL annotates several `raw_ptr` members to
indicate that they should silently fall back to being no-op `raw_ptr`
_only when MTECheckedPtr is enabled_.

This change affects /chrome/browser/enterprise/reporting.
