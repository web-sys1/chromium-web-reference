[mojo] Move definitions of -test-utils.h symbols into the main .cc file

The definitions are fairly small, and this avoids the overhead of pro-
cessing all the included headers twice, speeding up the build.

This change removes 5.45 GB (2.4%) of compiler input for the 'chrome'
target in a Linux build with default gn args. For a local (non-goma)
build on my machine that translates to a 33 CPU-minutes (1.6%) build
time savings.
