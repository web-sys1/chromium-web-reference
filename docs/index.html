ozone/wayland: send the entire rounded clip bounds

The mask filter for rounded corners must be applied to
the entire layer instead of a single quad. Thus, its
size and position in Chromium compositor looks like
quad1.size + quad2.size and the position of the
mask starts with the position of the quad nearest to 0,0.

Thus, send an entire rounded corners bounds to Wayland
compositor so that it can apply the rounded corners bounds
later when quads are composited together.
