[Extensions] Start extension site permissions management page

This CL adds a placeholder for a page which will allow users to
customize which sites extensions can and cannot run on (these settings
will apply to all extensions and will override extension host
permissions.)

If --enable-features=ExtensionsMenuAccessControl is passed, then
extension site permissions will be displayed in a new page. This CL
only adds the placeholder page and the following navigations (assume
flag is enabled/passed):
 - Navigating to chrome://extensions/sitePermissions goes to the new
   page
   - if the flag is not specified, redirect back to extension list view
 - Clicking the "Site permissions" link in the sidebar
