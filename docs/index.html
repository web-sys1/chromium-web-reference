[memories] Fixes color of history clusters visibility toggle menu item

<iron-icon> uses an <img> tag to display icons provided using the src
attribute. In this case the '--iron-icon-fill-color' CSS var does not
get applied to the svg. For the fill color to be correctly applied to
<iron-icon> the icon needs to be provided as a member of an icon set
using the icon attribute: icon="<iconset>:<icon>"
