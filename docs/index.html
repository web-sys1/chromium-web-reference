Add StreamAttempt and TcpStreamAttempt

This CL introduces:
1. An interface to represent a single TCP-based connection attempt.
2. An implementation of the interface that attempts a TCP handshake.

These classes are similar to the existing ConnectJob and
TransportConnectJob but have less responsibilities. Unlike
ConnectJobs, these classes don't resolve a host themselves. Instead,
these classes take an IPEndPoint to attempt a connection.

Subsequent CLs will add another implementation for TLS handshake.
See the design doc for details [1].
