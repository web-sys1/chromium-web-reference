[Lacros] Add support for chrome.clipboard.onClipboardDataChanged.

This CL makes Lacros extensions API support
chrome.clipboard.onClipboardDataChanged by doing the following:
* Instantiate ClipboardAPI in Lacros.
* Restore ClipboardExtensionApiTest (browser_tests) to the original
  form, and make it available to Lacros.

Prerequisite work for this are crrev.com/c/2826851 and
crrev.com/c/2827110, both of which have landed.

After this CL, chrome.clipboard API should be usable by Lacros. However,
some existing bugs have been revealed during manual testing. These will
be filed and dealt with as follow-ups.
