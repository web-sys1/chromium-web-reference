Add ResourceFormat param to TransferableResource::MakeGpu

As part of Multiplanar SharedImages, TransferableResource would take
SharedImageFormat instead of ResourceFormat param. Ideally we would
like to pass that param as part of MakeGL call, instead of calling
MakeGL and then setting format separately. This change adds the
ResourceFormat as part of MakeGL call and also renames MakeGL to
MakeGpu call for GPU accelerated path (other being MakeSoftware). This
will allow future change with just modifying from ResourceFormat to
SharedImageFormat created via SinglePlane() call.
