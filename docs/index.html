MPArch: Make dialog cancelled only from active frames.

It was possible to cancel dialogs from inactive frames in
DidNavigateAnyFramePostCommit.  This change ensures the cancellation
can only be called from active frames.

Also, it was difficult to see how LoadCommittedDetails::is_main_frame
will be in MPArch.  I have added some comments to explain what the value
will be.  I have also added DCHECK to ensure some functions are called
by active frames.
