[sync] Use modern C++ algorithms in DeviceInfoPrefs

This CL refactors DeviceInfoPrefs to use more modern and expressive
C++ constructs, improving readability, conciseness, and robustness.

- In `IsRecentLocalCacheGuid()`, the range-based for loop is replaced
  with a call to `std::any_of`.
- In `AddLocalCacheGuid()`, the loop for finding and removing an entry
  is replaced with `base::Value::List::EraseIf`. This is more robust
  as it correctly handles potential duplicate entries.
