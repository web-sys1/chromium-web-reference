Re-implement flashing overlay scrollbars on mouse enter

This was implemented in crrev.com/c/627298, but later removed because
the about_flag expired. Re-implemented that with one additional change -
only flash on children scrollbars and avoid flashing on root scrollbars
as moving a mouse back and forth between children and root results in
flashing viewport's scrollbar as well, which doesn't look nice.

This can be enabled via about_flags page or by passing
--enable-features=OverlayScrollbarFlashWhenMouseEnter
