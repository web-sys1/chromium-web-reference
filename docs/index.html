[FilesDarkMode] Make visual signals panel dark mode compliant

The visual signals panel is a multi use UI surface in Files app that
displays progress of file transfers and eventual status. The following
patch updates all the colors where possible, a few notable changes that
are a little outside just color change:

  - Remove the hardcoded color for the success and failure icons in
    icon.html, these are now set by the introduction of a new CSS rule.
  - The dismiss button previously had a box-shadow and border, the new
    spec defines no border and no box-shadow so have removed both of
    these.
  - Removed the .xf-success and .xf-failure classes, these are no longer
    being used anywhere so remove them and the comment as it was causing
    syntax errors.
  - Fix a presubmit warning about not closing single tags with the svg
    circle and path.

The changes here are also trying to isolate the changes to Files app
without updating the global webui components. So where changes are
non-obvious for why they are done, I have left an inline comment.
