Apply theme overlays in ChromeLauncherActivity to fix exception

ChromeLauncherActivity has a @android:style/Theme.NoDisplay theme. This
results in a IllegalArgumentException when CustomTabColorProviderImpl
tries to get the toolbar color (or ?attr/colorSurface) using the context
from ChromeLauncherActivity.

To fix this, we apply the ColorOverlay.ChromiumAndroid theme overlay in
ChromeLauncherActivity, which has colorSurface set.
