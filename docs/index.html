Register Terminal SWA as web app rather than crostini app

When TerminalSSH flag is turned on, terminal app will be published as
AppType::kWeb rather than AppType::kCrostini.

Terminal is somewhat of a hybrid app which is a lot web (it runs in the
browser), but is also strongly related to crostini since it is used to
start and stop VMs, and may require files to be shared in a VM before it
can view them.

Starting with chrome://flags#terminal-ssh, we will register terminal as
a web app so we can take advantage of using shortcuts for "Connect to
Linux" and "Connect to SSH".
