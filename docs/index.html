[CrOS Bluetooth] Fix disconnect connect status message

When navigating from device details page and back, details page state is
reset to disconnected in some cases where device properties does not
change, details page state is not recomputed leading to wrong device
state being shown. This CL makes sure onDeviceChanged is called each
time navigation to device details page occurs so device page state can
be computed.
