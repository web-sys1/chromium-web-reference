Don't build set of listed properties when transition-property:all is used.

We only use the set of listed properties when transition-property:all
was not used.  So if it is used, don't build it, because adding every
CSS property to it is expensive.

This is intended to fix the hashtable operations which show up in
profiles of the NewsSite-Next and NewsSite-Nuxt sub-benchmarks of
speedometer3 as close to 1% of the profile.  Those pages appear to use
transition-property: all extensively.

This builds on the change in
https://crrev.com/f59182460461941eec817852b877fb1408457372
