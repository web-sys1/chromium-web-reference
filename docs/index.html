[Extensions c2s] Display hover card for toolbar action views

Displays a hover card when mouse hovers a toolbar action view. If the
mouse moves between toolbar actions, the hover card transitions between
them. Hides the hover card when the mouse is moved outside of the toolbar
action views and on any other event.

Toolbar action hover card controller is based on tab hover card
controller. For now we are copying functionality, with a future goal of
having a base hover card.

Toolbar action hover cards are under a feature flag.
