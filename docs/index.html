Fix compare of uninitialized union in MediaQueryExpValue

This is a regression from moving all media query related classes
to Oilpan. WTF containers were allowed to compare MediaQueryExpValues
with memcmp, which is not correct, since it contains a potentially
uninitialized union.

We also don't really need to allow memcmp comparisons, so use
WTF_ALLOW_MOVE_AND_INIT_WITH_MEM_FUNCTIONS instead.
