[Editing][Serialization] Sanitizing html with @font-face must not crash

The staging document which is used for sanitizing html markup uses a
EmptyLocalFrameClient. Since the client doesn't have a URLLoaderFactory,
the resource loader fails to fetch the font and crashes the renderer.

This CL skips loading remote font resource if the document is for
markup sanitization.

Note that the sanitized markup won't have the resource that is not
fetched.
