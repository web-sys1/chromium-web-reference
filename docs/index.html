Split switch used for both tests and headless mode

The command line switch kOverrideUseSoftwareGLForTests was used for both
tests and headless mode. Since the name was misleading, this cl adds a
new switch called kOverrideUseSoftwareGLForHeadless, and uses it where
appropriate.

With both use cases clearly separate, this CL also re-enables the use of
SwANGLE on Windows and Linux for tests which use the
kOverrideUseSoftwareGLForTests switch.
