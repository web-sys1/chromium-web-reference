[Mullhet] Store password value in PasswordSuggestionDetails.

Suggestion struct should include information about the password
value to fill, the signon realm or Android app name, and the boolean
flag specifying whether the cross-domain consent dialog should be
triggered.

This CL makes a first step towards storing this information in the
suggestion struct by extending the Suggestion::Payload with another
possible value. The PasswordSuggestionDetails struct will be used
to store the information needed to handle the password manual
fallback suggestions.
