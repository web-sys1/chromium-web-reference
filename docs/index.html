Fix updating the display-state and resizable CSS properties

This commit moves updating the display-state and resizable CSS
properties to a slightly later stage. Before the change, CSS properties
were updated at the beginning of UpdateVisualProperties. After the
change, the CSS properties are updated later, when the
ApplyVisualPropertiesSizing finished processing and promises from
window.maximize(), window.setResizable() etc. are resolved.
