Fix menu highlighting in Tutorial

We changed how IPH + Tutorials works, with the IPH continuing the promo
while the Tutorial runs. However, the code in the menu was also
attempting to terminate and continue the IPH, and tying highlighting to
that. So because the two were conflicting, the highlight didn't work.

This CL removes the old management logic in the controller, and simply
highlights the menu item if the promo is still running.
