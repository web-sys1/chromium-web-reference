Terminal logging of chrome://(autofill|password-manager)-internals

This CL:
- Adds two features `k(Autofill|PasswordManager)LoggingToTerminal`
that toggles logging the content of the appropriate internals page to
the terminal.
- Removes a previous command line flag --show-autofill-internals that
had a similar effect but only on tests.
- Adds a TextLogReceiver member to LogRouter that observes the router
depending on the features configuration and the router factory.
- Fixes some typos in touched files.
