[feedback] updated JS to work with WebUI

This is for migrating Feedback app from Chrome Apps.

The chrome.app.window API is being deprecated. Replaced its create(...)
method with window.open(...).

Removed i18n JS call that is no longer needed.

Changed the way how the JS is interacting with backend since the
original background JS is no longer available with WebUI.

In this CL, a dummy FeedbackInfo object is used. A message handler will
be implemented to allow JS to retrieve the object from backend in a
following CL.
