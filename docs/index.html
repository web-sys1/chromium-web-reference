ash/display: Post path lookup for VPD ICC as a task.

Previously, the path lookup for VPD-written ICC files was done
in a blocking scope, causing dcheck builds to crash.

Instead, post it as a task. However, posting the Quirks lookup
from the same sequenced task results in invalid sequences.
Therefore, this change now allows Quirks ICCs to overwrite VPD
ICCs. In practice, this is unlikely to happen, and also Quirks
needs some additional modifications to ensure picking up those
ICCs from Quirks.
