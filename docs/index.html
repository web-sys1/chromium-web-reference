[rts] Pin RTS checkout

Currently the RTS checkout uses "latest" ref. This is dangerous in
general, because if the latest is broken, there is no easy way to revert
it, especially if you are a sheriff.

Pin the RTS checkout. In another CL autorolling via Skia autoroller will
be enabled.

As a side effective, the checkout has to be split by platform (because
different platform have different hashes), and installed into different
directories.
