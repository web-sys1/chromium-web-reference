[ntp] Primary based logo color change

With the upcoming Chrome Refresh 2023 changes, the NTP_LOGO_ALTERNATE
display property seems no longer suitable for determining logo color for
a themed (no background image) NTP. These changes update the logic so
that we check too against the chrome refresh 2023 feature flag(s).
