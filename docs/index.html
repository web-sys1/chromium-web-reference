Add ForTesting method to update permissions policy defaults

Some permission policy defaults depend on the value of feature flags.
This CL ads a method that can be called to recalculate the defaults for
the affected features. This should be used during testing features that
affect permission policy defaults.

This CL also uses this new method to de-flake
AdAuctionServiceImplRestrictedPermissionsPolicyTest.APICallsFromCrossSiteIFrame.
