Simplify FedCmAccountSelectionView (part 10/N)

This CL replaces InitDialogWidget() with CreateDialogWidget().
InitDialogWidget() lazily created the widget, otherwise it updated the
widget position. Some callsites wanted to create the widget, others
wanted to update the position.

With this CL, creation and position updating are separated. Each
callsite has been audited to do the right thing. Widget creation happens
in exactly one place, which is the same place a view creation.
