Switch RenderTest ignore field

Switches the 'ignore' field used by Skia Gold-backed Android RenderTests
to be a regular key instead of included in the optional keys. This is
necessary because a recent change to the Gold backend means that
optional keys cannot be used for ignore rules, which is the entire point
of including this key.
