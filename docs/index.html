weblayer: make feature enabling work with field-trial

Tests should be able to enable features that weblayer explicitly
disables. ConfigureFeaturesIfNotSet() is intended for this. Enabling
features via the command line may also include field-trial information.
This patch makes ConfigureFeaturesIfNotSet() account for that. In
particular, enabling a feature with a field trial is encoded in a single
string with the parts separated  by '<'. This makes parsing deal
with this.
