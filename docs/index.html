webauthn: don't block Sync.

Calling into Java to check device compatibility, and calculating the
identity public key, took a little bit of time and were blocking the UI
thread on Android.

Instead, do them on worker threads and decline to provide
phone-as-a-security-key data until they can completed, the same way that
we currently block for an FCM registration.
