[rust] Switch back from autocxx to cxx

This reverts commit 3309b582b20cd1c146598304df303523cab20701, with
modifications.

For now, Chromium will be using cxx for Rust <-> C++ interop. It's
possible that in the future we will use a scheme to generate Rust
bindings to pre-existing C++ code (such as bindgen, autocxx, or
rs_bindings_from_cc) but at the moment the complexity is unwarranted and
provides a maintenance burden as we keep such tools up to date with
clang.

This is the first in multiple commits - subsequent commits will
remove autocxx itself, but this one has all the fiddly stuff around
.gn files.
