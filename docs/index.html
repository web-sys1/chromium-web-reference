[Lacros] Enable GetChromeOSDeviceName

Migrate devicetype_utils.cc to use the chromeos::GetDeviceType()
provided by chromeos/constants instead of ash/constants.

Both devicetype_utils.cc and the strings used by it used to be only
available on Ash. Change some build files to enable them on Lacros.

This is done so that Lacros can use GetChromeOSDeviceName, which will
be used in some strings.
