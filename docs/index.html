Add a CfM-only flag to disable Chrome App verification for local testing

To support local development of the Chromebox for Meetings (CfM) Chrome
app, a flag is added that disables extension verification. This flag is
only intended for use on unsigned CrOS builds on CfMs via
/etc/chrome_dev.conf. The flag's behavior is gated on
BUILDFLAG(PLATFORM_CFM) to limit it to CfM devices.
