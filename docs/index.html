[Autofill] Fix pre-filled field skipping

Fixes two issues that occur when skipping pre-filled form fields:
- Checks `autofill_field->value` instead of `form.field[i].value` for
emptiness to account for re-fills of fields that were empty on page
load.
- `FieldFillingSkipReason::kValuePrefilled` cannot be reached anymore if
  the field is a select or select list element. This is required to
  make select fields with a `value` on page load be filled at all when
  `features::kAutofillSkipPreFilledFields` is activated
  (see https://source.chromium.org/chromium/chromium/src/+/main:components/autofill/core/browser/browser_autofill_manager.cc;l=591;drc=ca8f62c7f553b35a1b03701945532f4a800610f9).
