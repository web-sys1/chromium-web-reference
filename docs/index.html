[text-decorations] Decorating box support in |TextDecorationInfo|

This patch adds support for the decorating box to
|TextDecorationInfo|.

With this change, some functions of |TextDecorationInfo|
return values of the decorating box, instead of the target
box. The order of functions in the header file was adjusted.

For the code to work properly, |DecoratingBoxes| in the
|NGInlinePaintContext| must be set up properly from the tree
before painting decorations. This part is not landed yet.

This patch has no behavior changes because the feature is
under a runtime flag.
