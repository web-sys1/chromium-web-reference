[Gamepad] Check nullptr IOHIDDeviceCopyMatchingElements

https://chromium-review.googlesource.com/c/chromium/src/+/5118066,
where IOHIDDeviceCopyMatchingElements could return a nullptr during
GamepadDeviceMac::AddButtons. However, although infrequently, the same
issue also occurs during GamepadDeviceMac::AddAxes. This CL implements
the same fix for GamepadDeviceMac::AddAxes calls.
