Remove //printing/backend from //printing's public_deps.

Since //printing headers that are intended to be used publicly do not
include //printing/backend headers, remove this public_deps entry. Then
fix all the missing dependencies on //printing/backend.

As is, the public_deps entry propagates all the way to
//tools/v8_context_snapshot, and leaves developers bewildered as to
how a //printing/backend change can cause v8_context_snapshot generation
to fail.
