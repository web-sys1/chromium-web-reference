extensions: Alternate fix for ExtensionThrottleManager lifetime

Revert the `ObserverList` based fix in https://crrev.com/c/6651042
because `ExtensionURLLoaderThrottle` could be destroyed from other
threads and `ObserverList` is not thread safe and results in memory
corruptions.

This CL creates a `ExtensionThrottleManagerAccess` to provided
synchronized access to a manager instance for its throttles.
