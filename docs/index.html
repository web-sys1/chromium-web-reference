Prerender: Add tests for main frame redirections in prerendered pages

This CL adds tests for main frame navigation with redirections in
prerendered pages.

Test cases are as follows:

1) Prerendered page
     --(navigate)--> {SameOrigin, SameSiteCrossOrigin}
     --(redirect)--> {SameOrigin, SameSiteCrossOrigin, CrossSite}

2) Prerendered page
     --(navigate)--> {SameOrigin, SameSiteCrossOrigin}
     --(redirect)--> {SameOrigin, SameSiteCrossOrigin}
     --(redirect)--> {SameOrigin, SameSiteCrossOrigin, CrossSite}
