[PromptAPI] call.arguments_json should not be nullptr

This CL is to remove the TODO in HandleToolCallOutput and adjust the
call to disallow a nullptr value from call.arguments_json.

Model should return arguments of {} for tool that takes no argument.
Model will also return arguments of {} for tool that takes a
dictionary object with optional fields. JS APP is able to distinguish
them based on the tool name and its corresponding inputSchema.
