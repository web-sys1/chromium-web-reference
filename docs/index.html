Rename AggregationServiceKeyStorage to AggregationServiceStorage

Originally, it was planned to have two separate interfaces backed by the
same underlying database (for key storage and report request storage).
However, we are now merging these to allow for continued use of
base::SequenceBound (which does not support casting to two separate base
classes).

This will be used by a future cl (crrev.com/c/3733377).
