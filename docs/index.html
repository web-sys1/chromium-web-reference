Fix WebcamPrivate double response

The WebcamPrivate Set function can be used to set more than one
parameter in the same call. It counts how many parameters are being set
and issues a response when they have all finished. This commit fixes a
bug where the response could be sent before the number of parameters
being changed was fully determined. This could result in more than one
response being sent.
