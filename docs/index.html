skills: Fix flaky SkillsEmojiPicker test on Linux

SkillsBrowserTest.SkillsEmojiPicker was flaky on Linux due to a race
condition where the test dispatched a click event before the component
finished registering its document-level click listener (which happens
asynchronously via requestAnimationFrame).

This CL fixes the flakiness by waiting for a requestAnimationFrame in
the test setup to ensure all component initialization is complete before
tests start. It also re-enables the test on Linux.
