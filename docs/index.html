Enable WebAppSettings policy and RunOnOsLogin on ChromeOS

Enable WebAppSettings policy and corresponding policy handler in ChromeOS. This policy will be used to allow setting the RunOnOsLoginMode for PWAs on ChromeOS.

This implements the RunOnOsLogin feature for the managed scenario. The existing mechanism updates the app's RunOnOsLogin mode. The actual launch is done using the WebAppLRunOnOsLaunchManager after the external managers have been synchronized.

This feature is enabled with kDesktopPWAsRunOnOsLogin.
