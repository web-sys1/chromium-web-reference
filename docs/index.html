clang build: Update to cmake 3.23

Fixes a TODO about the mac/arm build using a different cmake version
than the other platforms.

Switched to the newest cmake version while at it.

Copied the cmake binaries to the bucket with:

    for x in cmake-3.23.0-windows-x86_64.zip \
             cmake-3.23.0-macos-universal.tar.gz \
             cmake-3.23.0-linux-x86_64.tar.gz ; do
        curl -LO "https://github.com/Kitware/CMake/releases/download/v3.23.0/$x" &&
        gsutil.py cp -n -a public-read $x gs://chromium-browser-clang/tools/$x
    done

No intended behavior change.
