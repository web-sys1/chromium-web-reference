remoting: Support overridable monitor resolution

This change allows the users to provide the number
of virtual monitors by specifying the initial
size / resolution of each monitor.

Users will be able to persist the monitor configuration
by editing the service file as follows:

sudo systemctl edit chrome-remote-desktop@$USER

And then adding the configuration for the monitors e.g.:

[Service]
Environment=CHROME_REMOTE_DESKTOP_WAYLAND_DESKTOP_SIZES=1280x720,1280x720
