Implement GetAIPageContent duplication metrics

Introduce UMA metrics inside optimization_guide::GetAIPageContent() to
measure redundant, concurrent, or overlapping page content extraction
requests. This helps analyze and measure the overhead from duplicated
extractions. It will be used to inform the priority of (and measure the
impact of) transitioning usage to the PageContentExtractionService.

This telemetry is logged via a new page-scoped tracking object, the
AIPageContentMetricsLogger.
