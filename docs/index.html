service worker: Remove GetCoreThreadId().

This always just returns BrowserThread::UI, so callers can just use
that directly. This CL:

  DCHECK_CURRENTLY_ON(BrowserThread::UI) in bulk.
- Replaces uses of GetTaskRunnerForThread(GetCoreThreadId()) with
  GetUIThreadTaskRunner().

- Collapses some unnecessary helpers in service_worker_test_helpers.cc.
- Removes some comments mentioning GetCoreThreadId().
