[iOS Blink][tvOS] Fix build break by adding crypto:test_support dependency

After https://crrev.com/c/7414587 enabled
crypto/apple/unexportable_key_mac.mm on iOS,
authenticator_impl_unittest.cc requires headers from
//crypto:test_support. This dependency was missing, causing build
failures on iOS Blink and tvOS. Also, unexportable_key_apple.mm uses
LAContext, which is not available on tvOS. Thus, this CL disables the
files on tvOS.
