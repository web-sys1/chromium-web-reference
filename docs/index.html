Make background dimming of TableViewTextButtonItem optional

The default behavior set alpha to 50% in disabled state, but
this does not fit in every views.
Make this feature optional. If it is disabled, disable state
items will still use buttonTextColor and buttonBackgroundColor.

It is the responsibility of the Table VC to set these color
before calling `configureCell:withStyler:`.
