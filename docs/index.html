[fuchsia] Support raw VMOs in mem_buffer_util

//base/fuchsia/mem_buffer_util.h provides function to convert from/to
the fuchsia.mem.Buffer type used to wrap potentially large values in
FIDL messages.
The type fuchsia.mem.Buffer is deprecated now that VMOs can indicate the
size of their contents (https://fxbug.dev/85452). This change adds
similar functions to convert from and to VMOs.
