Reland "Migrate OnTask navigation throttle unit tests to browser tests"

This is a reland of commit 5275846ed46d1bedda490cbbd598cc77de33e742

Reland changes: We add explicit waits in the integration tests to ensure
the blocklist is updated before we attempt to navigate to other URLs.
The missing waits introduced a race and had us falsely assume we could
navigate multiple levels deep after spawning a child tab.
