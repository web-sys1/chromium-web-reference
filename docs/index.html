Add utm_campaign logging for universal link native map

Counterfactual logging for the control arm:
- Adds the counterfactual feature flag
- When the counterfactual flag is enabled, eligible universal links (as
determined by the `MiniMapTabHelper::ShouldInterceptRequest` function
that is shared with the treatment arm) will have a
`utm_campaign=as-npc-bling` added for install tracking.

Treatment logging:
- Modifies the URL to add "as-npt-bling" utm_campaign before opening it in native preview. This makes sure that Maps Lite fallback also has the campaign ID.
