[infra] Do not call json.encode on Exe protobuf message.

It's pure happenstance that protobuf messages can be passed to
json.encode without failing and it's likely to produce incorrect results
under some situations. Instead, proto.to_jsonpb can be used to get a
string containing a json message conforming to the jsonpb format and
json.decode can be used to get the resulting object to be included as
part of the properties.
