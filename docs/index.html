WebGPU: add [NoAllocDirectCall] on GPUCommandEncoder functions

This patch adds [NoAllocDirectCall] on all the member functions of
GPUCommandEncoder that are qualified to be V8 fast apis and the
related blink web tests.

This patch also adds a check to the existing WebGPU fast API tests
to fail the test when no expected errors are thrown during the
execution of the test functions.
