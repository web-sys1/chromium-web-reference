Initialize a member var in ResizeShadow class

This CL fixes the use-uninitialized-value in ResizeShadow class.
ResizeShadow is invisible by default so |visible_| should be false by
default as well. Otherwise, |visible_| can be referenced before
assignment by ResizeShadowController via
ResizeShadow::UpdateBoundsAndVisibility and that causes the unintended
visible shadow.
