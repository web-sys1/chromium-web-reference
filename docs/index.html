Stop using DevTools build scripts in Chrome OS build

The DevTools build scripts are internal implementation details and are
not intended to be used outside of DevTools. We discovered this
dependency as we are removing `rjsmin.py` ourselves, since we no longer
depend on it. To unblock our autoroller and removal of the script, add
the `rjsmin.py` script to `third_party` and use that in chromevox.
