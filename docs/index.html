bluetooth: Update histogram expirations

This change extends the following soon-to-expire histograms for the next
year:

 * Bluetooth.Web.GetCharacteristics.Characteristic
 * Bluetooth.Web.GetPrimaryService.Services
 * Bluetooth.Web.GetPrimaryServices.Services
 * Bluetooth.Web.RequestDevice.RSSISignalStrength
 * Bluetooth.Web.RequestDevice.RSSISignalStrengthLevel

The Bluetooth.Web.ChooserInteraction histogram has been removed, as I
don't believe it provides any useful metrics.

In addition to extending the RSSI-related metrics, the thresholds for
how RSSI is mapped to a level for the device chooser UI has been updated
based on the latest data from this metric.
