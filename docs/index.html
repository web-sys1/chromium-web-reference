Remove dead marker painting code

The Highlight Pseudo markers are excluded from the marker_
set in HighlightPainter, so there is no need to handle then in
the switch statement that paint markers from markers_. It was super
confusing to have the code there when it was never called.

The code doing the exclusion is in
DocumentMarkerController::ComputeMarkersToPaint, which is itself
somewhat confusingly named since it only computes a subset.
