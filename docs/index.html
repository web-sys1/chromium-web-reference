GestureNav: Delete navigation sheet logic

Navigation sheet is not used in gesture navigation UI. This CL deletes
the related logic.

HistoryNavigationDelegate also becomes not necessary any more as it was
devised to minimize dependencies on chrome_java (mainly Tab, back when
it was still a part of it) and create two subdelegates(ActionDelegate
and SheetDelegate). Now that Tab was already modularized and
SheetDelegate is not used for gesture navigation, it can be removed
as well. NavigationHandler is passed Tab directly.

ActionDelegate was redefined to BackActionDelegate that consists of:
- getBackActionType: return the type of actions for back gesture
                     for the arrow bubble text
- onBackGesture: perform the actual back gesture action

which both need dependencies on TabbedChromeActivity. It is defined
in root ui coordinator, and passed to the coordinator. It effectively
replaces a function/runnable passed individually.

Now NavigationSheet is used for the popup brought up by long press
on back button only.
