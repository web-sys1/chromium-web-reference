[sync] Extend BookmarkModelView with support for account bookmarks

A new subclass is introduced, BookmarkModelViewUsingAccountNodes,
that exposes the newly-introduced permanent folders representing
account bookmarks.

To avoid issues, RemoveAllUserBookmarks() is renamed to fit better
the BookmarkModelView abstraction, and NOTIMPLEMENTED() is used in
the new class to avoid that it deletes non-account bookmarks.
