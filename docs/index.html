Open Price Drop URL action handler

Open Price Drop URL when the user taps on a price drop notification
or long presses the notification and presses 'Visit Site'. This is a simple implementation whereby a new Tab is opened regardless of the URL.
A more sophisticated approach will be implemented behind a separate
feature parameter whereby we attempt to search for the URL and open
the corresponding Tab if we find a Tab with the price drop URL. Given the complexity of the find an existing Tab approach, it is best to have a separate simpler variant in the event that there are unforeseen
issues with the more complex approach (so we can simply switch
the approach during an experiment).
