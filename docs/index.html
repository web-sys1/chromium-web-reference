Add Account Capabilities section to chrome://signin-internals

This change introduces a new section in chrome://signin-internals that
displays the account capabilities for each signed-in account.

For local builds, the UI also adds the ability to override the value of
individual capabilities, via a drop-down. The overrides are written to
prefs so they are persisted across restarts.

See screenshots in the linked bug.
