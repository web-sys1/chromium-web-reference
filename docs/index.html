Select to speak: Remove mv2 tests

Since select to speak has successfully launched in mv3, we are
removing the mv2 tests. This will help us remove duplicate tests and
unhook ourselves from the mv2 infrastructure, helping to unblock the
extensions team.

Note that we will keep the mv2 implementation
around for a couple of milestones just in case we need to rollback
in the case of a severe regression.
