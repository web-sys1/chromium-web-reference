[fuchsia] Create hardware video encoder.

Create FuchsiaVideoEncodeAccelerator, which lives on the GPU process
and communicates via MojoVideoEncodeAccelerator.

Create a dedicated IO thread in GpuServiceImpl for Fuchsia VEA, as
FIDL communication does not currently work on ThreadPool's worker
threads.

It is guarded behind the feature FuchsiaMediacodecVideoEncoder,
and requires fuchsia.mediacodec.CodecFactory capability in the GPU
process to work.
