Expose latest RecoveryIds in chrome://cryptohome.

RecoveryId is a unique identifier of cryptohome recovery operation. It
will be visible to users for transparency (so that a user can check if
the local recovery ids on their devices match their ledger entries).
This cl uses a new dbus call for getting extended info on an AuthFactor
and exposes latest RecoveryIds in chrome://cryptohome for debug purpose.
Similar mechanism will be used to expose user's RecoveryId history in
Settings.
