Add audt message for '/Last Browser' in Incognito containment test.

"/Last Browser" file contains the path to the latest Chrome executable
and is refreshed on each Chrome Startup.
Refreshing it while user browsers in Incognito mode only reveals a
timestamp on the last usage of Chrome, and not even the last usage of
Incognito.
Therefore it is safe to be modified in Incognito mode.
