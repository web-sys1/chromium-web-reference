[iOS][MediaPermissions][6/N] Adds infobar UI

This CL updates the modal permissions viewController and mediator.

The mediator now observes and interacts with the web state
to get/set permissions state. It also implements the
delegate & consumer, used to provide information needed
by the viewController.

The viewController implements the InfobarPermissionsModalConsumer, used
to retrieve information from the mediator.
