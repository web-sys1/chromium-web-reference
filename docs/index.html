Notify modifier update always.

The original code sends modifiers on KeyEvent sending.
However, we had an edge case that if the event triggered an accelerator
(such as ALT+TAB), KeyEvent is not propagated to exo::Keyboard,
so that the modifier state was not updated until next Key event comes.
This resulted some weird behavior in Lacros.

This change will send the modifier state update always.
