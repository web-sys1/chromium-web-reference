[UIA] Expose Toggle pattern on all checkable elements but expandable buttons

This CL make sure that whenever `aria-checked` property is mapped,
Toggle Pattern should be exposed. CL:4495447 introduced the
condition that Expand Collapse Pattern and Toggle Pattern should
not be exposed together for any element. However, the expected
behavior according to the spec is that they should not be
exposed together only when on a button element.

This issue was caught for the tree item elements, but the fix
should handle other non-buttons elements, as well, which support both
Expand Collapse Pattern and Toggle Pattern.
