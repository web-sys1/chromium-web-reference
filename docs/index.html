lacros: Plumb delivery failure for FSP operation forwarding.

The ChromeOS file system can send operations to Lacros-extension
implemented file systems (via fileSystemProvider API). If an operation
cannot be delivered (e.g. because the extension manifest does not
support it), the ChromeOS file system must be informed -- otherwise the
pending operation will never be completed and will show a user-visible
notification.

This CL implements this notification failure by adding a return
signature to ForwardOperation.

TEST=Install file system for one drive and file system for dropbox and
confirm that they work with Lacros.
