[CrosWindowManagement] Create Blink Extension stub.

This CL adds the Chrome OS Window Management Blink Extension to be
accessible via chromeos.windowManagement. This CL also changes the
ChromeOS namespace into an interface which is added, as this better
matches the convention for global objects that have optional members.

Future CLs will add a mojom interface into blink/public/ to be
implemented by Chrome.
