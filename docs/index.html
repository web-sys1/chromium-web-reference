Add dawn's test with implicit device synchorization to CI.

Implicit device synchorization is an optional feature of Dawn that
provides thread safety when using dawn APIs on multiple threads.

This CL adds a config to test dawn with this feature enabled on CI. This
is to make sure that this feature won't break existing dawn's tests and
there is no unexpected deadlocks if it is turned on.
