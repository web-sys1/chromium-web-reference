Implement WebRequest OnBeforeRequest for WebTransport

This CL implements WebRequest OnBeforeRequest for WebTransport and some
basic structures.

Change WillCreateWebTransportCallback
- chrome/browser/chrome_content_browser_client.cc
Call extensions::WebRequestAPI::ProxyWebTransport after safe browsing
check was passed or if it doesn't exist via MayInterceptWebTransport.
Then WebTransportHandshakeProxy invokes OnBeforeRequest extension
handlers if exists.

The callback has an argument WebTransportHandshakeClient in addition
to WebTransportError. This will be used to handle onHeadersReceived
events later-on.
