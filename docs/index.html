WebUI: Only focus first item in menus with using keyboard

Previously, when a user clicked into a menu item, this would dispatch
a focusin event to cr-menu-selector which would then try to focus the
first item in the list. However, this does not make sense with a mouse
and can cause issues when the menu is scrolled. This CL fixes this bug
by only moving focus if the focus event was caused by a keyboard.
