Use a HeapHashMap for UserScrollableAreas

It was noted in a comment in LocalFrameView that we could use a hash
map to speed up lookup (we need to be able to find a scrollable area
matching a given CompositorElementId). In preparation for using this
lookup more frequently in a future CL, we make the change to using a
hash map here.
