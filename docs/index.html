policy: Rename the unlaunched managed screensaver policies

This CL adds infixes (LoginScreen/LockScreen) to the managed
screensaver policies to better describe their intention.

The device policies are meant to configure the screensaver
for the login (sign-in) screen. The user policies refer to
the lock screen configuration of the screensaver.

This CL does not modify any schema, and only changes policies
not yet launched.
