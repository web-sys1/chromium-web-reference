[HistoryExc] Change back long press behavior for incognito in iOS.

Incognito history excellence project aims to remove the history entry
points from incognito mode, while providing consistency across
platforms. Long press to the back button is one of them, as this CL
updates it regarding the approved UX deck.

This CL updates the back press menu for incognito as;
- Replace "New tab" item with "New incognito tab"
- Replace default icon for NTP item with incognito icon.

All those changes are behind the flag which is disabled by default.
(update-history-entry-points-in-incognito)
