[iOS12 Deprecation]: Removed base::ios::IsMultiwindowSupported() API

Since iOS 12 deprecation, all iOS versions support multiwindow.
All build targets should have the multiwindow plist flag now. This CL
removes base::ios::IsMultiwindowSupported and all code relying on it
returning false.

It also removes PreviousSessionInfo.isMultiWindowEnabledSession property
and set the MultiWindow flag in testing/gtest_ios/unittest-Info.plist to
true.
