NativeIO: Remove the WeakPtrFactory from NativeIOManager.

None of the methods currently bound using weak pointers need the
NativeIOManager context. This CL eliminates three methods altogether,
and converts the remaining one into an inline lambda.

This slightly simplifies the lifetime of NativeIOManager instances.
