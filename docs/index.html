media: Add granular D3D12 Video Encoder error codes

Replaces the generic `kSystemAPICallError` with specific D3D12 error
codes in the D3D12 video encoder implementation to improve telemetry
and debugging.

Adds 13 new error codes to `EncoderStatus::Codes` corresponding to
specific DirectX API failures (e.g., CommandListReset, Map,
CheckFeatureSupport, FenceWait) and updates the D3D12 wrapper,
helpers, and delegates to report them accurately.
