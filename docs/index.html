[ios] Convert absl::optional<T> to std::optional<T>

Since https://crrev.com/c/4469528, absl::optional<T> is an alias for
std::optional<T>, so convert uses of the old type to the new one as
it is easier to use (no needs to have public_deps on absl, shorter
path to the header, ...).

This CL converts all uses in //ios/web_view and below.
