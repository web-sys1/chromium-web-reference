Fix anchor attribute in XML crash

I added a failure expectation because we still need to figure out if the
anchor element should work in XML documents or not, but this patch still
fixes a crash when the anchor attribute is set in XML.

If we decide that the anchor attribute shouldn't work in XML, then I'll
have to move the IDL and some methods from Element to HTMLElement which
will be a bigger patch.
