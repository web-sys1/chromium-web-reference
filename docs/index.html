Enable use counter mapping for animated font-palette

This was commented out in this review in May 2024:
https://chromium-review.googlesource.com/c/chromium/src/+/5560554

At the time the use counter value was very low and the sites listed on
https://chromestatus.com/metrics/css/timeline/animated/709 didn't seem
to actually animate font-palette. It's still the case that most of the
sites listed don't have any visible animated text. Most of them are
built using framer.com and just loading these sites doesn't trigger the
use counter in my testing. It could be that it depends on something
non-deterministic.

https://lowlands.nl/faq/ is among the sites and does have a very visible
animated font palette that does trigger the use counter.

https://textlab.dev/posts/animating-font-palette was also tested to
confirm it triggers the use counters.

Bottom line, the use counter seems to work and can be enabled.
