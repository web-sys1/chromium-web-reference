[Snooping protection] A minimal settings subpage.

This is the second in a series of CLs that implement the snooping
detection settings UI. See go/cros-spa-ui-impl for feature details, and
go/cros-spa-settings-mocks for settings mocks.

This CL creates the most minimal settings subpage needed to control the
snooping protection preference. A subsequent CL will add the full UI /
functionality.

The sequence of CLs planned is:
  1) Add route constants to the C++ backend.
  2) Create and route to a minimal subpage.          [<]
  3) Implement the real subpage and subpage tests.
  4) Define search tags, settings and deep links.
