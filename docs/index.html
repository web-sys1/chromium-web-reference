[CrOS Cellular] Fix activation code string

We show a different string based on whether or not the QR code scanner
is available. This description is rendered once the element loads, but
it is possible for us to learn whether or not the QR code scanner is
available after the element is rendered.

This CL simply updates the HTML to update the description anytime the QR
code availability changes so that if it takes a bit of time to determine
the availability, the string will update dynamically.
