system-extensions: Remove unnecessary 'done()' call

Tests were based on a dedicated worker examples that according to
testharness.js documentation[1] requires a done() call because there is
no "load" event. Service worker tests can use the `install` event so
they don't need an explicit call to done().

[1] https://web-platform-tests.org/writing-tests/testharness-api.html#web-workers
