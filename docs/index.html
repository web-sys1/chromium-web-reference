[iOS][Default Promo] Add timers for show/dismiss of non-modal promo

This CL adds the initial 3 second timer after an omnibox paste to show
the promo and the timer to dismiss the promo after it has shown for some
time.

It also observes messages and webstate to prevent the promo from showing
if a message is presented or the active webstate changes in between
the triggering event and the actual promo showing.
