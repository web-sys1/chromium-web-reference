[fyfre] Use the "promo" Gaia signin flow page for FRE

The "promo" version of the Gaia signin page indicates to the user that
they are signing in to Chrome, but does not mention sync benefits. It
is intended to be used for flows where the next page would be a sync
opt-in that can be declined while proceeding with the main flow.

Outside of the FRE, the "promo" page can be used for the profile creation flow by enabling the "PromoGaiaFlow" feature.
