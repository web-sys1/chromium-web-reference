Glic: Add metrics for turn source, daisy-chain depth, and  floaty mode

Add several new metrics to provide more detailed insights into user
interaction patterns with the Glic feature. These metrics will help in
understanding how different aspects of the UI are used and how users
navigate from Glic-provided content.

The new metrics include:
*   `Glic.Turn.Source`: Records the UI mode (Side Panel or Floaty)
    and the input mode (Text or Audio) for each turn.

*   `Glic.Tab.DaisyChainDepth`: Tracks the depth of tabs that are
    opened from other tabs originally created by Glic.

*   `Glic.Instance.Floaty.InitialMode`: Logs whether the floating UI
    was initiated with text or audio input active.
