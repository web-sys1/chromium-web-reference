[FRE] Disable the signin/out buttons when signin/out is in progress in signin FRE

This cl adds a property field to register when a signin or signout event
starts in SigninFirstRunProperties. While signin or signout is in
progress the "Continue as ..." and "Continue without an account" buttons
are disabled. In addition the selected account row is also disabled.

The property field will allow a loading spinner to be added to the UI
in future cls.
