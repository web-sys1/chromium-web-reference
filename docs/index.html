Replace illegal chars in PWA filenames with spaces

This changes how PWA shortcut names are sanitized on Windows to make
the result more readable.

On Windows, the shortcuts shown in the Start Menu and on the Desktop
are files. Because of this, the name shown can't have any characters
that are illegal in filenames.

Currently, PWA names are sanitized for this purpose by replacing
illegal characters with underscores. With this change, illegal
characters are replaced with spaces instead, hopefully with a more
readable result.

For example:
PWA name:               BattleCraft: Fight for Chaos
shortcut (current):     BattleCraft_ Fight for Chaos
shortcut (this change): BattleCraft  Fight for Chaos
