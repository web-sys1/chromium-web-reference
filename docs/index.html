Add getter for WebStateImpl::WebView

Some features may need to access the WKWebView.
The main use case is to pass it to other WebKit API that require
the web view itself.
Add an accessor in WebStateImpl (and in CRWWebController) to
retrieve the WKWebView if it exists.

It is still discouraged to access it directly if a proxy object
could be used instead. To avoid having unreviewed usage, guard
the call to the method using a passkey.
