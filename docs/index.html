wm: Fix multidisplay issue with vertical snap to maximize

Both the maximize cue and the phantom window animation from snap to
maximize weren't displayed properly on a external monitor. The issue was
both of them were using `Widget::SetBounds()` which is in screen with
`Layer::SetBoundsFromAnimation()` which is in parent.

While fixing this issue I also replaced the bounds animation with a
transform animation for some minor performance gains.
