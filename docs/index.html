update_client: Implement install data index.

update_client now provides a way to specify an `install_data_index` as a
`CrxComponent` attribute. If specified, the `install_data_index` is sent
to the update server. If there is a match, the server will provide the
corresponding installer data in the update response. This installer data
is then provided to the installer in `CrxInstaller::InstallParams` when
running it.
