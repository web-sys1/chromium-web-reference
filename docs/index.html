focus_mode: Add retries to `GetTaskLists`  and `GetTasks`

This adds simple retries to `GetTaskLists`  and `GetTasks` requests. The
retry policies are described below:
  - HTTP error 429:
    Fixed 30 seconds wait time. Max 3 retries.
  - HTTP error 408, 500, 502, 503, and 504:
    Exponential backoff with 1 second initial wait time. Max 5 retries.
