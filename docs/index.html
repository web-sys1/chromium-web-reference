Run an origin trial for Device Attributes API

To run an origin trial for this API, the runtime feature status needs to
be changed accordingly. The 'ManagedInterface' runtime feature should be
removed because 'Managed Configuration' has been at stable stage. The
interface is always exposed to website since we are using 'implied_by'
field to connect with each other.

The default value of Chrome feature 'EnableRestrictedWebApis' is changed
from false to true. Device users are still able to turn it off in the
flags page if needed.
