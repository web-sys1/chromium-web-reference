[Ozone/Wayland] Set visual_bounds_px_ of WaylandWindow

bounds_px_ of WaylandWindow is used by PlatformWindow to set window
bounds. It is different from the visible bounds given from the gpu
process due to potential delays.

This CL uses visual_bounds_px to reflect the visible WaylandWindow
bounds, and acknowledge xdg-toplevel configure events with the correct
serial.

See: https://docs.google.com/document/d/1qDaz3LC7PPFMuI_UmGxuYLWr9bJP8F7aTU-tjxXCBnA/edit?usp=sharing
