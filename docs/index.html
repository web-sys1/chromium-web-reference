[Extensions] Remove NOTIFICATON_USER_SCRIPTS_UPDATED notification

Currently, the notification is being used only in test code. Some tests
can have the notification observer removed entirely as loading an
extension inside a test now will wait for content scripts to load.

The remaining tests have been converted to use
UserScriptLoader::Observer. Subsequent patchsets will re-examine these
tests to see if they could be rewritten to not require the listener.
