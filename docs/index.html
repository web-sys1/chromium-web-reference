iwa: Check signatures once per Chrome session on all platforms

This CL updates the `IsolatedWebAppReaderRegistry` to check signatures
once per Chrome session, even on ChromeOS. This is a temporary
workaround until we have added an explicit signature checking step to
the Isolated Web App installation process.

For reference, as documented in the design doc [1], we only want to
check signatures during installation on ChromeOS, and not during every
Chrome session. However, given that this is solely a performance
optimization for lower-end hardware, we will defer this optimization for
now.
