Split build target of /c/common/url_constants

Move /c/common/url_constants.* and webui_url_constants.* to
/c/common:url_constants, so it can be a depended from build targets
under /c/common/.
Also clean up some public dependencies.

Note that they are not moved to /c/common:constants because that
target has external dependents and will cause link issues. Create a
new target to minimize the impact to the existing targets.
