[Color Pipeline] Fix high contrast colors that depend on toolbar colors

DownloadShelf, InfoBar, and NTP background should match toolbar colors.
However, during high contrast mode this is not happening. This CL fixes
those discrepancies. This CL also modifies how
COLOR_DOWNLOAD_SHELF_BUTTON_TEXT is returned in the ThemeHelper. This
ensures both ThemeService and ColorProvider are in sync as well as
ensure that there is no visual change for
COLOR_DOWNLOAD_SHELF_BUTTON_TEXT with the new DownloadShelf color in
high contrast dark mode.

Visual changes for DownloadShelf, InfoBar, and NTP background are
tracked in the linked bugs.
