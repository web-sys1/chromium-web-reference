[SuperSize] Viewer: Metadata: Add minor UI tweaks.

This CL adds a few QoL improvements to the Metadata View:
* Render numbers using formatNumber(), to get comma separator.
* In diff mode, highlight each "after" value (light yellow) that differs
  from the corresponding "before" value.
* Repurpose |window.supersize| for usage from DevTools console.
  * Move declaration from start-worker.js to ui-base.js.
  * Add supersize.metadata: The most recently loaded metadata.
  * Add supersize.prettyMetadata(): A helper to render SSV.metadata as
    JSON text that resembles the legacy Metadata View.
