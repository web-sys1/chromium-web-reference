[eche] Closing the Eche bubble when other bubbles open.

Please note that we have a method called ClickedOutsideBubble() that is called when a click event happens outside of the bubble. However, here in this CL we are only focusing on clicking on other tray bubbles. Those are the only clicks that result in closing Eche bubble.

In this CL, we also move the `views::WidgetObserver` from TrayBubbleWrapper to the TrayBubbleBase to make sure it covers all the implementations.
