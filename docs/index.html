Output INFO logs for signin/signout operations on Android

The current logs for signin and signout operations on Android are
DEBUG-level.  As these are quick high-level events which can be useful
for debugging, output these at INFO-level so that they can be manually
enabled in production builds (DEBUG logs are stripped from these [1]).

Also fix some existing logs to use string formatting rather than
concatenation per performance guidance at [2].

[1] https://source.chromium.org/chromium/chromium/src/+/main:docs/android_logging.md;l=38;drc=4a9e3965af274d897e637cecd5c2d4b354791a1f
[2] https://source.chromium.org/chromium/chromium/src/+/main:docs/android_logging.md;l=88;drc=4a9e3965af274d897e637cecd5c2d4b354791a1f
