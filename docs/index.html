[UPMLocalPwd] Split the proxy backend into account and profile

The account backend can only be used after the split which is part
of the UPM for local activation. It supports unenrollment.

The profile backend now has 2 possible versions:
- old UPM version: can be used for syncing or non-syncing passwords
- new UPM version: can only be used for local passwords and doesn't support
unenrollment
