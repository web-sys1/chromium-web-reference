[iOS] Match all Home module widths

Ensures that the width of the visible Magic Stack collection view, and
the module on top of it (if present) matches the width of the feed
module. The visible module within the Magic Stack becomes left aligned,
since we don't want it to take the entire width to allow for some of the
next module to peek out (The visible module width will be addressed in a
follow-up CL to avoid any text from the peeking module to show).

Also renames all instances of `ntpDelegate` to `NTPDelegate` as drive-by
refactoring.
