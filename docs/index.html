Add a helper to convert proto specifics field numbers to ModelTypeSet

There are many usages where we follow the same pattern to convert
repeated profot fields containing specifics field numbers into
ModelTypeSet. This change adds a new helper function -
GetModelTypeSetFromSpecificsFieldNumberList - to achieve the same, and
also updates the usages.
