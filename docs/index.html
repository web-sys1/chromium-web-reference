[Metadata][Metrics] Issuer selected metrics

Log a boolean histogram regarding whether the selected Autofill
suggestion had metadata. The histograms are logged only for cards with
an issuer id, and are pooled for each issuer i.e. only 1 histogram is
logged per issuer. For an issuer, if the card suggestion had metadata,
the histogram logs true. If it didn't, the histogram logs false.
