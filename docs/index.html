[TTS] Update test suite for literal tap feature

Updates the main Contextual Search test suite's existing longpress
and translation parameterized tests to also enable the new literal tap
feature, since that's likely our launch candidate now.

This updates the tests that use a non-resolving gesture to work for both
longpress (when not running the longpress resolving experiments), and
for the literal (non-resolving) tap. This updates the tests that use a
non-resolve gesture to be parameterized to test multiple experiments.

Removes the LiteralSearchTap test since that case is now well covered
by the parameterized tests.

Also includes speculative fixes for testChainedSearchCreatesNewContent
and testChainedTapsRemovedFromHistory.
