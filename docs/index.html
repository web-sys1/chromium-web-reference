[bedrock] Replace browser->window() across chrome/browser/ (phase 2 part 2.2)

Part of eliminating Browser::window(). This batch replaces browser->window()
(and the equivalent GetBrowserForMigrationOnly()->window()) with one of:

  * BrowserWindow::FromBrowser(browser) - the documented drop-in
    replacement, used when a BrowserWindow* is needed.
  * browser->GetWindow() - used when only the ui::BaseWindow interface
    is needed (avoids the heavier BrowserWindow downcast).

FromBrowser (BrowserWindow methods / BrowserWindow* locals):
  ash/boca/on_task/on_task_system_web_app_manager_impl_browsertest.cc
  extensions/api/passwords_private/passwords_private_delegate_factory.cc
  preloading/prefetch/search_prefetch/search_prefetch_service_browsertest.cc
  preloading/prefetch/search_prefetch/search_preload_unified_browsertest.cc
  preloading/prefetch/zero_suggest_prefetch/
      zero_suggest_prefetch_tab_helper_browsertest.cc
  preloading/prerender/prerender_omnibox_interactive_uitest.cc
  renderer_context_menu/render_view_context_menu.cc
  sessions/app_session_service.cc
  sessions/session_restore_browsertest.cc
  sessions/session_service.cc
  sessions/session_service_base.cc
  sessions/session_service_browsertest.cc
  site_isolation/site_per_process_text_input_interactive_uitest.cc
  ssl/https_upgrades_browsertest.cc
  ssl/typed_navigation_upgrade_throttle_interactive_uitest.cc
  themes/theme_service_browsertest.cc
  translate/chrome_translate_client.cc

GetWindow (ui::BaseWindow only):
  sessions/session_restore_interactive_uitest.cc

No behavior change.
