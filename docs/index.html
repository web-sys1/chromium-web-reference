[MIM] Rename |dismiss_count| to |user_act_count|

It was further clarified that it is not only the active dismissal
by gesture but also any user-intervened response (accepting sign-in
but not actually going all the way through completing sign-in
process) should be counted toward the condition that determines
the next MIM display. This CL renames the field |dismiss_count|
to |user_act_count| to include both dismissed/accepted close type.
Renaming protocol buf field is a backward-compatible change,
so may be acceptable at this point where no active users exist.
