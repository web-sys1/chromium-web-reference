Show magnifier during swipe to move cursor gesture for CrOS.

Add HandleSwipeToMoveCursorGestureAck to notify touch selection
controllers and clients of insertion drag events when swipe to move
cursor is occurring. This enables existing magnifier logic in
TouchSelectionControllerClientAura to be reused for showing a magnifier
during a swipe to move cursor gesture. (Note that the actual magnifier
is only implemented for CrOS, though the magnifier handling logic is
shared aura code.)

The overall approach sort of tries to follow the corresponding Android
implementation:
https://source.chromium.org/chromium/chromium/src/+/refs/heads/main:content/browser/renderer_host/render_widget_host_view_android.cc;l=3189;drc=fe8fa98aca34ccf3b40d3a65893dc72a48a07c52
There are some small differences due to some implementation details.
May be worth trying to unify the implementations a bit more in a later
CL.
