Increase timeout for radio-focus-navigation-group-focusable-focus.html test

This test takes close to 10 seconds to complete in CI environments,
which causes intermittent timeouts. After reviewing CI logs, it appears
that the test consistently runs for nearly the default 10-second timeout
limit.

To address this, this CL adds a <meta name="timeout" content="long">
declaration to increase the timeout to 60 seconds, ensuring the test
has enough time to complete and reducing flakiness.
