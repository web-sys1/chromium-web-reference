GlobalConfirmInfoBar: Add IsCloseable() method

When this method is not present, it always defaults to true as per the
default implementation of the parent class of ConfirmInfoBar.

This means that any overrides by children classes aren't picked up by
the delegate proxy.

By adding this method, it becomes possible to have non-closeable windows
for delegate children classes of ConfirmInfoBar.
