Ensure sync end notification is sent

In EGTests with signin in, especially the TabGridTestCase, the UI layer
is not notified of the end of the sync, and therefore is not updated.
This CL adds a call to notify_foreign_session_updated_cb_ in
StartLocalSessionEventHandler() to ensure that the notification is
always triggered.

See bug's description for more details: https://crbug.com/1422634.
