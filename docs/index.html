[DIP] Implement DocumentIsolationPolicyReporter

This CL implements in content/ the DocumentIsolationPolicyReporter interface
that was defined in
https://chromium-review.googlesource.com/c/chromium/src/+/6074457. When
navigating, a DocumentIsolationPolicyReporter is created. It is then stored in
the RenderframeHost that committed the navigation. The Mojo endpoint for this
interface will be passed to the network service in subsequent CLs.

This CL is part of a chain of CLs implementing DocumentIsolationPolicy
reporting:
1) Add DocumentIsolationPolicyReporter
2) Implement DocumentIsolationPolicyReporter <- you are here
3) Pass the reporter to URLloaderFactory
4) Plumb DIP reporter into workers
5) Plumb DIP reporter in CacheStorage
