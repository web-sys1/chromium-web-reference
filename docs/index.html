[Read Anything] Implement a coordinator to manage the read anything component

This CL updates the structure of the "Read Anything" component to
follow a basic MVC design pattern. We add a coordinator to own the
component and a controller to manage it. Business logic is
moved out of the views and placed in the controller. There are no
functional changes to the existing feature with this change.
