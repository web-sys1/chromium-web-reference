[Launcher] Add skeleton class of RemovedResultsRanker.

This CL adds a skeleton class of RemovedResultsRanker. Folllow up CLs
will add the implementations.

Once fully implemented, this will be a simple ranker at the end of the
ranking stack. It will store search result IDs which have been marked
(via UI) for removal, and will filter these results on a call to `Rank`.
