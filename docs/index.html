[iOS] Enabled shared bundle id for test apps

This enables ios_use_shared_bundle_id_for_test_apps. Using a shared
bundle id for all test apps makes it practical to get the extended
virtual addressing entitlement for all of these apps, which in turn
is needed for running tests with PartitionAlloc enabled.
