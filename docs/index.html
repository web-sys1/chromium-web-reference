logging: Use syslog format for Lacros

Ash logs are in syslog format (see go/cros-syslog-format-log-chrome).
Currently, Lacros logs are in Chrome format, which is a bit confusing
and inconvenient when debugging the interaction between Ash and Lacros
or inconsistencies in behavior between them. Hence change Lacros to
also use syslog format.
