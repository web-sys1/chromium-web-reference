Use InlinedVector in /ui/events/gesture_detection

Replace usages of base::StackVector with absl::InlinedVector. This is
not a pure rename, as the semantics are slightly different:

1. Methods like push_back() which were accessed with the -> operator in
   StackVector are instead accessed with . with InlinedVector.
2. The .container() method is not needed with InlinedVector

See also discussion thread in cxx@chromium.org:
https://groups.google.com/a/chromium.org/g/cxx/c/jTfqVfU-Ka0/m/caaal90NCgAJ
