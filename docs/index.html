Add |NGInlineCursor::BoxFragment|

When block fragmentation is enabled, it is often necessary to
know which fragment |NGInlineCursor| is at. This function
returns the |NGPhysicalBoxFragment| for the current root
|NGFragmentItems|.

To do this, constructors are adjusted to take
|NGPhysicalBoxFragment|. Usually callers know it.

Also fixes a crash in |NGCraretRect| when block fragmentation
is enabled.

Other places that use functions to be eliminated for block
fragmentation (|ContainingBlockFlowFragment|,
|CurrentFragment|, |FragmentItems|, etc.) are not reviewed
yet. Will do so in future patches.
