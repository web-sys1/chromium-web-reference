[RWI] Add UseCounter when user clicks on mdoc:// deep link

This CL:
- Adds a use counter for when user clicks on mdoc:// deep link (vs
  going through navigator.identity API)
- Adds a use counter for when user clicks on openid4vp:// deep link
- Adds Android.TabNavigationInterceptResult.ForMdoc and
  Android.TabNavigationInterceptResult.ForOpenId4Vp histograms. The
  purpose of the histograms is to track how frequently the external
  app launch triggered by mdoc:// and opendi4vp:// deep links is
  mediated by a Chrome dialog and thus can be canceled.
