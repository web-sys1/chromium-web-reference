Make media query classes GarbageCollected

To support calc() in media queries (and container queries), we need
to store a CSSValue in MediaQueryExpValue. This CL prepares for that.

Moving this to Oilpan is a little painful, since it means converting
certain objects from std::unique_ptr (less overhead) to
GarbageCollected (more overhead), but there is no other good option
as long as CSSValue is GarbageCollected.

Persistent is also not an option, because it would create a gc-root
inside already-GarbageCollected objects.
