ozone/wayland: set properties before first commit

Some compositors, e.g. KDE Plasma, rely on the client to set the
surface properties before the initial commit. This CL moves the commit
from XdgToplevelWrapperImpl::Initialize() to
WaylandToplevelWindow::CreateShellToplevel(), achieving the desired
outcome.
