[iOS Blink] Share Mac's tests helper utilitis in SystemMetricsTest

SystemMetricsTest.MeasureChildCpuUsage has been failing on iOS Blink
port since https://crrev.com/c/4930011. This CL shares the test helper
utilities for Mac port with iOS Blink port to fix the test failure.

The existing iOS port also runs the tests. This CL uses
BUILDFLAG(IS_IOS) && BUILDFLAG(USE_BLINK) flag instead of IS_APPLE
simply.
