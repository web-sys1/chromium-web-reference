[braille-ime] Migrate from window to globalThis

Because the window object is not defined in service workers, we need to
migrate away from using it. globalThis serves a similar purpose to
window -- so much so that where window is defined, globalThis is just
a synonym for it! However globalThis is defined in all contexts, making
it better suited for manifest v3.
