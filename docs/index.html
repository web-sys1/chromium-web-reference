Fix SharedArrayBuffer reverse origin trial

SharedArrayBuffer will de disabled on non cross-origin isolated
contexts. The UnrestrictedSharedArrayBuffer reverse origin trial makes
it possible to temporarily keep SharedArrayBuffer.

The previous implementation had two main problems:

1. It was relying on a per-process v8 flag (while origin trials are
set per document/worker).

2. It did not support meta tags (because the v8 flag needed to be set
early enough).

This CL fixes those problems by using a newly implemented v8 mechanism
(https://crrev.com/c/2867473) which defines the SharedArrayBuffer
constructor on a global object depending on the return value of a
callback.
