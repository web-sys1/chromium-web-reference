Add connection between PDF renderer and ScreenAI service for OCR.

Creates a direct connection between PDF renderer and ScreenAI service,
so that ScreenAIAnnotator would be usable to extract text and layout
information from images in PDF files without passing data through.
For more information, see: http://go/chrome-screen-ai

The connection is set directly between PDF rederer and ScreenAI service
to avoid the need to send the data through an extra layer like browser.
