[Almanac] Remove memoization from DeviceInfoManager

DeviceInfoManager memoizes the DeviceInfo result to speed up future
calls to GetDeviceInfo. This is a bit of a premature optimization, since
no-one repeatedly calls GetDeviceInfo anyway.

Instead, we can remove this storage, and rely on clients to do the same
thing if they need it.
