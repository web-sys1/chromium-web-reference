[Blink] Have CanvasRenderingContext2D implement GetColorSpace "natively"

Changes CanvasRenderingContext2D to implement GetColorSpace() directly
and layer GetSkColorSpace() on top of it rather than the other way
around. Note that this is a behavioral no-op due to [1][2][3].

This is a step toward eventual elimination of
CanvasRenderingContext::GetSkColorSpace().
