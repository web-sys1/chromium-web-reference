[Private Network Access] Add `targetAddressSpace` to request

Introduce a new option targetAddressSpace to requests. The IP address
space is grouped as local, private, public and unknown, and is speced in Private Network Access: https://wicg.github.io/private-network-access/#framework

We would like to introduce it as a new option in requests to support relaxing mixed content check with permission prompts. See: https://docs.google.com/document/d/1Q18g4fZoDIYQ9IuxlZTaItgkzfiz_tCqaEAI8J3Y1WY/edit#
