Add a tool to benchmark the speed of building a specific file locally.

While it can be used to benchmark a local action, it is most useful when
comparing it across configurations. It can answer questions such as:
* What is the effect of clang modules on build performance of a specific
  target?
* What is the overhead of chromium plugins?
* Is compiling to different targets more expensive?
* What is the overhead of debug, opmitizations, & sanitisers on compilation time?
