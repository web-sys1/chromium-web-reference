[UserEducation] Implement InteractionSequence for Tutorials System

See Tutorial System design:
https://docs.google.com/document/d/1d2cqkjrnZGXCkP_vStW97KFI_Ff3dIJcDkiRRIoISMY/edit?usp=sharing

InteractionSequence allows for the creation of expected sequences of
user-UI interactions, with callbacks at each step. It depends on
ElementTracker (upstream CL).

The types of events that are currently supported are:
- a UI element becomes visible to the user
- the user clicks on or otherwise activates a UI element
- a UI element stops being visible to the user

Other events (such as mouse hover, focus changes, etc.) may be
interleaved with these events. The sequence will either complete, or if
the user dismisses one of the required elements (a dialog or menu,
typically), the sequence is aborted.

Implementations/utilities for the general cross-platform case and Views
specifically are provided.
