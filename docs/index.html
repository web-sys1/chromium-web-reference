smartcard: Rename Mojo's kUnknownError to kUnknownCard

`SmartCardError::kUnknownError` maps to `SCARD_E_UNKNOWN_CARD`, which is
confusing. This CL renames it to `kUnknownCard` to clearly reflect its
underlying meaning.

Specifically, it updates:
* The `device::mojom::SmartCardError` enum.
* The `chrome.smartCardProviderPrivate` extension API `ResultCode` enum.
* DevTools Protocol (`SmartCardEmulation`) mappings.
