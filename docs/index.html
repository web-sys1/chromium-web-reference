desks_template: Shift single instance windows to the current desk.

Desks template will create a new desk with a couple windows opened.
Some windows only support a single instance (i.e. settings SWA,
Calculator Chrome App). These windows should be moved to the new desk,
which will be the active desk.

When told to launch an app, we will search the windows on existing
desks to see if there is a current app on a different desk of the
same name. Then, instead of trying to launch it, which will activate
it (and activate the desk it is currently lying on), we will move it
without animation to the newly created desk, which is the active desk.
