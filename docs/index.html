Read result from segmentation platform for showing chrome start

Once the segmentation platform is setup, Start config will cache the
result of segmentation. The next session the result will be available
to use. This CL adds experiment params to also get results from the
segmentation platform and compare to results from code logic and
record to UMA, but still uses result from code. Additionally
adds a "model" param which will enable using the result from the model
instead.
