Remove unused tag_type field in PDF accessibility text runs

Remove the tag_type field from AccessibilityTextRunInfo. This field was
only populated behind kPdfTags, which is disabled by default.

When kPdfTags is enabled with a valid structure tree, the structure
builder will be used instead of the heuristic builder that would read
tag_type. The field is therefore never used an can be removed.

Because tag_type was unused, the methods
StructureElementTypeToAccessibilityRole and
StructureElementTypeToHeadingLevel were also not used and are removed
from the heuristic tree builder as well.

As part of this change, TestSelectionConversionViaFindNodeOffset is
parameterized to run in both structured and heuristic modes.
