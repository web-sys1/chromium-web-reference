[Media] Fix DataBufferTest.ReadingWriting

This patch fixes an MSan error on the ReadingWriting test
in the DataBufferTest suite.

This error occurred due to accessing unitialized memory. This patch
fixes the issue by only accessing the initialized portion of memory.

In order to avoid this in the future, the data() method now returns
a base::span that is truncated to only return the valid portion of the
buffer. The writable_data() method still returns the entire buffer.
