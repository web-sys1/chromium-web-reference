Chromium Updater - Fix deadlock on self-update

Currently, on self-update there is a deadlock due to the update-internal
service requesting the prefs lock when the prefs lock is already held by
the update service. To mitigate, if the updater detects that there is
an existing updater with valid version, then don't perform wake and go
straight to wake done.
