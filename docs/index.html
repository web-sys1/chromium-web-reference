Convert json_schema_compiler to allow removing base::Value::GetAsList()

Adds a new Param::Create() signature in code generated by the
json_schema_compiler to allow users to switch from GetAsList() to the
modern base::Value calls.

This cl also begins to convert some of the related generated code, e.g.
converting unnecessary uses of base::Value* to base::Value, and
fixes some of the GetAsList() uses.
