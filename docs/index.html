[text-combine] Get rid of unncessary NOTREACHED

This patch gets rid of unncessary `NOTREACHED()` which is used for
getting the test case having zero width font for combined text but
content width is non-zero.

This is casues by following expression in `ListMarker::WidthOfSymbol()`:
  `LayoutUnit((font_data->GetFontMetrics().Ascent() * 2 / 3 + 1) / 2 + 2)`
even if font acent is very small, we have 2px width from last `+ 2`.
