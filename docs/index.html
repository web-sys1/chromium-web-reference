Desk Templates: Fix UAF triggered by HoverOnTemplateItemView test

The unit test `DesksTemplatesTest.HoverOnTemplateItemView` causes a UAF
when it tries to check the status on a template item view, even though
we have already exited the grid. This is due to certain grid exit
conditions being recently changed/updated. This CL fixes this issue by
removing a condition that is no longer expected, so that we aren't
checking something after it has already been destroyed.
