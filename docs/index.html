NavigationRequest: Simplify override-user-agent logic.

This patch simplify the logic in the NavigationRequest. It removes one
unused content/public method.

This patch removes:
- NavigationHandle::GetIsOverridingUserAgent()  [was unused]
- NavigationRequest::GetIsOverridingUserAgent()
- NavigationRequest::was_set_overriding_user_agent_called()
- NavigationRequest::was_set_overriding_user_agent_called_
- NavigationRequest::entry_overrides_ua()
- NavigationRequest::entry_overrides_ua_

Goals:
- Only one source of truth: |commit_params->is_overriding_user_agent|
- Unconditionally write the "is_overriding_user_agent" into
  NavigationEntry at DidCommitNavigation time.
