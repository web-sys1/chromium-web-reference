gfx::HDRMetadata: Fix IPC comments

The extended range metadata is now correctly plumbed over IPC, so
remove comments indicating otherwise.

Also add comments indicating the metadata names as they are referred
to in ISO 22028-5. Included in ISO 22028-5 but not included yet in
gfx::HDRMetadata are nominal diffuse white level (NDWL), content color
volume (CCV), and reference viewing environment (REVE).
