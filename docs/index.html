[Eche] Implement Unique Apps streamed metric

This metric tracks the number of unique apps a user opens each session.
Per user, per day aggregation will occur on the /uma timelines. There is
a rare edge case where a session can last more than a day; this is
rare but we still account for that and have a tracking metric for it.

A follow-up CL will move these metrics and other appropriate Eche
metrics to an AppMetricsRecorder class.
