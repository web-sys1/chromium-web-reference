views::Textarea - fix improper placeholder text alignment.

views::Textarea proposes improper placeholder alignment (vertically
centered) on macOS.
This patch removes historic platform-relevant code to ensure
Canvas::DrawStringRectWithFlags draws text vertically top aligned on all
platforms when 'Canvas::MULTI_LINE' flag is set and updates textArea
to use this flag on 'placeholder_text_draw_flags'.
