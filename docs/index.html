[iOS] Hide tab group creation snapshot when too small

Ensures that the snapshots of the TabGroup creation screen are hidden
when the view is too small.
The previous implementation was relying on the trait collection updates
but this doesn't work for keyboard show/hide events.

TViewController viewDidLayoutSubviews or updateViewConstraints aren't
called for those events, so we have to use layoutSubviews.
