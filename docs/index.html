Ensure XML documents do not go into view source mode.

View source mode for XML files enables a tree viewer of the XML file.
This mode is not used for non-main frames and fenced frames should
work the same as iframes. This allows the XML file to be displayed
as it is in fenced frames as opposed to a friendly tree view. The
change adjusts HasNoStyleInformation to return false since a
fenced frame does have styling information.
