Use the Android Data Saver API to insert "Save-data: on" headers.

Previously the Lite Mode setting used to result in Chrome adding
"Save-data: on" headers to its network requests. Now that Lite Mode is
deprecated, Chrome will rely on the Android Data Saver OS setting instead.

This change also restores data saver tests that have been removed in
https://chromium-review.googlesource.com/c/chromium/src/+/3419776
The tests have been updated to use the Android Data Saver setting
instead of the Lite-Mode setting.

Note that most of this CL has been written by jacobstanley@google.com.
