[@container] Handle container-relative units in SVGLength

Pass SVGLengthContext's element as the context for resolving container-
relative units.

The conversion logic for container-relative units is too complicated to
replicate in svg_length_context.cc, so we instead lean on
SVGLengthContext::ResolveValue to carry out the conversion if such
units are involved.
