Rename the bookmark-related interfaces of the Clipboard class to URL.

In //ui/base/clipboard/clipboard.h, the abstraction of a "bookmark" is
essentially a tuple consisting of a "URL and title". In
//ui/base/dragdrop/os_exchange_data_provider.h, the corresponding
abstraction for the GetURLs/SetURLs interfaces is the ClipboardUrlInfo
struct defined in //ui/base/clipboard/clipboard_url_info.h —— the core
content of this struct is also a URL and a title.

The Clipboard::WriteBookmark interface and its related interfaces should
be renamed to Clipboard::WriteURL, and this interface should also use
the ClipboardUrlInfo struct (which is located in the //ui/base/clipboard
directory).
