Convert the CustomElementReactionStack to be Agent supplement data.

This implementation now matches the specification text.
- Removed the stack CEReactionScope static variable and always push
an element on the stack.
- Associate CustomElementReactionStack with Supplement<Agent>
- Remove CustomElementReactionStack member from CustomElementRegistry as
it seems unused.

This is a reland of crrev.com/4bc40abe but without the dependant
change of CEReactionsScope looking up the current ExecutionContext.
Looking up the current ExecutionContext caused a bindings perf
regression and to avoid that we keep the `top_of_stack_` variable
around but only associated CustomElementReactionStack when we have
something to do.
