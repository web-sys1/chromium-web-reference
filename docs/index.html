[launcher]: Layout Recent Apps to match the grid

Recent apps is designed to hold 4 to 5 app items. Right now, the
section is laid out with empty space between each item, so the container
fills the row but at the same tiime distributes space equally between
elements.

For the case where there are less than 5 items, rather than
distributing the space, the apps should look like they match the main
grid apps' position.

This CL adds the necessary calculations to the sections so that space
between elements match the way it is calculated in the main grid and
apps in the section look aligned with the ones on the main grid.

Additionally, some test have bveen rewritten or discarded to consider
the new functionality.
