[Autofill] Return filled FieldType in FormFiller::FillField()

This CL lets FormFiller::FillField() return the filled FieldType.
This FieldType is then used to build the `field_type_map` that
determines the iframe security policy.

This is a preparatory change for union types: with union types,
we cannot build the `field_type_map` after filling because the
type may be ambiguous.
