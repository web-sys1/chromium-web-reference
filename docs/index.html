> TestMockTimeTaskRunner: Allow sequence local values.
>
> This CL makes TestMockTimeTaskRunner capable of exposing a
> SequenceLocalStorageMap so that tasks can store sequence local values.
> This is needed to satisfy certain runtime checks.
>
> Note that this CL introduces SequenceLocalStorageMapOverrideForTesting
> as an analogous of the existing
> ThreadTaskRunnerHandleOverrideForTesting. This is to handle the case
> where, e.g., one task runner calls RunUntilIdle() on another task
> runner: the existing ScopedSetSequenceLocalStorageMapForCurrentThread is
> not sufficient because it assumes there's no existing sequence local map
> on construction.
