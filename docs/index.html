[Resource file sharing] Introduce DataPackWithResourceSharing for Lacros

This CL adds DataPackWithResourceSharing which is used in Lacros when we
share resources file with Ash.
DataPackWithResourceSharing has 2 DataPacks, |fallback_data_pack_| and
|ash_data_pack_|.
If the resource with lacros_resource_id is included in ash resources,
get the ash_resource_id corresponded to lacros_resource_id by looking up
|mapping_table_| and get data in ash resources.

This CL doesn't affect the behavior since DataPackWithResourceSharing is
not yet used. It will be used in the following CLs.
