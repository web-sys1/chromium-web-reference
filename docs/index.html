Ensure account info availability during Lacros FRE flow init

`LacrosFirstRunSignedInFlowController` when called from browser startup
creates a `TurnOnSyncHelper` before it is able to obtain the account
info associated with the primary profile. This is because refresh
tokens are not yet available for that account. So we now wait for
them before proceeding with the initialization of the signed in flow.
