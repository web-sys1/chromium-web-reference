Python script to output stats on Kombucha tests

Initial check-in of an ad-hoc python script that outputs status on the
Kombucha tests. The implementation relies on some assumptions that these
tests use RunTestSequence or RunTestSequenceInContext. This script will
be run every quarter to pull these stats to be included in our quarterly
newsletter.
