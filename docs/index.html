Add histograms for socket brokering

Add histograms Network.BrokeredTcpClientSocket.SocketCreationTime and
Network.BrokeredUdpClientSocket.SocketCreationTime to measure the time
that the socket brokering call to the browser process takes.

This will permit estimating the potential speedup from prefetching
brokered sockets.
