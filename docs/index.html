Sent attachedToTarget events to all connected CDP sessions

This CL makes sure all connected CDP sessions receive the
attachedToTarget event. Previously only one of the sessions
received that event (the one that ended up creating the
RFDAH). This CL also allows all connected sessions to
block on the debugger, i.e. each session now has to call
Runtime.runIfWaitingForDebugger to resume the navigation.
Previously only the session that created the RFDAH was
able to resume the navigation.
