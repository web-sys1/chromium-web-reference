Log UMA metrics for HttpHeaderInjection policy

This CL introduces UMA logging to track the performance and usage
frequency of the HttpHeaderInjection policy.
- Updates HttpHeaderInjectionMatcher to log rule matching duration.
- Updates HttpHeaderInjectionClient to log whether a request was modified.
- Updates corresponding unit tests to verify metrics.
- Defines the new histograms in histograms.xml.
