[vault] Differentiate missing and unusable PINs

Have DownloadAuthenticationFactorsRegistrationState distinguish between
Google Password Manager PIN entries that are not present and those that
are present, but unusable for recovery.

This will make it possible to set the public key of the current GPM PIN
when joining a new PIN if the current PIN is not usable for recovery. No
behaviour change is intended by this CL.
