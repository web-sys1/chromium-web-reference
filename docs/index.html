Change the canonical unit for time category from ms to s.

According to the spec, https://drafts.csswg.org/css-values-4/#time,
the canonical unit for <time> is Seconds.

And this CL fixes css calc unit type canonicalzation bugs
which occurred because CSSMathExpressionOperation::ResolvedUnitType()
didn't return the actual unit type for its operands_.
