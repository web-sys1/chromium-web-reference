NoteTaking: Remove extensions listeners in favor of AppService

NoteTakingHelper was already listening to AppService for updates on web
apps. This CL adds NoteTaking intent information to AppService for
Chrome Apps which removes the need to separately listen for extensions
updates. This simplifies the logic and means there are fewer
observations that need their lifetimes carefully managed.
