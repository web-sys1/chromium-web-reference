Tell pkgbuild to compress as compactly as possible

In macOS 12 Monterey, pkgbuild gained the ability to compress
a component package to a greater extent if given the minimum OS
needed for that package. Use this ability if present.

Note that this will fail to detect macOS 12 if run with Python 2.
