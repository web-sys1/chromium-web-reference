Add EnableProxyOverrideRulesForAllUsers implementation

See crrev.com/c/7080465 for the CL that added the policy

The EnableProxyOverrideRulesForAllUsers implementation is done in two
parts:
* In the policy handler, only error messages are added when its own
  schema is not matched, or when affiliation is checked against it to
  add an appropriate error message to the main proxy override rules
  policy.
* In ProxyConfigServiceImpl, the conversion from pref to ProxyConfig
  will skip proxy override rules in unaffiliated cases where the policy
  is not set/set to 0.

BYPASS_POLICY_COMPATIBILITY_CHECK=Enum value 2 won't be supported.
