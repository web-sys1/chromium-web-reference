[finds] Fix crash in FindsTabHelper for Incognito tabs

The FindsService is not available in Incognito mode, which caused FindsServiceFactory::GetForProfile to return a null pointer. The FindsTabHelper constructor did not handle this case and would crash due to a CHECK failure.

This change adds a null check to ensure that the FindsTabHelper is only created when the FindsService is available.
