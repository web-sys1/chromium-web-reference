[WebRTC] Metronome: Add UI flag for Android WebView.

In order to run the WebRtcMetronomeTaskQueue experiment on Android
WebView, we are required to add a UI flag for it
(https://chromium.googlesource.com/chromium/src/+/main/android_webview/docs/developer-ui.md#Adding-your-flags-and-features-to-the-UI).
This CL does that.

The base::Feature of interest lives inside third_party/webrtc_overrides
so we had to add a new template to get the flag available inside of
Java code.
