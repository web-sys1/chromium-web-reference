[viz] Add more fuzz tests for Viz mojom traits.

This change introduces new FUZZ_TESTs for various Viz compositing types to ensure correct serialization and deserialization via Mojo. The added types include ViewTransitionElementResourceId, SurfaceInfo, ReturnedResource, CompositorFrameMetadata, CompositorRenderPass, OffsetTag, OffsetTagValue, OffsetTagDefinition, SharedImageFormat, TransferableResource, CopyOutputResult, and TreesInVizTiming. The BUILD.gn file is updated to include these new tests.
