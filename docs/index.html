Update shelf pin state when show in shelf changes for an app

Better handle changes in app show in shelf state - if the app is
pinned at the time, and show in shelf changes to false, unpin it.
When the show in shelf changes to true, ensure the app gets pinned if
user prefs indicate the app should be pinned.

Note that currently, apps not supposed to be shown in shelf will not
get pinned to shelf (even if they're pinned by policy). Though, if the
app show in shelf state changes after app installation (which can happen
due to certain policies), the pin state was not getting updated,
except for apps that were disabled by policy. For apps disabled by
policy there is special logic to add apps to shelf if show in shelf is
set to true - the cl preserves that logic.
