Pciguard: Add notification controller

- Adds a notification controller that is responsible for displaying
the appropriate notification.

- Adds support in system_tray_client to navigate to the privacy and
security settings page. This is needed to support one of the
notification's features to route to the Settings App.

- This class is only the initial implementation. In the near future
this class will observe Pciguard and TypeCDaemon Dbus signals to
determine when to show the notifications.
