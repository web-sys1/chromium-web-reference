[sync] Introduce Javascript API to add recovery methods

This advanced API maps to the the equivalent
TrustedVaultClient::AddTrustedRecoveryMethod() and allows the identity
page (Gaia) to register a public key that is known to be trusted.

Among other purposes, this is designed to allow the user to resolve
degraded sync recoverability states from a web page.
