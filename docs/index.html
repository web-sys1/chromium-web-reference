Expose SVGLength to Length conversion on SVGLengthContext

Add a new "helper" to SVGLengthContext that resolves to a Length. This
is primarily to be used in cases where resolving needs to be done
against a specified reference dimension (for percentage resolution).
Factor out some "building blocks" that will be used to implement similar
cases.

Convert LayoutSVGTextPath::LayoutPath() to use this method rather than
the problematic SVGLength::ScaleByPercentage().

Use FloatValueForLength() directly because we shouldn't be seeing non-
specified Lengths as a result of the conversion.
