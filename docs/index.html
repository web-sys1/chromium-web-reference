[Launcher pop-in] Sort post-burn-in categories by arrival order.

A previous CL introduced tracking of the burn-in iteration number for
categories. This CL introduces a method of sorting the categories prior
to publication.

For categories which first appear during the post-burn-in period, sort
according to burn-in iteration number (i.e. arrival order), and then
sort by score. Pre-burn-in category sorting is unchanged.
