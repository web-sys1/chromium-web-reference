Update disabled link color from NativeThemeGTK

This CL updates the disabled link color for GTK version 3 to use the
CSS selector "GtkLabel#label.link:link:disabled".

This is both more correct and eliminates the need for the |base_theme|
and |color_scheme| parameters from the SkColorFromColorId() function.

Follow up CLs will use the SkColorFromColorId() function to
instantiate GTK ColorMixers for the ColorProvider project.
