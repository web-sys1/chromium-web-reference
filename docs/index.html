Update the SendTabToSelfToolbarIconController to use TabInterface

Modify the tab activation logic to check if a background tab belongs to
the current browser directly via the TabInterface. Previously, the code
used TabStripModel::GetIndexOfTab() to determine if a tab was present in
the given browser window. By checking the browser window interface
directly on the tab, the logic correctly verifies the tab's owner before
attempting to find its index and activate it.
