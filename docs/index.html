Consolidating Submenu and Context menu implementations of Saved Tab Group recall UI

This is second part of the fix to Saved Tab Group recall UI to fix issue
with favicons not being show correctly. While we fixed this for the App
Menu's submenu for Saved Tab Groups, the context menu used a different
implementation. In order to fix this issue, deleting the DialogModel
based context menu implementation and instead using a single model for
both these implementations. The duplicate DialogModel implementation
makes things harder to maintain. This also addresses the default favicon
issue seen with the context menu.
