Add LacrosWindowHandler for full restore to support Lacros.

DD: go/full-restore-for-lacros

This CL is used to prepare for saving and restoring Lacros browser
windows:

1. Create LacrosWindowHandler to observe BrowserAppInstanceRegistry for
Lacros windows, to get browser session ids for saving and restore
Lacros windows.

2. Rename IsFullRestoreAvailableForLacros to
MaybeCreateFullRestoreServiceForLacros to create FullRestoreService in
MaybeCreateFullRestoreServiceForLacros.

Lacros could be launched at the very early stage during the OS startup
phase, and MaybeCreateFullRestoreServiceForLacros is called before
Launching Lacros. So create FullRestoreService to construct
LacrosWindowHandler to observe BrowserAppInstanceRegistry for Lacros
windows before launching Lacros, to avoid missing any Lacros windows.

Tests will be added later in a separate follow up CL.
