[tvOS] Prevents forwarding handled events to super.

This prevents forwarding handled events to super. On tvOS 26, keyboard
input can trigger duplicate remote button events, so handled events
are not propagated. UIPressTypeMenu events are an exception and are
still passed to super to allow to move the app to the background.
