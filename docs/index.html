[iOS] Update displayTabGridInMode to support Tab Groups

This change adds kTabGroups to the TabGridOpeningMode enum so that users
can be redirected to the Tab Groups page on the tab grid.

This command will be used in an upcoming Tab Groups promo to open the
Tab Groups page for users who tap the primary action button.
