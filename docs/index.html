[UIA] Return UIA_WindowControlTypeId for all dialogs

In order to maintain the AT's focus "trapped" within the dialog window,
all dialogs need to return UIA_WindowControlTypeId as their control type
and need to implement the WindowProvider. The latter was already taken
care of, but the former is fixed in this CL.

Please note that, although we're now returning the expected control type
value, the focus doesn't get "trapped" as we'd expect. I'll investigate
more and will come back with a complementary fix in a subsequent CL.
