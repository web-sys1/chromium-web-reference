[Metadata][Clank] Limit Keyboard Accessory chip width for virtual card suggestions

crrev.com/c/3965178 constrains the chip width of credit card suggestions on the keyboard accessory. Virtual cards are also credit cards, but their suggestions are not being width constrained.

This is because virtual card suggestions have a different `frontend_id` than the regular credit card suggestions, and we use this to decide whether or not to width constraint the chips. Fix this issue by including virtual card suggestion's `frontend_id` to be width constrained.
