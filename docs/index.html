[Fenced Frame] Do not support beforeunload&unload events

This CL disables the beforeunload event and the unload event in
the fenced frame and adds WPT tests to ensure the beforeunload
handler is not called.

This CL doesn't add a TC for the unload event since await keyword
in async function wouldn't be worked in unload handler and it
doesn't guarantee to write the result in writeValueToServer()
to the primary main test page.

Skip virtual/fenced-frame-*/wpt_internal/fenced_frame/navigate-ancestor.html
