personalization: custom search provider

Implement a custom search provider for personalization search
results. This allows personalization app to have detailed
control over search results that generate links into
subpages of Personalization App depending on which features
are available or activated.

Personalization search results will be grouped under Settings
results in Productivity Launcher. The personalization search
results currently exist under Settings with PersonalizationHub
feature off, so they will stay in that category (but get a new icon)
for now.

This CL does not dynamically generate results. A single search
result is generated if the user searches for the literal text
"Personalization". Follow up CLs will save documents into
a search index, as with os-settings search and help-app-ui search.
