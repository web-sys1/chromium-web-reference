CSP: Include mojo CSP header type in blink::CSPDirectiveList

This is a small refactoring of the code in
blink::CSPDirectiveList. Instead of storing in this class the header
information as a String, a
network::mojom::blink::ContentSecurityPolicyType and a
network::mojom::blink::ContentSecurityPolicySource, we store directly
the mojo type network::mojom::blink::ContentSecurityPolicyHeader,
which encompasses the three of them.

This is part of a project to harmonize the CSP code in Blink and in
services/network, and will make it easier to synchronize Content
Security Policies between the two.
