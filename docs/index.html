[DebugUITabStrip] Add additional fields to mojo::Tab in tab-strip-internals

This CL introduces Phase 3 of the 'tab-strip-internals' page, a debug
WebUI intended to visualize the live state of the TabStrip model.

Changes:
- Updated `resource_ids.spec` to expand includes size
- Updated `tab_strip_internals.mojom` to add additional fields to mojo::Tab
- Updated `tab_strip_internals_util` to populate newly added fields from tabs::TabInterface
- Updated `c\b\r\tab_strip_internals\BUILD.gn` to include `c\b\u\w\tabs` mojo generators

Future phases will add support for session restore information from the
SessionService to the 'tab-strip-internals' page.
