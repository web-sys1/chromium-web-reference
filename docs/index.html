[RWM Ramp-down 2/4] Add helper to determine adjacent window launch behavior

This helper method determines if a new window should be opened
adjacently (split-screen) or in full screen, replacing the logic that
previously relied on the experimental feature flag. By default, windows
will open adjacently, except for mixed-profile launcher
(regular-to-incognito or vice-versa) when the incognito full-screen
feature is enabled.
